## 開放原始碼的商業模式

### 雙重授權

```python
# 開放原始碼商業模式
# 1. 雙重授權（MySQL、Red Hat）
#    - GPL 授權（免費）
#    - 商業授權（收費，含支援）

# 2. 服務支援（Canonical、Red Hat）
#    - 提供技術支援和顧問服務

# 3. 附加功能（MySQL Enterprise）
#    - 開源版本 + 商業功能

# 4. 雲端服務（Google、Amazon）
#    - 使用開源技術，提供雲端服務
```
