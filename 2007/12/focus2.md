# Web 2.0 的成熟：互動式 Web 應用

## Web 2.0 的定義

Web 2.0 是 2004 年 Tim O'Reilly 提出的概念，指的是新一代的網路應用。

### Web 2.0 的核心原則

```
Web 2.0 原則：
──────────────
1. 網路作為平台
2. 資料是智慧資產
3. 軟體above-the-line（即 end-user facing）
4. 輕量級程式模型
5. 順暢的軟體發行
6. 社交網路
7. 使用者貢獻內容
```
