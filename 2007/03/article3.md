# jQuery 與前端革命

## 前言

jQuery 1.0 的發布標誌著前端開發的革命性時刻。

## jQuery 的優勢

```javascript
// 簡潔的選擇器
$("#id")
$(".class")
$("div > p")

// 鏈式操作
$("div")
  .addClass("highlight")
  .css("color", "red")
  .hide();

// AJAX 封裝
$.getJSON("/api/data", function(data) {
    console.log(data);
});
```

## 結論

jQuery 徹底改變了前端開發，讓 JavaScript 變得優雅。
