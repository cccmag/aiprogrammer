# 本月新知

## 2007 年 3 月程式語言與技術動態

### 動態語言的繁榮

**Python 持續普及**

Python 在 2007 年繼續在科學計算和網路開發領域獲得採用。版本 2.5 的穩定性使其成為許多新創公司的首選語言。

**Ruby on Rails 熱潮**

Rails 框架的流行帶動了 Ruby 語言的普及。2007 年初，許多 Web 開發者開始學習 Ruby。

**PHP 統治 Web 開發**

根據統計，2007 年約有 70% 的網站使用 PHP。WordPress、Drupal 等 CMS 的流行鞏固了 PHP 的地位。

### JavaScript 的復興

**jQuery 1.0 正式發布**

1 月底，jQuery 1.0 正式發布，為 JavaScript 開發者提供了簡潔的 DOM 操作 API。

**AJAX 成為標準**

越來越多的網站採用 AJAX 技術，提供類似桌面應用的互動體驗。

### JVM 生態

**Java SE 6 發布**

2006 年 12 月，Java SE 6 正式發布，帶來了更好的效能和腳本支援。

**JRuby 持續進步**

JRuby 在 Java 平台上運行 Ruby，提供了另一種語言選擇。

### .NET 平台

**C# 3.0 開發中**

微軟正在開發 C# 3.0，將帶來 LINQ 等革命性功能。

### 函數式語言

**Haskell 社群成長**

Haskell 在學術界和工業界都有穩定的追隨者群體。

**Scala 的興起**

Scala 在 JVM 上結合了物件導向和函數式編程，開始獲得關注。

### 開源資料庫

**MySQL 5.1 開發中**

MySQL 繼續是 Web 應用的首選資料庫。

**PostgreSQL 8.3 準備中**

PostgreSQL 8.3 將包含向量拒絕等新功能。
