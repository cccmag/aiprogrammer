# 回顧與結語

## 2007 年 4 月號結語

### 本月主題回顧

四月是行動 Web 開發的轉捩點。從 WAP 的文字時代到 WebKit 的豐富介面，從功能手機到智慧型手機，行動網路正在經歷一場革命。

#### 行動 Web 開發的啟示

行動 Web 開發告訴我們一個重要的道理：**好的使用者體驗不取決於客戶端的能力，而取決於開源標準與創新之間的平衡**。

從 1999 年的 WML 到 2007 年的 WebKit，每一代行動 Web 技術都在解決一個核心問題：如何在資源受限的環境下提供有意義的內容。

#### 本月技術亮點

**WebKit 開源**是 2006 年最受矚目的事件之一，為行動瀏覽器統一奠定了基礎。

**iPhone 即將發布**是 2007 年最受期待的事件。六月揭曉的 iPhone 將徹底改變行動網路生態。

**JavaScript 框架持續演進**：Prototype.js、Dojo、iUI 等框架正在為行動 Web 開發做準備。

### 未來展望

**iPhone SDK 的影響**：原生應用 vs Web 應用的辯論將持續。

**HTML 5 的標準化**：離線應用、Canvas、地理位置 API 將成為標準。

**行動 Web 2.0**：社交網路、位置服務、豐富互動的融合。
