# 本月新知

## 2007 年 4 月 Web 與程式技術動態

### 網頁開發技術

**WebKit 開源一週年：蘋果與 KDE 的合作結晶**

2006 年 4 月蘋果宣布開源 WebKit 引擎，一年後的 2007 年 4 月，WebKit 已成為行動瀏覽器領域的領先引擎。Safari Mobile、iPhone 的瀏覽器、以及 Android 的 Chrome 都基於 WebKit。WebKit 的 Nitro JavaScript 引擎讓行動裝置上的 JavaScript 執行效率提升了 2-3 倍。

**Firefox 3 開發正如火如荼**

Mozilla 社群正在加緊開發 Firefox 3 代號「Gran Paradiso」。新版本將支援 Google Gears 離線應用、改良的書籤同步、以及全新的地點感知瀏覽。Firefox 3 的 UI 也進行了大改版，採用了作業系統原生的視窗裝飾。

**IE8 開發中：微軟的標準化努力**

微軟在 4 月確認 IE8 正在開發中，並承諾更好的 CSS 標準支援。IE8 將支援 ACID2 測試，這是網頁標準相容性的重要指標。微軟同時宣布將參與 W3C 的 HTML 5 標準制定。

### 程式語言與框架

**Ruby on Rails 1.2.3 發布**

Rails 社群在 4 月發布了 1.2.3 版本，修復了多個安全漏洞。Rails 1.2 引入的 RESTful 路由和多國語言支援正在被越來越多開發者採用。

**Dojo Toolkit 1.0 正式發布**

Dojo 基金會發布了 Dojo Toolkit 1.0，這是一個全面的 JavaScript/Ajax 框架。Dojo 提供了從 DOM 操作到圖形繪製、離線儲存的完整工具集。

**jQuery 1.1.2 效能大提升**

jQuery 團隊發布了 1.1.2 版本，在 DOM 操作效能上提升了 2-10 倍。jQuery 簡潔的語法，讓它成為最受歡迎的 JavaScript 庫之一。

### 行動裝置開發

**iPhone 傳言：六月即將揭曉**

整個四月，科技部落格都在猜測蘋果將在六月全球開發者大會（WWDC）上公布 iPhone。各種間諜照和規格傳言充斥網路。分析師預測 iPhone 將徹底改變智慧型手機市場。

**Android SDK 預覽版問世**

Google 在四月低調發布了 Android SDK 預覽版，讓開發者得以搶先體驗這款開放手機平台的開發環境。Android 使用 Java 作為應用程式開發語言，並基於 Linux 核心。

**Windows Mobile 6 正式發布**

微軟在四月發布了 Windows Mobile 6，代號「Crossbow」。新版本強化了 Office Mobile 的功能，並加入了 Internet Explorer Mobile 的顯著改進。Windows Mobile 6 支援 Silverlight Mobile（.NET Compact Framework 3.5）。

### 開源專案

**OpenID 2.0 規範發布**

OpenID 基金會在四月發布了 OpenID 2.0 規範，大幅提升了安全性與易用性。Yahoo!、Google 和微軟都宣布支援 OpenID。單一登入（SSO）的願景即將成真。

**Git 1.5.0 發布**

Linux 核心開發使用的版本控制系統 Git 在四月發布了 1.5.0 版本。Git 的效能和穩定性持續提升，正在逐漸被更多專案採用。

### 業界動態

- **Yahoo! 收購 Tumblr 傳言**：四月的科技圈猜測 Yahoo! 可能收購微型部落格平台 Tumblr
- **Facebook 開放平台**：Facebook 在四月持續擴展其開放平台，吸引了數千個應用程式
- **MySpace 進軍開放平台**：MySpace 宣布將推出自己的開放平台，與 Facebook 競爭
