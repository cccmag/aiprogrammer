# 問答系統的進展：Watson 之前的探索

## 前言

2007 年，IBM 正在開發 Watson 系統，目標是打敗 Jeopardy! 冠軍。

## 問答系統架構

```
問答系統元件：
──────────────────
1. 問題分析       - 實體識別、類型分類
2. 文件檢索       - 維基百科、新聞等
3. 答案抽取       - 命名實體識別
4. 答案排序       - 信心度評估
```

## 結語

Watson 在 2011 年打敗了人類冠軍，展示了 QA 系統的潛力。
