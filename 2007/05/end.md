# 回顧與結語

## 2007 年 5 月號結語

### 本月主題回顧

五月是 Web 服務技術的重要轉捩點。從 SOAP 到 REST，從 XML 到 JSON，Web 服務的設計正在經歷根本性的變化。

### 未來展望

**REST 將成為公共 API 的主流**。簡單的 URI、標準的 HTTP 方法、人類可讀的格式——這些特性使 REST 易於使用和理解。

**JSON 將繼續勝出**。雖然 XML 在企業應用中仍有價值，但新的 Web API 將越來越傾向使用 JSON。

**OAuth 將解決第三方認證問題**。使用者將能夠安全地授權第三方應用訪問他們的資料，而無需透露密碼。
