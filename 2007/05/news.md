# 本月新知

## 2007 年 5 月 Web 服務與程式技術動態

### Web 服務技術

**Amazon EC2 正式商業化**

2007 年 5 月，Amazon AWS 服務持續增長。EC2（Elastic Compute Cloud）從 beta 轉向商業服務，越來越多的開發者開始使用雲端運算資源。

**Google 發布 Blogger API 3.0**

Google 發布了新版 Blogger API，支援 RESTful 設計。這是 Google 開放平台策略的一部分，越來越多的 Google 服務開始提供 REST API。

**Yahoo! Pipes 發布**

Yahoo! 發布了 Pipes——一個視覺化的資料處理工具，讓非程式設計師也能組合和處理 Web 資料來源。Pipes 的設計理念是「資料管線即服務」。

### 程式語言與框架

**Django 0.96 發布**

Python Web 框架 Django 發布 0.96 版本，完善了 ORM 和管理介面。Django 的「快樂開發」哲學越來越受到歡迎。

**Spring 2.5 開發中**

Java 社群期待 Spring Framework 2.5 的發布。新版本將支援註解驅動的開發，簡化 XML 配置。

**Node.js 概念驗證**

Ryan Dahl 在 2007 年開始了他的 Node.js 專案概念驗證，採用事件驅動、非阻塞 I/O 的設計。

### 資料格式

**JSON 越來越流行**

越來越多的 Web API 選擇 JSON 而非 XML。JSON 的簡潔性和 JavaScript 原生支援是其主要優勢。

**YAML 1.2 規範發布**

YAML 1.2 規範在 2007 年發布，YAML 作為設定檔和資料交換格式的應用持續增長。

### 業界動態

- **Facebook API 持續演化**：Facebook 在五月繼續擴展其開放平台 API
- **eBay 開放平台**：eBay 宣布更開放的 API 策略
- **PayPal API 標準化**：PayPal 簡化了 Web Payments 開發
