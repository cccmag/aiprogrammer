# 機器人作業系統：ROS 的興起

## 前言

2007 年，Willow Garage 啟動了 ROS（Robot Operating System）專案，為機器人軟體開發帶來了革命性的變化。

### 發布-訂閱模型

```python
# ROS 通訊範例
"""
┌────────────────────────────────────────────────────────┐
│         ROS 發布-訂閱模型                              │
├────────────────────────────────────────────────────────┤
│                                                        │
│   [Publisher] ──(topic)──→ [Subscriber]              │
│       │                      │                        │
│       │    roscore           │                        │
│       └──────────────────────┘                        │
│                                                        │
│   例如：                                              │
│   /camera/image_raw ─→ 影像處理節點                   │
│   /cmd_vel ─→ 馬達控制節點                            │
│   /scan ─→ 雷射雷達資料                               │
│                                                        │
└────────────────────────────────────────────────────────┘
"""

# Python 範例
import rospy
from std_msgs.msg import String

def talker():
    pub = rospy.Publisher('chatter', String, queue_size=10)
    rospy.init_node('talker', anonymous=True)
    rate = rospy.Rate(10)
    while not rospy.is_shutdown():
        hello_str = "hello world %s" % rospy.get_time()
        rospy.loginfo(hello_str)
        pub.publish(hello_str)
        rate.sleep()

def listener():
    rospy.init_node('listener', anonymous=True)
    rospy.Subscriber("chatter", String, callback)
    rospy.spin()

def callback(data):
    rospy.loginfo(rospy.get_caller_id() + "I heard %s", data.data)
```

## 結論

2007 年 ROS 的啟動，為機器人軟體開發帶來了標準化的可能。雖然當時 ROS 還處於早期階段，但它為日後十年機器人产业的發展奠定了基礎。

ROS 的出現使得研究者可以專注於演算法開發，而不必重複造輪子。這種開放、協作的精神，正是推動機器人技術進步的重要力量。
