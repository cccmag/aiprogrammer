# 本月新知

## 2007 年 1 月程式與科技技術動態

### 重大事件

**iPhone 發表：科技產業的革命性時刻**

2007 年 1 月 9 日，蘋果公司執行長 Steve Jobs 在 Macworld Conference & Expo 正式發表 iPhone。這款結合手機、音樂播放器與網路設備的智慧型產品，採用了全新的多點觸控介面，徹底改變了人機交互的方式。Jobs 稱其為「一個領先業界五年的產品」。iPhone 預計於 6 月正式上市，引發全球科技產業的熱烈討論。

**Google 宣布收購 YouTube**

2006 年 10 月完成的這筆收購在 2007 年初持續發酵。YouTube 成為 Google 旗下重要的影片分享平台，每日觀看次數突破 1 億次。這筆收購也象徵著網路影片時代的來臨，對數位媒體產業產生深遠影響。

### 程式語言與框架

**Ruby on Rails 1.2 準備發布**

Rails 社群在 2007 年初正準備推出 Rails 1.2，這是 Rails 框架的重要更新。Rails 以其「Convention over Configuration」理念，持續影響著 Web 開發的實踐方式。

**Python 2.5 持續流行**

Python 2.5 於 2006 年 9 月發布後，在 2007 年初獲得廣泛採用。其新增的 `with` 語句和改良的 IDE 支援使其成為科學計算和網路開發的熱門選擇。

**jQuery 1.0 正式發布**

2007 年 1 月，jQuery 正式發布 1.0 版本。這款輕量級 JavaScript 庫以其簡潔的語法和強大的 DOM 操作能力，迅速獲得前端開發者的青睞。jQuery 的誕生標誌著 JavaScript 庫戰國時代的開始。

**Dojo Toolkit 持續發展**

Dojo 作為一個全面的 JavaScript 框架，在 2007 年初持續獲得關注。它提供了從 UI 元件到資料存取的全方位解決方案。

### 人工智慧與機器學習

**Geoffrey Hinton 的深度學習研究**

2007 年初，Geoffrey Hinton 和他的學生們正在推進深度學習的研究。Hinton 在限制玻爾茲曼機和深度信念網路方面的工作，為後來深度學習的突破奠定基礎。

**GPU 運算的興起**

NVIDIA 在 2006 年底推出 CUDA 平台後，2007 年初科學計算社群開始積極探索利用 GPU 進行平行運算的可能性。這種將遊戲顯卡用於科學計算的做法，為日後深度學習的蓬勃發展鋪平道路。

**開源 AI 框架興起**

Torch 和 Theano 等開源 AI 框架在 2007 年初持續發展，為研究者提供了靈活的實驗平台。

### 作業系統與平台

**Windows Vista 正式發布**

Microsoft 的新一代作業系統 Windows Vista 於 2007 年 1 月 30 日正式向企業用戶發布。Vista 帶來了 Aero 玻璃介面、UAC 安全機制和搜尋功能等新特性。

**Mac OS X 10.5 Leopard 準備中**

Apple 預計在 2007 年發布 Mac OS X 10.5 Leopard，將帶來 Time Machine 備份和 Spaces 虛擬桌面等新功能。

### 網路與 Web 2.0

**Web 2.0 熱潮持續**

2007 年初，Web 2.0 概念持續發酵。AJAX 技術、社交媒體、使用者生成內容成為網路發展的主流。

**Facebook 持續成長**

Facebook 在 2007 年初已擁有超過 2000 萬活躍用戶，並持續拓展其平台功能。

### 業界動態

- **Adobe 收購 Macromedia**：Flash 技術整合持續進行
- **Sun 發布 Java SE 6**：提升效能與平台支援
- **Palm 推出 Centro**：智慧型手機大眾化嘗試
- **Nokia N95 發布**：多媒體智慧型手機的旗艦

### 標準與規範

- **ECMAScript 4 草案**：JavaScript 下一代標準討論中
- **HTML 5 草案初期**：W3C 開始討論新世代 HTML 標準
- **Atom Publishing Protocol**：Web 服務新標準
