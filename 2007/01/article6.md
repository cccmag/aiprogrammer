## Geoffrey Hinton 的貢獻

### 2006 年的突破

2006 年，Hinton 發表了關於深度信念網路（Deep Belief Networks）的論文，提出了層貓預訓練（layer-wise pre-training）的方法：

```python
# 深度信念網路的基本概念
DEEP_BELIEF_NETWORK = {
    "結構": "多層受限玻爾茲曼機 (RBM)",
    "訓練方法": "層貓預訓練 + 微調",
    "突破": "首次能有效訓練多層網路"
}

# 受限玻爾茲曼機
RBM = {
    "可見層": "輸入資料",
    "隱藏層": "特徵表示",
    "連接": "僅可見-隱藏，無可見-可見、隱藏-隱藏",
    "訓練": "對比散度 (Contrastive Divergence)"
}
```

## 深度信念網路的原理

### 網路結構

```python
# 深度信念網路示意
"""
       輸出層
         │
    ┌────┴────┐
    │         │
    ▼         ▼
  隱藏3    隱藏3
    │         │
    └────┬────┘
    ┌────┴────┐
    │         │
    ▼         ▼
  隱藏2    隱藏2
    │         │
    └────┬────┘
    ┌────┴────┐
    │         │
    ▼         ▼
  隱藏1    隱藏1
    │         │
    └────┬────┘
    ┌────┴────┐
    │         │
    ▼         ▼
   可見層   可見層
"""

# 每層都是一個受限玻爾茲曼機
# 訓練時，從底層開始逐層訓練
```

### 訓練過程

```python
# 層貓預訓練概念
def pretrain_dbn(dbn, visible_data):
    """逐層預訓練深度信念網路"""
    current_input = visible_data

    for layer in dbn.layers:
        # 訓練這一层的 RBM
        rbm = RBM(layer.n_visible, layer.n_hidden)
        rbm.train(current_input)

        # 更新輸入為隱藏層表示
        current_input = rbm.hidden_activations(current_input)

    return dbn

# 而後使用反向傳播微調整個網路
def fine_tune(dbn, training_data, labels):
    # 使用監督式學習微調
    backpropagation(dbn, training_data, labels)
```

## 結論

2007 年是深度學習發展史上的重要年份。Hinton 的研究為這個領域奠定了理論基礎，開創了讓機器能夠學習更深層表示的方法。
