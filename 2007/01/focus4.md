# 行動網路的變革：3G 與行動上網時代

## 前言

iPhone 的成功不僅靠硬體和介面，更需要網路連接能力的配合。2007 年正值 3G 網路普及，為行動網際網路時代拉開序幕。

### EDGE：過渡時期的選擇

2007 年初，EDGE（Enhanced Data rates for GSM Evolution）是多數網路供應商提供的最高速資料服務：

```python
# EDGE 網路特性
EDGE_SPECS = {
    "名稱": "Enhanced Data rates for GSM Evolution",
    "發布時間": 2003,
    "理論速率": "473.6 kbps",
    "實際速率": "100-200 kbps",
    "覆蓋範圍": "全球 GSM 網路",
    "iPhone 支援": True  # 初代 iPhone 僅支援 EDGE
}
```

iPhone 最初僅支援 EDGE（2.75G），這是批評者攻擊的重點之一。

## 行動網路與 iPhone

### Steve Jobs 的策略

在 iPhone 發表時，Jobs 選擇 EDGE 而非 3G 的理由：

1. **3G 耗電** —— 增加發熱、縮短電池壽命
2. **3G 覆蓋不足** —— 美國 3G 網路尚未普及
3. **WiFi 彌補** —— 城市地區 WiFi 覆蓋良好

這個決定在當時頗受爭議，但某種程度反映了蘋果的實用主義。

## 結論

2007 年是行動網路發展的轉捩點。3G 網路的普及、WiFi 熱點的增長、以及 iPhone 對完整網路體驗的重視，共同開創了行動網際網路的新時代。

雖然初代 iPhone 只支援 EDGE，但其內建的 WiFi 支援和對完整網頁的堅持，預示了未來行動網路的發展方向。
