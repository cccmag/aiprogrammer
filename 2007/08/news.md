# 本月新知

## 2007 年 8 月程式與科技技術動態

### 重大事件

**Ubuntu 7.10 Gutsy Gibbon Beta 發布**

2007 年 8 月，Canonical 發布了 Ubuntu 7.10 的 Beta 版本，代號「Gutsy Gibbon」。這個版本帶來了多項重大改進，包括 Compiz Fusion 桌面特效、簡化的網路設定、精確的硬體偵測等。Ubuntu 7.10 將桌面 Linux 的易用性推向了一個新的高度。

**Linux 核心 2.6.22 發布**

2002007 年 7 月，Linux 核心 2.6.22 正式發布，這個版本包含了多項重要的硬體支援和效能改進，為各種 Linux 發行版提供了最新的核心基礎。

### 開源資料庫

**MySQL 5.1 發布候選版本**

2007 年 8 月，MySQL 5.1 進入 RC（候選發布）階段。MySQL 5.1 帶來了劃分表（Table Partitioning）、事件排程器、視圖效能改進等重要功能，進一步強化了 MySQL 在企業級應用中的地位。

**PostgreSQL 8.3 開發中**

2007 年 8 月，PostgreSQL 社群正在積極開發 8.3 版本。這個版本將包含 Heikki Linnakangas 的 SSL 改進、多種新資料類型（如 UUID、HTML）、以及對 XML 的原生支援。

### 版本控制系統

**Git 1.5.3 發布**

2007 年 8 月，Git 社群發布了 Git 1.5.3，這是 Linus Torvalds 創建的分散式版本控制系統的持續改進版本。Git 以其優異的效能和分支管理能力，正在逐步取代傳統的 CVS 和 SVN。

**Mercurial 走紅**

與 Git 同時，Mercurial 也獲得了廣泛關注。 Mercurial 以其簡單的介面和跨平台支援，成為許多開發者的選擇。

### 瀏覽器大戰

**Firefox 3 Alpha 階段**

2007 年 8 月，Mozilla 正在緊張開發 Firefox 3，代號「Gran Paradiso」。Alpha 版本的 Firefox 3 帶來了新的 Gecko 1.9 渲染引擎，大幅提升了效能和標準相容性。

**Opera 9.2 發布**

挪威的 Opera 軟體在 8 月發布了 Opera 9.2，持續強化其快速和小巧的特點。

### 行動裝置

**Android SDK 發布**

2007 年 11 月，Google 將正式發布 Android，但 8 月時 SDK 已經開始提供給開發者。這個基於 Linux 核心的開源手機平台，引起了整個科技產業的關注。

### 雲端運算

**Amazon EC2 正式商用**

2007 年 8 月，Amazon Web Services 的 EC2（Elastic Compute Cloud）已經正式商用，為開發者提供了強大的雲端運算能力。

**Google App Engine 預備**

Google 正在積極開發 App Engine，一個讓開發者能夠在 Google 基礎設施上執行 Web 應用的平台。

### 程式語言

**Python 2.6 開發衝刺**

Python 2.6 正在積極開發中，預計將帶來更多的除錯工具和效能改進。

**Ruby 1.9 Preview**

Ruby 1.9 正在準備中，將帶來 YARV 虛擬機器，大幅提升 Ruby 的執行效能。

### 開源大事記

- **OpenOffice.org 2.3**：辦公室軟體持續改進
- **GIMP 2.4**：影像處理軟體新版本
- **Blender 2.4**：3D 動畫軟體改進
- **VLC 0.8.6**：多媒體播放器更新

### 企業動態

- **Red Hat Summit 2007**：企業 Linux 的年度盛會
- **IBM 開源策略**：加強對開源專案的支持
- **Novell 與 Microsoft 合作**：智慧財產權協議
