# 雲端運算的概念：IaaS、PaaS、SaaS

## 雲端運算的定義

雲端運算（Cloud Computing）是通過網路提供運算資源的服務模式。

### 雲端運算的特徵

```
雲端運算的特徵：
─────────────────
1. 按需自助服務
2. 廣泛的網路存取
3. 資源池化
4. 快速彈性擴展
5. 按使用量計費
```

## 雲端服務模型

### 1. IaaS（基礎設施即服務）

```
IaaS 模型：
───────────
┌──────────────────────────────┐
│      應用                    │
├──────────────────────────────┤
│      運行時                  │
├──────────────────────────────┤
│      作業系統                │
├──────────────────────────────┤
│      虛擬化                  │
├──────────────────────────────┤
│      伺服器                  │
├──────────────────────────────┤
│      儲存                    │
├──────────────────────────────┤
│      網路                    │
└──────────────────────────────┘

代表：Amazon EC2, S3
```

```python
# IaaS 使用範例（boto）
import boto.ec2

conn = boto.ec2.connect_to_region('us-east-1')
conn.run_instances('ami-12345', instance_type='t1.micro')
```

### 2. PaaS（平台即服務）

```
PaaS 模型：
───────────
┌──────────────────────────────┐
│      應用                    │
├──────────────────────────────┤
│      運行時                  │
├──────────────────────────────┤
│      作業系統                │  ← 由雲端處理
├──────────────────────────────┤
│      虛擬化                  │
├──────────────────────────────┤
│      伺服器                  │
├──────────────────────────────┤
│      儲存                    │
├──────────────────────────────┤
│      網路                    │
└──────────────────────────────┘

代表：Google App Engine, Heroku
```

```python
# PaaS 使用範例（Google App Engine）
from google.appengine.ext import webapp

class MainPage(webapp.RequestHandler):
    def get(self):
        self.response.out.write('Hello, World!')

app = webapp.WSGIApplication([('/', MainPage)])
```

## 雲端部署模型

### 公用雲（Public Cloud）

```bash
# 公用雲特性
# - 由第三方擁有和管理
# - 通過網路公開存取
# - 多租戶架構
# - 按使用量計費
```

### 私有雲（Private Cloud）

```bash
# 私有雲特性
# - 企業內部運行
# - 更高的控制和安全性
# - 更高的成本
# - 適合有嚴格合規要求的組織
```

### 混合雲（Hybrid Cloud）

```bash
# 混合雲特性
# - 結合公私雲的優勢
# - 核心資料在私有雲
# - 彈性工作負載在公用雲
# - 典型的使用模式
```

## 結語

雲端運算代表了從「擁有基礎設施」到「使用服務」的轉變。這種模式降低了進入門檻，提高了彈性，改變了軟體產業。
