# 未來展望：雲端運算的發展方向

## 2007 年的雲端運算

2007 年，雲端運算剛剛興起：
- Amazon AWS 已上線
- Google App Engine 即將發布
- Microsoft Azure 在開發中

## 未來預測

### 短期（2008-2010）

```
預測：
───────
2008    Google App Engine 發布
2008    Azure 發布
2009    雲端存儲價格持續下降
2010    雲端成為主流
```

### 長期（2010-2015）

```
預測：
───────
2011    雲端 ERP 和 CRM 普及
2012    混合雲成為標準
2013    行動裝置與雲端結合
2014    物聯網與雲端整合
2015    全面雲端化
```

## 雲端的影響

## 結語

2007 年是雲端運算的萌芽期。未來十年，雲端將徹底改變我們使用運算資源的方式。
