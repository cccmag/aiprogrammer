# Silverlight 發布：微軟的回應

## 前言

2007 年，微軟發布了 Silverlight，這是對 Adobe Flash 和 AJAX 趨勢的回應。

## Silverlight 的定位

```
┌────────────────────────────────────────────────────────┐
│              Silverlight 與競爭者                      │
├────────────────────────────────────────────────────────┤
│                                                        │
│  Silverlight：                                        │
│  - .NET 架構                                          │
│  - XAML 介面描述                                      │
│  - 跨瀏覽器、跨平台支援                               │
│                                                        │
│  對比 Flash：                                         │
│  - 更強的.NET 整合                                    │
│  - 更好的視訊品質                                      │
│  - 但生態系較小                                       │
│                                                        │
└────────────────────────────────────────────────────────┘
```

## 主要功能

- 豐富圖形動畫
- 視訊串流
- .NET 程式碼執行

## 結論

Silverlight 展現了微軟在 RIA 領域的野心，但最終被 HTML5 的發展所掩蓋。
