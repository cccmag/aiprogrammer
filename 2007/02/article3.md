# Adobe AIR：跨平台桌面應用

## 前言

2007 年，Adobe 發布了 AIR（Adobe Integrated Runtime），允許開發者使用 Web 技術建立跨平台桌面應用。

## Adobe AIR 的特點

### 支援多種技術

```javascript
// Adobe AIR 支援的技術
AIR_TECHNOLOGIES = {
    "前端": "HTML, CSS, JavaScript",
    "設計": "Flash, Flex, ActionScript",
    "資料": "SQLite 本地資料庫"
};
```

### 離線能力

AIR 應用可以像桌面軟體一樣離線運行，同時具備 Web 技術的開發效率。

## 結論

Adobe AIR 為 Web 開發者打開了桌面應用的大門，促進了 RIA（豐富網路應用）的發展。
