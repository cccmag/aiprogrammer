# 未來展望：Web 的下一步

## 前言

2007 年的 Web 2.0 革命才剛開始。讓我們展望未來十年的發展方向。

## HTML 5 的願景

### 下一代 HTML 標準

2007 年，HTML 5 的討論正在熱烈進行：

```
┌────────────────────────────────────────────────────────┐
│              HTML 5 的新特性                           │
├────────────────────────────────────────────────────────┤
│                                                        │
│  語意標籤：                                           │
│  - <header>, <footer>, <nav>, <article>              │
│                                                        │
│  圖形：                                               │
│  - <canvas> 繪圖                                      │
│  - SVG 內嵌                                           │
│                                                        │

│  表單：                                               │

│  - 表單驗證                                           │

└────────────────────────────────────────────────────────┘
```

## 即時 Web

### WebSocket 與 Comet

2007 年，即時雙向通信的技術正在發展：

- WebSocket 標準制定中
- Comet 長輪詢技術
- 為日後即時應用奠基

## 行動 Web

### 智慧型手機時代的 Web

iPhone 的發表（2007 年 1 月）開啟了行動 Web 的新時代。

## 雲端與 Web

### 所有東西都在雲端

從桌面軟體到 Web 應用，再到雲端服務，軟體正在全面向雲端遷移。

---

## 結論

Web 2.0 開創了一個以使用者為中心、開放協作的網路新時代。這場革命將在未來十年持續深化。
