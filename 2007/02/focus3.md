# 社交媒體的崛起

## 前言

2007 年見證了社交媒體的爆發式成長。YouTube、Facebook、Twitter 等平台正在改變人們分享和獲取資訊的方式。

## Twitter 的微型部落格

### 2007 年的 Twitter

2006 年 3 月推出的 Twitter，在 2007 年初正快速成長：

```python
# Twitter 特色

    "推文限制": "140 字元",
    "簡單性": "極簡介面",
    "即時性": "手機也能發推",
    "開放性": "API 免費開放"
}
```

### Twitter vs Facebook

| 特性 | Twitter | Facebook |
|------|---------|----------|
| 內容長度 | 140 字 | 無限制 |
| 隱私 | 公開為主 | 可設定 |
| 社交模型 | 追蹤者 | 朋友 |
| 用途 | 即時資訊 | 社交 |
