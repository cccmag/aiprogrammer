# PHP 6 開發中：Unicode 與新特性

## 前言

2007 年，PHP 社群正在開發 PHP 6，這是 PHP 語言史上最重要的更新之一。

## PHP 6 的主要特性

### Unicode 支援

```php
<?php
// PHP 6 將全面支援 Unicode
// 這將改善多語言處理能力

// mb_* 函式將成為預設
$str = "你好世界";
echo strlen($str); // 將正確返回字元數
?>
```

### 其他改進

- 移除ereg擴展
- PDO 成為預設資料庫抽象層
- 命名空間支援（後來推遲到 PHP 5.3）

## 結論

PHP 6 將為 PHP 帶來現代化特性，但 Unicode 支援的複雜性導致開發延遲。
