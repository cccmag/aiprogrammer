# Firefox 與開源瀏覽器

## 前言

Firefox 的崛起打破了 IE 的壟斷，為瀏覽器市場帶來了真正的競爭和創新。

## Mozilla 專案的起源

### 從 Netscape 到 Firefox

```
┌────────────────────────────────────────────────────────┐
│            Firefox 發展時間線                          │
├────────────────────────────────────────────────────────┤
│                                                        │
│  1994：Netscape Navigator 發布                        │
│  1998：Netscape 開放原始碼，成立 Mozilla 專案          │
│  2002：Phoenix（Firefox 前身）發布                    │
│  2004：Firefox 1.0 正式發布                          │
│  2006：Firefox 2.0 發布                               │
│                                                        │
│  市場佔有率：                                         │
│  - 2004：~5%                                        │
│  - 2007：~15%                                       │
│                                                        │
└────────────────────────────────────────────────────────┘
```

## Firefox 的創新

### Firefox 對比 IE 的優勢

| 功能 | Firefox | IE |
|------|---------|-----|
| 分頁瀏覽 | 原生支援 | 需 IE7 |
| 擴充套件 | 數千個 | 有限 |
| 安全性 | 更安全 | 漏洞多 |
| 跨平台 | Windows/Mac/Linux | 僅 Windows |
| 標準支援 | 良好 | 落後 |

## 瀏覽器標準之爭

### 開放標準的重要性

Firefox 堅持 W3C 標準，使得網頁可以跨瀏覽器運作。
