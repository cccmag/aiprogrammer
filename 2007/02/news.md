# 本月新知

## 2007 年 2 月 Web 2.0 與網路技術動態

### Web 2.0 熱潮

**Web 2.0 概念持續發酵**

2007 年初，Web 2.0 已成為科技產業最熱門的關鍵字。這個由 O'Reilly Media 創辦人 Tim O'Reilly 在 2004 年提出的概念，正在深刻改變網路的發展方向。

**YouTube 持續爆發式增長**

2006 年被 Google 以 16.5 億美元收購後，YouTube 在 2007 年初每日觀看次數已突破 1 億大關。影片分享文化徹底改變了網路媒體的面貌。

### AJAX 技術成熟

**jQuery 1.0 正式發布**

1 月底，jQuery 1.0 正式發布，這款輕量級 JavaScript 庫迅速獲得開發者青睞。jQuery 簡化了 DOM 操作和 AJAX 調用，讓動態網頁開發變得更加容易。

**Prototype 與 Scriptaculous**

這些 JavaScript 框架在 2007 年初繼續主導前端開發領域。Prototype 提供了類似的擴展，Scriptaculous 則專注於視覺效果和動畫。

**Dojo Toolkit 1.0 準備中**

Dojo 作為一個全面的 JavaScript/DHTML 工具包，正在開發 1.0 正式版本。

### 社交媒體

**Facebook 突破 5000 萬用戶**

2007 年 2 月，Facebook 宣佈用戶數突破 5000 萬，較去年同期增長超過 10 倍。開放平台的政策吸引了大量第三方應用程式。

**Twitter 用戶持續增長**

2006 年 3 月推出的 Twitter，在 2007 年初已擁有數十萬用戶。這種「微型部落格」的概念正在改變人們分享資訊的方式。

**MySpace 仍是社交網路龍頭**

2007 年初，MySpace 以近 6000 萬活躍用戶領先 Facebook，但後者的增長速度令人矚目。

### Mashup 生態

**Google Maps API 的成功**

Google Maps API 催生了無數地圖 Mashup 應用。2007 年初，大量的混搭服務展示了開放 API 的威力。

**Amazon E-Commerce Service**

Amazon 的電子商務 API 允許開發者建立價格比較、產品推薦等 Mashup 應用。

**Yahoo! Pipes 發布**

Yahoo! 推出了革命性的 Web 應用程式，讓非程式設計師也能建立 Mashup。

### 瀏覽器之爭

**Firefox 2.0 市場佔有率上升**

Firefox 的市場佔有率在 2007 年初突破 20%，對 Internet Explorer 的主導地位形成挑戰。

**IE7 的影響**

Microsoft 推出的 IE7 帶來了分頁瀏覽、 RSS 支援等新功能，但許多使用者仍選擇堅守 Firefox 或跳槽到 Opera。

### 網路服務與雲端

**Amazon Web Services 持續擴張**

S3 儲存服務和 EC2 運算服務在 2007 年初獲得越來越多開發者採用。

**Google App Engine 預告**

Google 預告即將推出 App Engine 平台，這將進一步降低網路應用開發的門檻。

### 標準與規範

**HTML 5 草案準備中**

W3C 開始討論 HTML 5 規範，試圖為下一世代網頁技術訂定標準。

**ECMAScript 4 討論中**

JavaScript 的下一版本正在熱烈討論中，計劃加入類別、包裝等新特性。
