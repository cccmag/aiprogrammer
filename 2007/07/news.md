# 本月新知

## 2007 年 7 月程式與科技技術動態

### 重大事件

**jQuery 1.1 正式發布**

2007 年 7 月，jQuery 團隊發布了 jQuery 1.1 版本，這是自 2006 年 8 月 jQuery 1.0 發布以來最重要的更新。jQuery 1.1 大幅簡化了 DOM 操作，使得選擇器、事件處理和動畫效果變得前所未有的簡潔。新版本在效能上也有了顯著提升，選擇器速度平均提升了 3 倍。jQuery 的「Write Less, Do More」理念開始席捲前端開發界。

**Internet Explorer 8 Beta 1 發布**

2007 年 7 月，Microsoft 發布了 Internet Explorer 8 Beta 1，這是 IE 團隊回應 Web 標準相容性的重要一步。IE8 在 CSS 標準支援上有了明顯改善，同時引入了 Activity Streams、WebSlice 等新功能。開發者社群對此反應熱烈，期待微軟能夠提供更好的 Web 標準支援。

### JavaScript 框架生態

**Dojo Toolkit 1.0 發布**

2007 年 7 月，Dojo Foundation 發布了 Dojo Toolkit 1.0。這是一個全面的 JavaScript 框架，涵蓋了從 DOM 操作、事件處理到 UI 元件、資料存取、動畫效果等各方面。Dojo 的模組化設計和豐富的 UI 元件庫使其成為企業級 Web 應用的熱門選擇。

**Prototype 1.6.0 發布**

2007 年 7 月，Prototype JavaScript framework 發布了 1.6.0 版本。Prototype 以其優雅的原型繼承和函式鏈式調用聞名，其 `$$()` 選擇器和 `Ajax` 對象對前端開發產生了深遠影響。

**YUI 2.3.0 發布**

Yahoo! 在 2007 年持續更新 YUI (Yahoo! User Interface Library)，7 月發布的 2.3.0 版本增強了 DataTable、Calendar 等元件的功能。YUI 以其完善的文檔和企業級品質，成為大型 Web 應用的首選框架。

### 瀏覽器與引擎

**Safari 3 Beta 發布 for Windows**

2007 年 7 月，Apple 發布了 Safari 3 的 Windows Beta 版本。這是 Safari 首次登陸 Windows 平台，顯示 Apple 希望將 Safari 打造成跨平台的瀏覽器。Safari 3 以其快速的 JavaScript 執行速度（使用 Nitro 引擎）引起了開發者關注。

**Firefox 3 Alpha 階段**

2007 年 7 月，Mozilla 正在積極開發 Firefox 3，代號「Gran Paradiso」。Firefox 3 將帶來全新的 Gecko 1.9 渲染引擎和優化的 JavaScript 引擎（TraceMonkey 前身），效能將有大幅提升。

### 網頁技術標準

**ECMAScript 4 草案討論**

2007 年 7 月，TC39 委員會持續討論 ECMAScript 4 的規範草案。新版本將引入類別系統、模組、迭代器、生成器等新特性，為 JavaScript 帶來更強大的面向物件程式設計能力。

**HTML 5 草案成熟**

W3C 的 HTML 5 工作小組在 2007 年 7 月持續推進草案制定。HTML 5 引入了 `<video>`、`<audio>`、`<canvas>` 等新元素，以及豐富的 API，為 Web 應用開發開啟了新篇章。

### 伺服器端技術

**Apache 2.2.4 發布**

2007 年 7 月，Apache HTTP Server 發布了 2.2.4 版本，修復了多個安全漏洞並提升了效能。

**Lighttpd 1.4.18 發布**

Lighttpd 這個輕量級 Web 伺服器在 7 月發布更新，持續最佳化效能和穩定性。

### 開發工具

**Eclipse 3.3 Europa 發布**

2007 年 7 月，Eclipse 發布了代号「Europa」的 3.3 版本。這是 Eclipse 歷史上最大規模的同步發布，一口氣發布了 21 個專案的新版本。Europa 為 Java 開發者提供了更現代化的 IDE 環境。

**NetBeans 6.0 Beta 發布**

Sun Microsystems 在 2007 年 7 月發布了 NetBeans 6.0 Beta，強化了對 Ruby、PHP 和 C++ 的支援。

### 業界動態

- **微軟收購 aQuantive**：線上廣告技術投資
- **Yahoo! 發布 BOSS API**：搜尋開放平台策略
- **Adobe 發布 AIR Beta 2**：跨平台桌面應用 Runtime
- **Sun 發布 OpenJDK**：開源 Java 計畫推進

### 程式語言

- **Python 2.6 開發中**：效能改進和除錯工具強化
- **Ruby 1.8.6 發布**：Ruby 1.9 漸近
- **PHP 6 規劃中**：Unicode 支援和名稱空間

---

*本期特別報導：2007 年 7 月是 JavaScript 框架生態蓬勃發展的月份，jQuery 1.1、Dojo 1.0 的相繼發布，標誌著前端開發進入了一個新的時代。這些框架以簡化 DOM 操作為核心，大幅提升了開發效率，為 Rich Internet Applications 的發展奠定了基礎。*
