# 本月新知

## 2007 年 6 月版本控制與程式技術動態

### 版本控制重大事件

**Git 1.5.0 正式發布**

2007 年 6 月，Git 1.5.0 正式發布。這是 Git 發展史上的重要里程碑。Git 1.5 加入了许多新功能和效能改進，包括更快的 SHA-1 計算、更好的歸併支援、以及改進的 Git Web 介面。

**Git 的快速發展**

從 2005 年 Linux Torvalds 創造 Git 開始，Git 在兩年內經歷了快速發展。2007 年，越來越多的專案開始採用 Git，包括 Linux 核心、Ruby on Rails、Android 專案等。

### 其他版本控制新聞

**Subversion 1.4.4 發布**

Apache Subversion 社群發布了 1.4.4 版本，修復了多個錯誤。Subversion 仍是企業版本控制的主流選擇。

**Mercurial 1.0 開發中**

另一個分散式版本控制系統 Mercurial 正在開發 1.0 版本。

### 程式語言與框架

**Python 2.5 正式發布**

Python 2.5 在 2006 年 9 月發布後持續獲得採用。新聞包括 Django 對 Python 2.5 的支援最佳化。

**Java SE 6 持續普及**

Java SE 6（代號 Mustang）自 2006 年 12 月發布後，企業應用正在快速升級。

### 開發工具

**Eclipse 3.3 Europa 發布**

2007 年 6 月，Eclipse 基金會發布了 Europa 版本。這是 Eclipse 的年度同步發布，包含了多個專案的更新。

**Git GUI 工具興起**

隨著 Git 的普及，各種 Git GUI 工具開始出現，包括 gitk、git-gui、以及第三方工具。

### 業界動態

- **Google Code 採用 Subversion**：Google Code 托管服務支援 Subversion
- **GitHub 測試版**：GitHub 正在開發中，將於 2008 年正式上線
- **SourceForge 持續服務**：SourceForge 仍是開源專案的主要托管地
