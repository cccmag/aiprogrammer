# Linus Torvalds 專訪：Git 的創造故事

## 前言

2005 年，Linus Torvalds 創造了 Git。以下是他對 Git 設計理念的阐述。

## 為什麼要創造 Git

Linus 的理由：
1. BitKeeper 授權問題
2. 需要更快的工具
3. 需要更好的分支和歸併

## Git 的核心設計

```bash
# Git 的設計目標
# 1. 速度
# 2. 簡單設計
# 3. 強力支援非線性開發
# 4. 完全分散式
# 5. 能處理大型專案（Linux 核心）
```

## 結語

Git 的成功證明了開放原始碼社群的力量。
