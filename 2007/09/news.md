# 本月新知

## 2007 年 9 月程式與科技技術動態

### 重大事件

**MySQL 5.1 RC2 發布**

2007 年 9 月，MySQL 5.1 發布了 RC2（候選發布 2）版本。這個版本包含了多項重要的新功能，距離正式發布越來越近。MySQL 5.1 的劃分表功能、事件排程器和效能改進，使其成為企業級應用的有力選擇。

**PostgreSQL 8.3 開發衝刺**

2007 年 9 月，PostgreSQL 社群正在為 8.3 版本的最終發布做準備。這個版本將包含多項企業級功能，包括對 XML 的原生支援、UUID 類型和向量索引增強。

### NoSQL 興起

**CouchDB 0.8 發布**

2007 年 9 月，CouchDB 發布了 0.8 版本，這是一個面向文件的 NoSQL 資料庫，使用 JSON 儲存文件，RESTful API 進行訪問。CouchDB 的「離線優先」設計和靈活的文件結構，引起了開發者的廣泛關注。

**MongoDB 早期發展**

2007 年，MongoDB 正在積極開發中。其「文件資料庫」的概念，挑戰了傳統關聯式資料庫的設計理念。

### 商業資料庫

**Oracle 11g 發布準備**

Oracle 11g 正式發布在即，這是 Oracle 資料庫的重大更新版本，將帶來更多企業級功能和效能改進。

**IBM DB2 9.5 發布**

2007 年，IBM 發布了 DB2 9.5，引入了 pureXML 技術，提供原生的 XML 資料處理能力。

### 雲端資料庫

**Amazon SimpleDB 服務**

2007 年，Amazon Web Services 推出了 SimpleDB，這是一個雲端 NOSQL 資料庫服務，為開發者提供了簡單的資料儲存和查詢能力。

### 開源趨勢

**SQLite 3.4 發布**

SQLite 持續演化，3.4 版本帶來了更多的 SQL 標準支援和效能改進。

**Firebird 2.0 發布**

這個開源 SQL 資料庫發布了新版本，增強了其作為替代方案的吸引力。

### 開發工具

**MySQL Workbench 開發中**

MySQL 的視覺化工具正在積極開發，為資料庫設計和管理提供了更現代化的介面。

**phpMyAdmin 2.11 發布**

Web 介面的 MySQL 管理工具持續更新，改善了使用體驗。

### 業界動態

- **Sun 收購 MySQL 談判** -- 預計 2008 年初完成收購
- **PostgreSQL 社群成長** -- 更多的企業採用
- **NoSQL 會議** -- 首次 NoSQL 研討會舉辦

### 技術標準

- **SQL:2003 標準** -- 已被多個資料庫支援
- **ACID 特性** -- 仍然是資料庫的黃金標準
- **MVCC** -- 多版本並發控制越來越普及

---

*本期特別報導：2007 年 9 月是資料庫技術發展的重要時刻。傳統關聯式資料庫繼續強化企業功能，而新興的 NoSQL 資料庫開始嶄露頭角。這個月份的技術發展預示著資料庫領域未來十年的多樣化趨勢。*
