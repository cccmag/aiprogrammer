# 結語

## 從單一 Agent 到 Agent 生態系

多代理系統不只是一種技術模式，更是一種思考 AI 應用的新方式。當我們不再試圖用一個模型解決所有問題，而是像組建人類團隊一樣組建 Agent 團隊時，整個設計範式都改變了。

### 關鍵學習

1. **專業化帶來效能** — 一個專精的 Agent 遠勝一個萬能但平庸的 Agent。角色定義的品質決定了系統的表現。

2. **溝通是核心** — 多 Agent 系統的瓶頸往往不是模型能力，而是通訊效率。訊息格式、協調策略、非同步模式都是關鍵設計決策。

3. **安全不容忽視** — 多 Agent 系統引入了新的攻擊面。跨 Agent 注入、權限提升、資料外洩——這些威脅需要從架構層面解決。
