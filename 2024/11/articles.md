# 文章集錦

## DevOps 與自動化部署專輯

### 程式相關（10 篇）

#### 1. [DevOps vs 傳統維運](article1.md)

探討 DevOps 與傳統維運模式的根本差異。從組織文化、工具鏈、流程自動化等面向比較兩者優劣。包含實際案例說明 DevOps 轉型如何提升部署頻率和系統穩定性。

#### 2. [Dockerfile 撰寫](article2.md)

深入講解 Dockerfile 的撰寫技巧與最佳實踐。從基礎指令到多階段構建，從快取機制到映像瘦身，涵蓋生產環境 Dockerfile 的完整實戰指南。

#### 3. [Docker 常用指令](article3.md)

整理 Docker 日常開發與維運中最常用的指令。從基本容器管理到網路、儲存操作，每個指令附有實例說明。適合作為 Docker 指令速查手冊。

#### 4. [docker-compose 配置](article4.md)

詳細介紹 Docker Compose 的配置語法與應用場景。從服務定義、網路配置到環境變數管理，涵蓋開發和生產環境的 Compose 配置最佳實踐。

#### 5. [GitHub Actions 工作流](article5.md)

實戰講解 GitHub Actions 的工作流配置。包含事件觸發、矩陣構建、快取優化、Docker 映像建置與部署。提供多個可直接套用的工作流範本。

#### 6. [自動化測試整合](article6.md)

探討如何在 CI/CD 管線中整合各類測試。從單元測試到整合測試、E2E 測試，再到程式碼品質檢查和安全掃描。涵蓋測試覆蓋率閘道和測試報告最佳實踐。

#### 7. [部署策略：藍綠、滾動](article7.md)

深入分析藍綠部署和滾動更新兩種主流部署策略。比較各自優缺點、適用場景和實作方式。包含 Kubernetes 和 Docker Compose 的實戰配置範例。

#### 8. [PM2 行程管理](article8.md)

介紹 PM2 行程管理工具在 Node.js 應用部署中的應用。從基本行程管理到叢集模式、零停機部署、日誌管理和監控。適合 Node.js 生產環境部署參考。

#### 9. [ELK 日誌系統](article9.md)

詳細介紹 ELK（Elasticsearch、Logstash、Kibana）日誌系統的架構與部署。從日誌收集、解析到視覺化展示，涵蓋 Docker 環境下的完整 ELK 堆疊配置。

#### 10. [Terraform 入門](article10.md)

基礎設施即程式碼的入門指南。從 Terraform 的基本概念到實戰應用，包含 Provider 配置、資源定義、狀態管理和模組化設計。以 AWS 為例展示完整專案。
