## 值得關注的開源專案

### Bun 1.x
JavaScript 執行環境的顛覆者，2024 年達到重大穩定性里程碑。

### Biome
前身是 Rome 工具的 eslint/prettier 替代方案，採用 Rust 撰寫。

```javascript
// Biome 設定範例
const biomeConfig = {
  $schema: './node_modules/@biomejs/biome/configuration_schema.json',
  organizeImports: { enabled: true },
  linter: {
    enabled: true,
    rules: {
      recommended: true,
      style: {
        useConst: 'error',
        noVar: 'error'
      }
    }
  },
  formatter: {
    enabled: true,
    indentStyle: 'space',
    indentWidth: 2,
    lineWidth: 100
  }
};

console.log('Biome 設定已載入');
```

### Deno 2

### KittyCAD
開源硬體設計工具，展現了開源在非軟體領域的潛力。
