# 本月新聞 — 資料庫與雲端服務

## AWS 推出新一代 Amazon RDS 最佳化執行個體

AWS 於本月宣布新一代 RDS 最佳化執行個體正式上線，支援 MySQL 8.4 與 PostgreSQL 16 的效能提升達 40%。新執行個體採用 AWS Graviton3 處理器，提供更低的延遲與更高的吞吐量，適合高負載的資料庫工作負載。詳情請參考 https://www.google.com/search?q=AWS+RDS+Graviton3+performance+2024

## MongoDB 8.0 正式發布

MongoDB 8.0 版本帶來查詢效能的大幅提升，包括全新的查詢最佳化引擎與水平擴展增強。新版本支援更精細的資料分片策略，以及改良的變更串流 (Change Streams) 功能，適合即時資料處理場景。詳情請參考 https://www.google.com/search?q=MongoDB+8.0+release+2024

## Redis 7.4 新增向量搜尋功能

Redis 7.4 版本新增向量搜尋模組，支援以 RediSearch 進行相似度查詢，為 AI 應用中的嵌入向量檢索提供高效解決方案。此功能讓 Redis 不僅是快取層，更能作為輕量級向量資料庫使用。詳情請參考 https://www.google.com/search?q=Redis+7.4+vector+search+2024

## Firebase 推出 Data Connect 服務

Google Firebase 推出 Data Connect，這是一項整合 Cloud SQL 與 Firebase 的新服務，讓開發者可以透過 GraphQL 操作關聯式資料庫，同時享有 Firebase 的即時同步能力。此服務大幅簡化了後端開發流程。詳情請參考 https://www.google.com/search?q=Firebase+Data+Connect+2024

## Cloudflare 發表 D1 資料庫正式版

Cloudflare 正式推出 D1 資料庫，這是一個建立在 Workers 邊緣運算平台上的全球分散式 SQLite 資料庫。D1 支援自動複寫與低延遲讀取，適合邊緣運算場景中的資料儲存需求。詳情請參考 https://www.google.com/search?q=Cloudflare+D1+database+GA+2024

## Prisma 5 新增關係查詢快取

Prisma ORM 在 5.3 版本中引入關係查詢快取功能，大幅減少 N+1 查詢問題的影響。開發者可以透過簡單的設定啟用快取策略，顯著提升 GraphQL 與 REST API 的資料載入效能。詳情請參考 https://www.google.com/search?q=Prisma+5+relation+cache+2024

## PostgreSQL 17 發布候選版

PostgreSQL 全球開發團隊發布 17 版本候選版，新增增量備份、改良的平行查詢執行，以及更好的 JSON 資料處理能力。這些新功能讓 PostgreSQL 在處理現代應用資料時更加靈活高效。詳情請參考 https://www.google.com/search?q=PostgreSQL+17+release+candidate+2024

## 雲端成本持續受關注

根據最新報告，企業雲端支出在 2024 年持續增長，如何有效管理雲端成本已成為 CIO 最關心的議題之一。多家廠商推出 FinOps 解決方案，幫助企業在雲端資源使用與成本之間取得最佳平衡。詳情請參考 https://www.google.com/search?q=cloud+cost+optimization+2024+FinOps
