# 2018 年 1 月號新聞摘要

## Python 生態系

- Python 3.6 在 2017 年 12 月發布 3.6.4 版本，修復多項 bug 並提升效能。f-string 格式化已成為 Python 3.6 最受歡迎的新特性，社群採用率持續上升。

- Python Package Index (PyPI) 在 2018 年 1 月已收錄超過 13 萬個套件，為歷年來最高。機器學習相關套件的下載量顯著成長。

- Python 軟體基金會宣布 2018 年將加強對 Python 2.7 的安全維護，但仍建議開發者遷移至 Python 3.x。

## 人工智慧與機器學習

- Google 發布 TensorFlow 1.5 版本，新增 CUDA 9 支援與 TensorFlow Lite 預覽版。Keras 已成為 TF 官方高階 API，佔 TF 使用者約 60%。

- PyTorch 0.3.0 發布，強化了 TorchScript 與模型部署能力。Facebook 宣布 PyTorch 在研究社群的使用率已超越 TensorFlow。

- Amazon SageMaker 在 1 月推出新功能簡化模型訓練流程，降低機器學習入門門檻。

## 資訊安全

- Intel 處理器爆發「Meltdown」與「Spectre」漏洞，影響大多數現代處理器。Google Project Zero 揭露此問題後，各家雲端服務商緊急修補。

- Python 官方呼籲開發者檢查 requirements.txt 中的依賴套件安全性，建議使用 pip-audit 工具掃描已知漏洞。

## 開發者工具

- GitHub 宣佈全球開發者數量突破 2500 萬，2017 年新增開源專案超過 800 萬個。GitHub Education 計畫提供免費工具給學校使用。

- Visual Studio Code 1.19.2 發布，修復多項穩定性問題，Python 擴展下載量突破 1000 萬次。

- Docker 在 1 月發布 Enterprise Edition 2.0，簡化容器編排流程。

## 雲端與基礎設施

- AWS 發布 2017 年第四季財報，雲端服務營收成長 45%。Lambda 無伺服器服務使用量大幅增加。

- Microsoft Azure 推出新區域，擴展全球雲端基礎設施。Azure Functions 正式支援 Python。
