# 2018 年 1 月號結語

## 本月回顧

2018 年的第一期雜誌，我們聚焦於 **Python 3.6 生態系**，這個版本在 2016 年 12 月發布後，經過一年的沉澱，到了 2018 年 1 月已經成為 Python 社群的主流選擇。

### 重點回顧

1. **f-string 格式化** - 大幅提升字串格式化的可讀性與效能
2. **型別提示** - 為 Python 帶來靜態型別檢查的可能性
3. **async/await** - 完善了 Python 的異步程式設計能力
4. **NumPy 與 Matplotlib** - 資料科學與視覺化的基礎工具

## 學習建議

### Python 程式設計師

如果你還沒有遷移到 Python 3.x，建議在新的一年裡開始這個過渡。Python 2.7 的支援將在 2020 年結束，現在是開始準備的最佳時機。

### 機器學習愛好者

NumPy 是所有機器學習庫的基礎。建議深入理解矩陣運算，這將幫助你更好地理解深度學習框架的內部運作。

## 下期預告

2018 年 2 月號將介紹 **版本控制與 Git**，這是每位程式設計師必備的技能。無論你是獨立開發者還是團隊成員，Git 都能幫助你更好地管理程式碼版本。
