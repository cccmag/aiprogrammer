# 預訓練的價值

## 為何預訓練有效？

預訓練語言模型的成功來自於以下幾個關鍵洞察：

### 1. 語言的通用性

語言本身包含了大量通用知識：
- **語法結構**：主詞動詞一致、名詞單複數等
- **語義關係**：同義詞、反義詞、上下位關係
- **世界知識**：事實、概念、因果關係

這些知識可以從大規模無標注語料中學習。

### 2. 遷移學習的力量

在大型資料集上學習的表示可以遷移到其他任務：
- ImageNet 預訓練改變了電腦視覺
- 大型語言模型預訓練正在改變 NLP

遷移學習減少了對特定任務資料的依賴。

### 3. 深層表示的層級性

大型神經網路的不同層級編碼不同類型的資訊：
- **淺層**：詞彙級特徵、語法結構
- **深層**：語義關係、語用推理

預訓練讓模型學習到豐富的層級表示。

## 預訓練任務的設計

有效的預訓練任務應該：
1. **有挑戰性**：需要理解語言才能完成
2. **任務無關**：學到的知識適用於多種下游任務
3. **可規模化**：可以使用大規模無標注資料

### 語言模型任務
- 預測下一個詞（GPT）
- 預測被遮蓋的詞（BERT MLM）

### 句子層級任務
- 判斷句子關係（BERT NSP）
- 句子順序預測

## 預訓練的規模效應

研究發現，模型規模與資料規模對效能有顯著影響：
- **更大的模型**：通常效果更好
- **更多的資料**：更大的提升
- **更長的訓練**：收益遞減

但也帶來計算成本與部署挑戰。

## 何時需要預訓練？

| 場景 | 建議 |
|------|------|
| 有大量任務相關資料 | 可能不需要微調 |
| 資料有限 | 強烈建議使用預訓練模型 |
| 任務是新領域 | 使用領域適配預訓練 |
| 生產環境延遲敏感 | 考慮模型壓縮 |

## 預訓練的局限性

預訓練不是萬能的：
- **領域差距**：預訓練語料與目標任務可能不同
- **任務不匹配**：預訓練目標與下游任務可能有差異
- **計算成本**：預訓練需要大量資源
