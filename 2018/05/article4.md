# GitHub 被微軟收購

## 前言

2018 年 6 月 4 日，微軟宣佈以 75 億美元收購 GitHub。這是開源生態的重大事件。

## 收購詳情

- 收購價：75 億美元
- GitHub 保持獨立營運
- GitHub CEO Chris Wanstrath 辭職

## 社群反應

擔憂：
- 微軟會改變 GitHub 的中立性？
- 擔心專案被鎖定

樂觀：
- 微軟對開源的支援已經改變
- GitHub 可以獲得更多資源

## 後續發展

收購完成後，GitHub 持續創新，推出了 GitHub Actions 等新功能。

## 結論

這次收購對開源生態的影響是利大於弊的。
