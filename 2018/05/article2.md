# Node.js 10.0 發布

## 前言

2018 年 4 月，Node.js 10.0 正式發布，這是 Node.js 的重要里程碑。

## 主要更新

### N-API 穩定化

N-API 提供了穩定的 C/C++ API 用於開發原生附加元件，不再依賴 V8 內部 API。

### Error Code 改進

錯誤現在包含有意義的錯誤碼，方便程式化處理。

### npm 6.0

整合了最新的 npm 6.0，效能大幅提升。

## 結論

Node.js 10.0 為未來的 LTS 版本奠定了基礎。
