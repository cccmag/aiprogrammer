# 本月新知

## 2018 年 5 月程式與 AI 技術動態

### 程式語言與框架

**Rust 1.26 穩定版發布**

Mozilla 開發的系統程式語言 Rust 發布 1.26 穩定版。新增特點包括：
- `impl Trait` 穩定化
- 支援 128 位元整數類型
- 改進了 pattern matching
- 更快的編譯速度

**Node.js 10.0 發布**

Node.js 10.0 帶來了多項改進：
- N-API 穩定化
- 錯誤處理改善
- npm 6.0 整合
- HTTP/2 效能提升

**React 16.3 發布**

Facebook 發布 React 16.3，引入了新的生命週期方法和 Context API。這是 React 架構的重要更新，為未來版本鋪路。

**GitHub 被微軟收購**

2018 年 6 月 4 日，微軟正式宣佈以 75 億美元收購 GitHub。這一收購引發了開源社群的一些討論，但多數開發者持樂觀態度。

### AI 與機器學習

**PyTorch 0.4 發布**

PyTorch 0.4 是該框架的重要版本：
- 整合了 Caffe2
- 改進了 JIT 編譯器
- 更好的 Windows 支援
- 張量運算優化

**Google I/O 2018 AI 進展**

Google 在 I/O 2018 上展示了多項 AI 進展：
- Google Assistant 的 Duplex 技術
- Smart Compose 功能
- ML Kit 行動端 ML 工具
- TensorFlow.js 改進

**word2vec 與詞嵌入的應用**

詞嵌入技術（word2vec, 2013）在 2018 年已經相當成熟。Google、Facebook 等公司都在使用詞嵌入來處理自然語言。

**CNN 在影像辨識的突破**

卷積神經網路（CNN）持續在影像辨識領域取得突破。ResNet（2015）等架構已經成為標準。

**LSTM 在序列處理中的應用**

長短期記憶網路（LSTM）在機器翻譯、語音辨識等序列處理任務中表現出色。

### 開發工具與雲端服務

**Visual Studio Code 普及**

VS Code 在 2018 年持續獲得開發者青睞，成為最受歡迎的程式編輯器之一。

**AWS 發布新 ML 工具**

Amazon 在本月發布了多項 ML 相關服務，包括 SageMaker 的新功能和 Rekognition 的改進。

### 業界動態

- **Facebook F8 開發者大會**：展示了 AR/VR 和 AI 的最新進展
- **IBM 發布 AI 工具**：Watson 平台持續優化
- **Tesla 自動駕駛進展**：展示最新的自動駕駛技術
- **騰訊 AI Lab 論文**：在頂級 AI 會議發表多篇論文

### 開源專案

- **Keras 2.2.0 發布**：增強了功能性和文件
- **ONNX 格式獲得更多支援**：成為模型交換的標準格式
- **LLVM 6.0.1 發布**：改進了編譯器基礎設施

### 未來展望

類神經網路和深度學習正在改變各行各業。TensorFlow、PyTorch、Keras 等框架的成熟，讓更多人能夠接觸和使用深度學習技術。
