# 常見架構與應用：CNN、RNN、LSTM 的基礎

## 前言

不同的任務需要不同的網路架構。本章將介紹三種最重要的類神經網路架構：CNN、RNN 和 LSTM。

## 卷積神經網路（CNN）

### 核心概念

- **卷積層**：使用濾波器提取特徵
- **池化層**：減少空間維度
- **全連接層**：分類或迴歸

### 經典架構

- LeNet (1998)
- AlexNet (2012) - ImageNet 突破
- VGGNet (2014)
- ResNet (2015) - 殘差連接

### 應用場景

- 影像分類
- 物體偵測
- 人臉識別
- 醫學影像分析

## 循環神經網路（RNN）

### 結構特點

RNN 有循環連接，可以處理序列資料：

```
┌─────────────────────────────────────────────────────┐
│                   RNN 展開圖                         │
├─────────────────────────────────────────────────────┤
│                                                     │
│   x_t ──► h_t ──► y_t                              │
│           │                                         │
│           ▼                                         │
│         x_{t+1} ──► h_{t+1} ──► y_{t+1}            │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 問題

標準 RNN 難以處理長期依賴（梯度消失問題）。

## 長短期記憶網路（LSTM）

### 解決方案

LSTM 使用門控機制來控制資訊流動：

```
┌─────────────────────────────────────────────────────┐
│                   LSTM 結構                         │
├─────────────────────────────────────────────────────┤
│                                                     │
│   遺忘門：決定忘記什麼                               │
│   輸入門：決定記住什麼                               │
│   輸出門：決定輸出什麼                               │
│                                                     │
│   長期記憶（C_t）穿過每個時間步                       │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 公式

```python
# 遺忘門
f = sigmoid(W_f * [h_{t-1}, x_t] + b_f)

# 輸入門
i = sigmoid(W_i * [h_{t-1}, x_t] + b_i)
C_tilde = tanh(W_C * [h_{t-1}, x_t] + b_C)

# 更新細胞狀態
C_t = f * C_{t-1} + i * C_tilde

# 輸出門
o = sigmoid(W_o * [h_{t-1}, x_t] + b_o)
h_t = o * tanh(C_t)
```

### 應用

- 機器翻譯
- 語音辨識
- 文字生成
- 影片分析

## 結語

CNN、RNN 和 LSTM 是深度學習的三種基石架構。選擇哪種架構取決於你的任務型別。
