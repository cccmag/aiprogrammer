# PyTorch 0.4 發布

## 前言

PyTorch 0.4 是該框架的重要版本，於 2018 年 5 月發布。

## 主要更新

### 合併 Caffe2

PyTorch 0.4 正式整合了 Caffe2，提供了更穩定的骨幹架構。

### 0.5.0 的預覽功能

包括更好的 JIT 編譯器和 TorchScript。

### Windows 支援改善

PyTorch 在 Windows 上的穩定性大幅提升。

### Tensor 和 Variable 合併

簡化了張量操作 API。

## PyTorch 的優勢

- 動態計算圖
- 調試方便
- 研究首選框架
- 活躍的社群

## 結論

PyTorch 正在成為深度學習研究的首選框架。
