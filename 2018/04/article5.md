# Docker 18.03 整合 Kubernetes

## 前言

2018 年，Docker 正式在 Docker Desktop 中整合 Kubernetes，讓容器編排變得前所未有的簡單。

## 整合內容

### 單一平臺

開發者現在可以在同一個平臺上：
- 使用 Docker Compose 定義本地開發環境
- 使用 Docker Swarm 或 Kubernetes 部署到生產環境

### kubectl 整合

Docker Desktop 包含了完整的 kubectl，讓你可以直接管理叢集。

## Kubernetes vs Docker Swarm

| 特性 | Kubernetes | Docker Swarm |
|------|------------|--------------|
| 學習曲線 | 較陡 | 較平緩 |
| 功能豐富度 | 更豐富 | 基本 |
| 產業採用 | 更廣泛 | 較少 |

## 結論

對於大多數團隊來說，Docker Desktop 整合 Kubernetes 是一個很好的起點。
