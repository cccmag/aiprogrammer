# 機器學習概述：什麼是機器學習？

## 前言

機器學習（Machine Learning）是人工智慧的核心技術之一，它使電腦能夠從資料中學習，而不需要明確地編寫程式邏輯。這個概念看似簡單，卻徹底改變了我們對電腦程式的理解。

## 什麼是機器學習？

### 經典定義

Tom Mitchell 在 1997 年提出的定義：
> 「一個程式在某項任務 T 上，隨著經驗 E 的增加，其效能 P 也隨之提升。」

舉例來說，一個垃圾郵件辨識系統：
- 任務 T：辨識垃圾郵件
- 經驗 E：過去標註過的郵件資料
- 效能 P：正確分類郵件的比例

## 機器學習的三大類型

### 1. 監督式學習（Supervised Learning）

監督式學習是最常見的機器學習類型。訓練資料包含輸入和正確輸出（標籤）。演算法學習輸入到輸出的映射關係。

**應用場景**：
- 電子郵件分類（垃圾郵件 vs 正常郵件）
- 房價預測
- 疾病診斷
- 圖像辨識

**經典演算法**：
- 線性迴歸（Linear Regression）
- 邏輯斯迴歸（Logistic Regression）
- 決策樹（Decision Tree）
- 隨機森林（Random Forest）
- 支援向量機（Support Vector Machine, SVM）
- 類神經網路（Neural Network）

### 2. 非監督式學習（Unsupervised Learning）

非監督式學習的訓練資料沒有標籤。演算法需要自己發現資料中的結構和模式。

**應用場景**：
- 客戶分群
- 異常偵測
- 推薦系統
- 維度削減

**經典演算法**：
- K-means 分群
- 階層分群（Hierarchical Clustering）
- 主成分分析（PCA）
- 自組織映射（SOM）
- 關聯規則（Association Rules）

### 3. 強化學習（Reinforcement Learning）

強化學習的學習方式是透過與環境互動，代理人（Agent）根據獎勵信號學習最佳策略。

**應用場景**：
- 遊戲 AI（AlphaGo、Atari 遊戲）
- 機器人控制
-自動駕駛
- 推薦系統

**經典演算法**：
- Q-Learning
- SARSA
- Deep Q Network (DQN)
- Policy Gradient
- Actor-Critic

## 機器學習的發展歷程

```
1950s ── 類神經網路的誕生
  │
  │  McCulloch-Pitts 神經元模型
  │  感知器（Perceptron）的發明
  │
1960s ── 機器學習的理論基礎
  │
  │  感知器收斂定理
  │  類神經網路的第一次熱潮
  │
1970s ── AI 寒冬
  │
  │  感知器的限制被證明（Minsky, 1969）
  │  類神經網路研究陷入停滯
  │

  │  反向傳播演算法（Backpropagation）

  │
1990s ── 統計學習的興起

  │  支援向量機（SVM）

  │  提升方法（Boosting）
  │
2000s ── 大數據與深度學習

  │  Hadoop、Spark 等大數據工具

  │  卷積神經網路（CNN）
  │

  │
```

## 機器學習的應用領域

### 電腦視覺
- 圖像分類、物體偵測
- 人臉識別、指紋識別
- 醫學影像分析
- 自駕車視覺系統

### 自然語言處理
- 文字分類、情感分析
- 機器翻譯
- 智慧客服
- 文字生成

### 推薦系統
- 電商推薦、音樂推薦
- 影片推薦、新聞推薦
- 個人化搜尋

### 遊戲與強化學習
- 遊戲 AI
- 棋類程式（AlphaGo）
- 機器人控制

## 機器學習的挑戰

1. **資料品質**：Garbage in, garbage out
2. **過擬合**：模型在訓練資料上表現良好，但在新資料上表現差
3. **解釋性**：深度學習模型是「黑盒子」
4. **計算資源**：大型模型需要大量 GPU/TPU 運算
5. **隱私問題**：訓練資料可能包含敏感資訊

## 結語

機器學習已經成為現代軟體開發不可或缺的技能。從 scikit-learn 的簡單工具到 TensorFlow 的深度學習框架，現在是學習機器學習最好的時代。
