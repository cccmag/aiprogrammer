# 自動機器學習（AutoML）興起

## 前言

AutoML 是機器學習領域的一個重要趨勢，旨在自動化模型設計和超參數調整的過程。

## 主要方法

### 1. 類神經網路架構搜尋（NAS）

自動搜尋最佳的網路架構：

- RNN 架構
- CNN 架構
- Transformer 架構

### 2. 超參數優化

自動調整模型的超參數：

- 網格搜索（Grid Search）
- 隨機搜索（Random Search）
- 貝葉斯優化（Bayesian Optimization）

### 3. 自動化特徵工程

自動生成和選擇有用的特徵。

## 工具和框架

| 工具 | 開發者 | 特點 |
|------|--------|------|
| Auto Keras | 恩智浦實驗室 | 開源、基於 Keras |
| H2O AutoML | H2O.ai | 自動訓練多個模型 |
| Google Cloud AutoML | Google | 企業級解決方案 |
| Amazon SageMaker Autopilot | Amazon | AWS 整合 |

## 結論

AutoML 降低了機器學習的門檻，讓更多人能夠使用 ML 技術。
