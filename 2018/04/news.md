# 本月新知

## 2018 年 4 月程式與 AI 技術動態

### 程式語言與框架

**Python 3.7 進入 Beta 階段**

Python 3.7 在本月進入 Beta 測試階段。這個版本帶來了多項重要改進：
- 資料類別（Data Classes）：大幅簡化類別定義
- 情境管理器（Context Variables）：支援异步程式設計
- 延遲求值屬性（Postponed Evaluation of Annotations）
- 效能提升：函式呼叫約快 20%

**TypeScript 2.8 發布**

Microsoft 發布 TypeScript 2.8，引入了條件型別（Conditional Types）和独占分發（Distributive）特性。這使得型別系統更加強大，特別適合處理泛型和映射型別。

**Go 1.10 發布**

Go 語言團隊發布 Go 1.10，這是自 Go 1.0 以來的第十個主要版本。新版改善了建置緩存、提升了效能，並增強了 HTTP/2 支援。

### AI 與機器學習

**TensorFlow 1.7 整合 Keras**

Google 在本月發布 TensorFlow 1.7，最大亮點是正式將 Keras 2 整合為 TensorFlow 的官方高階 API。這使得建立神經網路變得更加直覺，吸引了更多開發者使用 TensorFlow。

**PyTorch 0.4 預覽版**

PyTorch 團隊在本月展示了 0.4 版本的預覽功能，包括：
- 改進的 JIT 編譯器
- 更好的 Windows 支援
- 張量運算優化

**AutoML 持續發展**

Google Brain 團隊在 AutoML 領域取得新進展。自動化的神經網路架構搜尋（NASNet）在 ImageNet 分類任務上達到了新的最佳成績。

**強化學習應用於機器人控制**

加州大學柏克萊分校的研究團隊展示了使用強化學習控制機器人的成果，機器人成功學習了複雜的操控技能，如折疊衣物。

### 開發工具與雲端服務

**GitHub 開發者數量突破 8000 萬**

GitHub 宣布其平台已擁有超過 8000 萬註冊開發者，成為全球最大的程式碼託管平台。GitHub Enterprise 也持續獲得企業用戶青睞。

**AWS 發布 SageMaker 新功能**

Amazon Web Services 為其機器學習平台 SageMaker 增添了新功能，包括：
- 自動模型調整（Auto-tuning）
- 更好的 Jupyter 整合
- 部署優化工具

**Google Cloud Platform 更新**

Google 在本月更新了其雲端平台，新增了：
- Cloud TPU 支援（測試版）
- AI Hub 測試版
- 改進的 Kubernetes 支援

### 業界動態

- **蘋果收購 Shazam**：蘋果以 4 億美元收購音樂辨識服務 Shazam，將其技術整合進 iOS 生態系統
- **NVIDIA 發布 Titan V 農業版**：為資料中心和深度學習應用優化
- **IBM 發布量子處理器**：50 量子位元的量子處理器進入測試階段
- **DeepMind 與 Unity 合作**：使用遊戲引擎訓練 AI

### 開源專案

- **TensorFlow Object Detection API 發布**：簡化物體偵測模型的建立
- **fast.ai v1.0 開發中**：這個高階深度學習庫即將迎來重大更新
- **ONNX 格式獲得更多支援**：Facebook、Microsoft 和 AWS 共同推動的神經網路交換格式

### 標準與規範

- **W3C 發布 WebAuthn 草案**：無密碼認證的 Web 標準邁向成熟
- **ECMAScript 2018 定案**：非同步迭代器和 rest/spread 屬性正式納入標準

### 未來展望

機器學習工具的普及讓更多開發者能夠輕鬆建立 ML 應用。TensorFlow 整合 Keras、PyTorch 的快速發展，以及 AutoML 的興起，都預示著 AI 技術正在從研究走向大規模應用。
