# Google Cloud AI Platform 更新

## 前言

2018 年，Google Cloud 持續強化其 AI 和機器學習平臺，提供更多企業級功能。

## 主要更新

### Cloud ML Engine

- 支援超參數調整
- 更好的 GPU/TPU 支援
- 模型部署和版本管理

### AI Hub (Beta)

一個用於發現、分享和部署 AI 模型的平臺。

### Cloud AutoML

讓沒有 ML 經驗的開發者也能訓練自定義模型：
- AutoML Vision
- AutoML Natural Language
- AutoML Translation

## 特色功能

1. **預訓練模型**：可直接使用的視覺、語言處理模型
2. **BigQuery ML**：直接在 BigQuery 中訓練 ML 模型
3. **AI Platform Notebooks**：Jupyter 筆記本環境

## 結論

Google Cloud AI Platform 為企業提供了完整的 ML 工作流程。
