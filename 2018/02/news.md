# 2018 年 2 月號新聞摘要

## 版本控制領域

- GitHub 在 2018 年 2 月宣佈全球註冊用戶已超過 2500 萬，2017 年新增開源專案超過 800 萬個。GitHub Education 計畫持續擴展，提供免費工具給學校使用。

- Git 2.16 版本在 2 月發布，代號 "Takamura"，包含多項改進，包括更好的互動式 rebase 體驗和 GPG 簽名驗證增強。

- GitHub 推出新功能「Required Reviews」，強化團隊協作時的程式碼審查流程。

## Python 生態系

- Python 3.7 beta 1 在 2 月發布，包含資料類別（dataclass）、語法展開增強、速度提升等新特性。正式版預定於 6 月發布。

- PyTorch 0.4.0 在 2018 年 5 月前的重要版本，強化了 Tensor 整合與自動求導功能。Keras 作者 François Chollet 宣佈 Keras 將成為 TensorFlow 官方高階 API。

- Jupyter Notebook 5.3 發布，改善了輸出截斷和 Markdown 渲染。

## 雲端與基礎設施

- AWS 在 2 月推出新服務簡化容器部署流程。Amazon SageMaker 新增超參數優化功能，降低機器學習門檻。

- Google Cloud Platform 推出 Cloud Run，支援無伺服器容器運行。

## 開源與社群

- Microsoft 收購 GitHub 的消息在 2018 年 6 月確認（此前傳聞），2 月時 GitHub 仍為獨立公司。

- Mozilla 發布 Firefox 58，改進了頁面載入速度與開發者工具。

## 人工智慧

- Google Brain 團隊發布 AutoML 研究的最新進展，自動機器學習取得新突破。

- NVIDIA 發布 Titan V 顯示卡，為深度學習提供更強運算能力。
