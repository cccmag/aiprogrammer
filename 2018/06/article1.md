# Apple WWDC 2018 更新

## 前言

2018 年 6 月，Apple 舉行了 WWDC 開發者大會，發布了多項重要更新。

## 主要發布

### iOS 12

- Performance 改進
- ARKit 2：增強實境平臺
- Siri Shortcuts
- 螢幕使用時間

### Core ML 2

Core ML 是 Apple 的機器學習框架，Core ML 2 帶來了：
- 更快的模型推論
- 更好的模型壓縮
- Vision 和 NLP 更新

### Swift 4.2

Swift 語言持續改進，包括：
- 編譯器優化
- 新的語法功能

### ARKit 2

- 多人 AR 體驗
- 更好的物體偵測
- USDZ 格式支援

## 結論

Apple 在 AI/ML 方面的投入持續增加，Core ML 使開發者能夠輕鬆在 iOS 應用中加入 ML 功能。
