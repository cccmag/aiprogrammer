# 本月科技新聞 — 2018年10月

## Google 發表 BERT 模型

Google AI Language 在 10 月正式發表 BERT（Bidirectional Encoder Representations from Transformers），這是一款革命性的語言表示模型。BERT 在 11 項 NLP 任務上刷新了最佳紀錄，包括 SQuAD 問答測試、GLUE 基準測試等。BERT 的核心創新在於雙向 Transformer 架構，透過 Masked Language Model 和 Next Sentence Prediction 兩種預訓練任務，學習深層的語言表示。

## NVIDIA 推出 Turing 架構 GPU

NVIDIA 發表基於 Turing 架構的 RTX 系列顯示卡，採用全新的 RT Core 與 Tensor Core設計。RTX 2080 Ti 提供 11 TFLOPS 的 FP32 運算能力，並支援即時光線追蹤。對於深度學習訓練，Turing 的 Tensor Core 可加速矩陣運算，提升 Transformer 模型的訓練效率。

## PyTorch 1.0 正式版發布

Facebook AI Research 發布 PyTorch 1.0 正式版，整合了 TorchScript 與 eager execution，並提供完整的 Python API。1.0 版本改善了效能與穩定性，新增了 JIT 編譯器、量化工具與分散式訓練功能。許多研究機構已開始使用 PyTorch 進行 NLP 研究。

## TensorFlow 2.0 預覽版登場

Google 發布 TensorFlow 2.0 預覽版，簡化了 API 並默認啟用 eager execution。2.0 版本大幅改善了使用者體驗，移除了一些冗餘的 API，並整合了 Keras 作為中央高級 API。對於 BERT 等模型訓練，2.0 提供了更直覺的開發流程。

## GitHub 突破一億 repos

GitHub 宣布其平台已托管超過一億個 repositories，成為全球最大的程式碼托管平台。這個里程碑反映出開源軟體生態系的蓬勃發展，也顯示協作開發模式已成為主流。眾多的開源 AI 模型與工具促進了技術傳播。
