# 2018 年 7 月 — TensorFlow 與深度學習框架新聞

## TensorFlow 1.10 發布

Google 在 2018 年 7 月發布 TensorFlow 1.10 版，將 Keras 整合進核心 API，`tf.keras` 成為官方推薦的高階 API。此版本還包含 Eager Execution 的進一步優化與 TF Lite 改善。

參考：[TensorFlow 1.10 Release](https://www.google.com/search?q=TensorFlow+1.10+release+July+2018)

## Keras 被正式採納為 TensorFlow 官方 API

Keras 作者 François Chollet 在 2018 年宣佈 Keras 將作為 TensorFlow 的官方高階 API。社群反應熱烈，許多開發者認為這簡化了深度學習的入門門檻。

參考：[Keras + TensorFlow Integration](https://www.google.com/search?q=Keras+TensorFlow+integration+2018)

## PyTorch 持續成長挑戰 TensorFlow

2018 年中，PyTorch 在學術界的採用率已超越 TensorFlow。Facebook AI 團隊持續投入開發，PyTorch 1.0 預覽版已開放測試，動態圖模式深受研究人員喜愛。

參考：[PyTorch academic adoption 2018](https://www.google.com/search?q=PyTorch+academic+adoption+2018)

## AutoML 成為熱門研究方向

Google Cloud AutoML 在 2018 年初推出後，自動化機器學習模型設計成為產學研熱門話題。自動神經網路架構搜尋（NAS）論文大量湧現。

參考：[AutoML NAS research 2018](https://www.google.com/search?q=AutoML+neural+architecture+search+2018)

## 深度學習框架市佔率統計

2018 年中的調查顯示：TensorFlow 仍占企業市場主導地位，但 PyTorch 在研究論文中的使用率已超過 60%，成為學術界首選框架。

參考：[Deep Learning Framework Market Share 2018](https://www.google.com/search?q=deep+learning+framework+market+share+2018+TensorFlow+PyTorch)

## NVIDIA 發布 Tensor Core 優化庫

NVIDIA 在 2018 年強化了深度學習優化庫，Tensor Core GPU 的效能利用率進一步提升。TensorFlow 1.10 針對 Volta 架構進行優化。

參考：[NVIDIA Tensor Core optimization 2018](https://www.google.com/search?q=NVIDIA+Tensor+Core+deep+learning+2018)
