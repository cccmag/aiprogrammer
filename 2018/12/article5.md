# 開源 AI 專案回顧

## 深度學習框架

### TensorFlow
Google 維護的深度學習框架：
- 1.7 版本整合 Keras
- 2.0 預覽版發布
- 生態最完整

### PyTorch
Facebook 維護的框架：
- 1.0 正式版發布
- 研究界採用率上升
- 動態計算圖受歡迎

### 其他框架
- **Keras**：高層 API 標準
- **MXNet**：Apache 頂級專案
- **CNTK**：Microsoft 維護

## 模型庫

### Hugging Face Transformers
2018 年發布的模型庫：
- 支援 BERT、GPT、Transformer-XL
- 統一的模型 API
- PyTorch 與 TensorFlow 雙支援

### TensorFlow Model Garden
官方模型集合：
- 影視模型實現
- 研究模型實現

### Torchvision 與 tf.keras.applications
影視與影象預訓練模型：
- ResNet、VGG、Inception
- 影象分類、分割模型

## 研究工具

### AllenNLP
Allen Institute for AI 的研究平台：
- 為 NLP 研究設計
- 豐富的模型實現
- 完整的實驗支援

### fairseq
Facebook 的序列到序列工具：
- 機器翻譯研究
- 非自回歸翻譯

### Detectron2
Facebook 的物體偵測庫：
- Mask R-CNN 實現
- 豐富的預訓練模型

## 模型部署

### TensorFlow Serving
模型服務框架：
- 高效能推斷
- 模型版本管理
- REST API

### TensorFlow Lite
移動端部署工具：
- 模型壓縮
- 硬體加速
- iOS 與 Android 支援

## AutoML 工具

### Google Cloud AutoML
拖放式模型訓練：
- AutoML Vision
- AutoML NLP
- AutoML Translation

### Auto-Keras
Keras 的 AutoML 套件：
- 自動化架構搜尋
- 超參數優化

## 實驗管理

### TensorBoard
TensorFlow 視覺化工具：
- 損失與指標追蹤
- 計算圖視覺化
- 嵌入向量檢視

### Weights & Biases
實驗追蹤服務：
- 雲端儲存
- 協作功能
- 自動生成報告
