# 1. NLP：預訓練元年

## 2018 年 NLP 的三個轉折點

2018 年可以被稱為「NLP 預訓練元年」：

### ELMo（2 月）
Peters 等人發表「Deep Contextualized Word Representations」，提出 ELMo 模型：
- 使用雙向 LSTM 語言模型
- 動態的上下文詞向量
- 在 6 個 NLP 任務上刷新紀錄

ELMo 驗證了預訓練語言模型的有效性，但仍有局限性。

### GPT（6 月）
OpenAI 發表「Improving Language Understanding by Generative Pre-Training」：
- 使用單向 Transformer Decoder
- 在大型語料上預訓練
- 在多項任務上取得领先成绩

GPT 展示了大型預訓練模型的潛力，開啟了大模型時代的序幕。

### BERT（10 月）
Google 發表「BERT: Pre-training of Deep Bidirectional Transformers for Language Understanding」：
- 雙向 Transformer Encoder
- Masked Language Model + Next Sentence Prediction
- 在 11 項 NLP 任務上刷新紀錄

BERT 是 2018 年最重要的 NLP 突破，影響深遠。

## 預訓練革命的意義

### 遷移學習的勝利
NLP 終於找到了有效的遷移學習範式：
- 減少對標注資料的依賴
- 加快模型開發速度
- 提高小型團隊的研究能力

### 產業應用加速
預訓練模型降低了 NLP 應用的進入門檻：
- Google 將 BERT 應用於搜尋引擎
- 企業開始大規模部署 NLP 應用
- 新創公司湧現

### 生態系統形成
圍繞預訓練模型，形成了完整的工具鏈：
- Hugging Face Transformers
- 各雲平台的 NLP API
- 預訓練模型市場

## 2019 年展望

預訓練模型將繼續演進：
- 更大的模型
- 更高效的方法
- 更多應用場景
