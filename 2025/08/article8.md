# .gitignore 與檔案管理

## 為什麼需要 .gitignore

專案中有許多檔案不應該被 Git 追蹤：
- 編譯產物（.pyc、.o、.exe）
- 環境設定（.env、config.local）
- 套件依賴（node_modules/、vendor/）
- IDE 設定檔（.vscode/、.idea/）
- 作業系統檔案（.DS_Store、Thumbs.db）

## .gitignore 語法

```gitignore
# 註解以 # 開頭

# 忽略特定檔案
.env
config.local

# 忽略目錄
node_modules/
__pycache__/

# 使用萬用字元
*.pyc
*.log

# 否定模式（不忽略）
!important.log

# 忽略根目錄的特定檔案
/build
```

## Python 管理 .gitignore

```python
import os
import pathlib

class GitIgnoreManager:
    TEMPLATES = {
        "python": [
            "__pycache__/", "*.py[cod]", "*.so",
            ".env", "venv/", ".venv/",
            "*.egg-info/", "dist/", "build/"
        ],
        "node": [
            "node_modules/", "npm-debug.log",
            ".env", "dist/"
        ]
    }
    
    def create(self, project_type="python"):
        patterns = self.TEMPLATES.get(project_type, [])
        with open(".gitignore", "w") as f:
            f.write(f"# .gitignore for {project_type}\n")
            f.write(f"# Generated by GitIgnoreManager\n\n")
            for p in patterns:
                f.write(f"{p}\n")
        print(f"已建立 .gitignore ({project_type})")
    
    def add_pattern(self, pattern):
        with open(".gitignore", "a") as f:
            f.write(f"\n{pattern}")
        print(f"已新增規則：{pattern}")
    
    def is_ignored(self, filename):
        import fnmatch
        if not os.path.exists(".gitignore"):
            return False
        with open(".gitignore") as f:
            for line in f:
                line = line.strip()
                if line and not line.startswith("#"):
                    if fnmatch.fnmatch(filename, line):
                        return True
        return False

mgr = GitIgnoreManager()
mgr.create("python")
mgr.add_pattern("temp/")
print(f"__pycache__ 被忽略？{mgr.is_ignored('__pycache__/')}")
```
