# Git 版本控制與協作開發

## 為什麼學習 Git？

在現代軟體開發中，版本控制系統已經成為不可或缺的工具。Git 是目前最廣泛使用的分散式版本控制系統，由 Linus Torvalds 在 2005 年為管理 Linux Kernel 而開發。

### Git 的優勢

- **分散式架構**：每個開發者都有完整的儲存庫副本，離線也可工作
- **分支模型**：輕量級分支讓並行開發變得簡單
- **資料完整性**：使用 SHA-1 雜湊確保內容不被竄改
- **社群生態**：GitHub、GitLab、Bitbucket 等平台建構了龐大的協作網路

## Git 在業界的應用

根據 Stack Overflow 2025 年調查，超過 90% 的專業開發者使用 Git 進行版本控制。無論是 Web 開發、行動應用、資料科學還是嵌入式系統，Git 已成為跨領域的共通語言。
