# 本月科技新聞

## GitHub Copilot 支援更多語言

GitHub 宣布 Copilot 現在完整支援 Rust、Go 和 Java 的進階程式碼生成，開發者可以透過自然語言描述直接生成完整的函式實作。詳情請參考 https://www.google.com/search?q=GitHub+Copilot+2025+新功能。

## Git 2.47 發布

Git 2.47 版本引入了全新的「--remerge-diff」選項，讓開發者更容易檢視合併衝突的解決過程。此外，提升了大型儲存庫的效能表現。更多資訊請參閱 https://www.google.com/search?q=Git+2.47+release+notes。

## GitLab 推出 AI Code Review

GitLab 16.0 整合了 AI 驅動的程式碼審查功能，能夠自動檢測常見的程式碼問題和安全漏洞，並提供修復建議。詳見 https://www.google.com/search?q=GitLab+AI+Code+Review。

## 開源社群動態

Linux Kernel 社群繼續採用郵件列表搭配 Git 的開發流程，Linus Torvalds 在最近的訪談中強調了 Git 在大型專案中的重要性。參閱 https://www.google.com/search?q=Linus+Torvalds+Git+2025。

## 版本控制安全事件

多家企業近期遭遇供應鏈攻擊，攻擊者利用 Git 憑證外洩入侵內部儲存庫。業界呼籲採用 GPG 簽署提交和使用 SSH 金鑰認證。更多資訊請查閱 https://www.google.com/search?q=Git+供應鏈攻擊+2025。

## SourceHut 推出新服務

SourceHut 宣布支援 Git 的 patches 工作流程，提供更輕量的協作平台選擇，吸引許多追求簡潔的開源專案遷移。參閱 https://www.google.com/search?q=SourceHut+git+patches+2025。
