## 濃縮回顧

### 什麼是程式語言？

程式語言是人類與電腦溝通的中介。它將人類的邏輯思維轉換為電腦可以執行的指令。Python 以其簡潔優美的語法，成為初學者學習程式設計的最佳選擇。

### Python 的核心理念

```python
# Python 之禪：優美勝於醜陋，簡單勝於複雜
import this
```

### 從變數開始

變數是儲存資料的基本單元。Python 的動態型別讓變數使用更加靈活：

```python
x = 42          # 整數
x = "hello"     # 現在是字串
x = [1, 2, 3]   # 現在是列表
```

### 資料結構的選擇

| 資料結構 | 用途 | 範例 |
|---------|------|------|
| 列表 | 有序序列 | `[1, 2, 3]` |
| 字典 | 鍵值對 | `{"name": "Alice"}` |
| 集合 | 不重複元素 | `{1, 2, 3}` |
| 元組 | 不可變序列 | `(1, 2, 3)` |

### 函數：程式的積木

```python
def greet(name):
    """向使用者打招呼"""
    return f"你好，{name}！"
```

### 撰寫程式的流程

1. **需求分析**：確定程式要做什麼
2. **設計**：規劃程式的結構
3. **實作**：撰寫程式碼
4. **測試**：驗證程式正確性
5. **除錯**：修復發現的問題
6. **維護**：持續改善與擴充
