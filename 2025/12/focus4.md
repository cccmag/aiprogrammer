# 焦點 4：Web 技術演進

## HTMX 的復古革命

2025 年 HTMX 成為 Web 開發的熱門話題。它讓開發者用 HTML 屬性直接處理 AJAX、SSE 與 WebSocket，無需 JavaScript 框架。這種「回到 hypermedia」的理念在後端主導的團隊中特別受歡迎。

## WebAssembly 突破瀏覽器

WASM 在 2025 年跨出瀏覽器，WASI 0.2 標準確立後，Wasmtime、Wasmer 等 Runtime 開始在伺服器端、邊緣計算與 IoT 領域普及。Component Model 讓不同語言寫的 WASM 模組可以互相呼叫。

## Edge Runtime 成熟

Cloudflare Workers、Deno Deploy 與 Vercel Edge Functions 在 2025 年進入主流。邊緣運算不再是實驗性技術，許多生產應用完全遷移至 Edge。WinterCG 標準化進展順利，跨平台相容性大幅提升。

## React 的停滯與替代方案

React 在 2025 年沒有重大版本更新，社群開始轉向 SolidJS、Svelte 5（引入 runes）與新的前端框架 Qwik。React 的市場佔有率首次出現下滑，但生態系成熟度仍使其在大型企業中佔主導地位。

## CSS 的新時代

CSS 在 2025 年迎來 Container Queries、has() 選擇器與巢狀語法的全面支援。開發者不再需要 Sass/SCSS，原生 CSS 已足夠強大。Tailwind CSS 4.0 以新的引擎與效能優化繼續主導 utility-first 路線。
