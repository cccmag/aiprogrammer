# 2025 年十大技術新聞

## 1. DeepSeek 開源模型震撼 AI 界

DeepSeek R1 在數學推理能力上達到 GPT-4 等級，但訓練成本僅為其十分之一。這證明了開源 AI 的可行性，並引發了 AI 民主化的全球討論。[了解更多](https://www.google.com/search?q=DeepSeek+R1+open+source+AI+2025)

## 2. Rust 正式進入 Linux Kernel

經過多年的討論與準備，Rust 程式碼在 2025 年正式被 Linux Kernel 接納。這不僅是技術里程碑，更代表記憶體安全在系統程式設計中的重要性被正式確認。[了解更多](https://www.google.com/search?q=Rust+Linux+kernel+2025+memory+safety)

## 3. TypeScript 超越 JavaScript

TypeScript 的 GitHub 活躍倉庫數首度超越 JavaScript，結束了 JavaScript 長達 15 年的統治。大型企業與開源專案全面轉向 TypeScript。[了解更多](https://www.google.com/search?q=TypeScript+surpasses+JavaScript+GitHub+2025)

## 4. Mojo 正式開源

Modular 公司宣布 Mojo 編譯器開源，讓 AI 開發者可以在 Python 生態系中獲得 C 級效能。Mojo 社群在開源後半年內成長了 5 倍。[了解更多](https://www.google.com/search?q=Mojo+open+source+Modular+2025)

## 5. npm 供應鏈攻擊

2025 年 4 月發生的 npm 供應鏈攻擊影響了超過 10 萬個專案。多家科技巨頭聯手成立「開源安全聯盟」以應對此類威脅。[了解更多](https://www.google.com/search?q=npm+supply+chain+attack+2025+security)

## 6. AI 生成程式碼比例達 41%

GitHub Octoverse 報告指出，2025 年有 41% 的新程式碼由 AI 輔助產生。開發者的角色從生產者轉變為審查者與引導者。[了解更多](https://www.google.com/search?q=AI+generated+code+41+percent+GitHub+Octoverse+2025)

## 7. 歐盟 AI 法案正式生效

全球首部全面性 AI 監管法規於 2025 年 9 月生效。高風險 AI 系統需符合透明度、問責性與人為監督要求，影響全球 AI 開發標準。[了解更多](https://www.google.com/search?q=EU+AI+Act+2025+regulations)

## 8. OpenAI 轉向開源策略

2025 年底 OpenAI 出人意料地釋出 GPT-4 級別的開源模型。這一策略轉變被視為對 Llama 與 DeepSeek 等開源競爭對手的回應。[了解更多](https://www.google.com/search?q=OpenAI+open+source+GPT+level+model+2025)

## 9. HTMX 大爆發

HTMX 在 2025 年成為 GitHub 成長最快的 Web 框架之一。它挑戰了 SPA 為王的開發模式，讓後端開發者重新思考 Web 的 hypermedia 本質。[了解更多](https://www.google.com/search?q=HTMX+popular+2025+web+framework)

## 10. Zig 語言的崛起

Zig 憑藉 comptime 編譯時執行機制與無隱含記憶體分配的設計，在系統程式設計領域快速成長，TigerBeetle 等高效能資料庫選擇 Zig 作為主要開發語言。[了解更多](https://www.google.com/search?q=Zig+language+2025+systems+programming)
