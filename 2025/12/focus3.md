# 焦點 3：開源社群的重大變革

## Rust 進入 Linux Kernel

2025 年最重大的開源事件：Rust 程式碼正式成為 Linux Kernel 的一級公民。初始支援提供 GPU 驅動程式與 NVMe 驅動的 Rust 實作，記憶體安全議題在核心層級獲得解決。Linus Torvalds 在 2025 開源高峰會上說：「Rust 是 Kernel 的未來。」

## OpenTofu 站穩腳跟

Terraform 的開源分支 OpenTofu 在 2025 年達到穩定狀態，累積超過 3000 萬次下載。多家雲端廠商宣布支援 OpenTofu 作為 IaC 標準，徹底改變了基礎設施即代碼的生態。

## OpenAI 的開源轉向

2025 年初 OpenAI 釋出較小模型的開源版本，引發業界震盪。這被解讀為面對 Llama、Mistral 與 DeepSeek 等開源模型的壓力。2025 年下半年，多家 AI 公司跟進開源策略。

## npm 的信任危機

2025 年 4 月，npm 發生大規模供應鏈攻擊事件，數十個套件被植入後門。這導致社群全面檢討套件管理安全機制，Sigstore、SLSA 框架採用率大幅上升。

## Apache 基金會 25 週年

Apache Software Foundation 在 2025 年慶祝 25 週年，旗下超過 350 個專案持續活躍。Kafka、Spark、Flink 等數據專案仍是企業部署的核心。
