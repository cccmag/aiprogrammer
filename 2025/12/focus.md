# 2025 年程式人年度回顧 — 主題概述

## 為什麼是年度回顧？

2025 年是程式人產業劇變的一年。AI 從輔助工具變成核心開發環節，新語言 Mojo 與 Zig 崛起，TypeScript 正式超越 JavaScript 成為最受歡迎的 Web 語言，Rust 進軍 Linux Kernel。這一年發生的事太多，值得我們坐下來好好回顧。

## 本期架構

本期設計為七大焦點與十大專題的雙層結構：

### 七大焦點

| 焦點 | 主題 | 關鍵字 |
|------|------|--------|
| 1 | 語言生態 | Python 霸主、Mojo 爆發、TypeScript 超車 |
| 2 | AI 工具 | Copilot、Cursor、OpenCode、Claude Code |
| 3 | 開源變革 | Rust 進 Kernel、Linux 政治動盪、OpenAI 開源轉向 |
| 4 | Web 演進 | HTMX、WASM、Edge Runtime、React 停滯 |
| 5 | 雲端基建 | K8s 簡化、Serverless 成熟、FinOps |
| 6 | 職涯變化 | AI 取代 vs 輔助、遠端工作常態化 |
| 7 | 2026 預測 | Agent 化、AI Native 語言、去中心化 |

### 十大專題

從 Python 數據、GitHub 統計到技術教訓與未來展望，構成完整的 2025 拼圖。
