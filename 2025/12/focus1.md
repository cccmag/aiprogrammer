# 焦點 1：2025 程式語言生態回顧

## Python 穩坐霸主

2025 年 Python 在 TIOBE 指數達到 15.8%，持續領先。AI/ML 生態系是主要驅動力 —— PyTorch 2.x、JAX、Mojo 相容層都鞏固了 Python 的數據科學地位。然而，效能需求正催生新一波語言的誕生。

## Mojo 的爆發年

Mojo 在 2025 年成長 180%，雖然基數小，但成長曲線驚人。它結合 Python 語法與 MLIR 編譯器，讓 AI 開發者能在不犧牲效能的前提下使用熟悉語法。Mojo 已可用於生產環境的模型推論加速，多家新創公司將其納入技術棧。

## TypeScript 正式超越 JavaScript

2025 年中，TypeScript 的 GitHub 活躍倉庫數首度超越 JavaScript。大型專案幾乎全面採用 TypeScript，Deno 與 Bun 的原生 TypeScript 支援進一步加速了遷移。TC39 甚至開始討論將類型語法納入 ECMAScript 標準。

## Rust 的系統級勝利

Rust 在 2025 年進入 Linux Kernel 6.x 的正式支援模組，Android 也將 Rust 列為新開發的首選系統語言。成長率 22.1%，職缺年增 35%。Ferrocene 認證編譯器讓 Rust 進入航空與汽車領域。

## Zig 的異軍突起

Zig 以 45% 的成長率成為年度黑馬。它的編譯時編譯（comptime）機制深受系統程式設計師喜愛，TigerBeetle 等高效能資料庫專案全面採用 Zig。
