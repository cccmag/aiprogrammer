# 本期焦點：資料結構與演算法基礎 — Python 實戰

## 主題介紹

資料結構與演算法是程式設計的基石，無論你是剛開始學習程式的新手，還是已有幾年經驗的開發者，扎實的基礎都能幫助你寫出更高效、更優雅、更可靠的程式碼。

本期以 Python 為實作語言，從最基本的陣列、串列開始，逐步深入到堆疊、佇列、鏈結串列、樹等資料結構，再進入排序、搜尋、遞迴與動態規劃等經典演算法主題。每個主題都附有可直接執行的 Python 程式碼，讓你邊學邊練。
