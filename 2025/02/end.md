# 編後語

## 為什麼選這個主題？

資料結構與演算法是程式設計的內功。無論你寫的是網頁應用、手機 App、嵌入式系統，還是資料分析程式，良好的資料結構選擇和演算法設計都能讓你的程式更快、更穩定、更容易維護。

## 本期特點

本期的最大特色是「動手做」。我們不是只講理論，而是提供了完整的 Python 程式碼，從堆疊、佇列到快速排序、動態規劃，每行程式碼都可以實際執行。我們相信，學習程式最好的方式就是動手寫程式。

## 關於程式語言選擇

我們選擇 Python 作為本期實作語言，原因有三：
1. Python 語法簡潔，接近虛擬碼，適合表達演算法思想
2. Python 擁有豐富的資料結構支援（list、dict、deque 等）
3. Python 是目前台灣最受歡迎的入門程式語言之一

## 給讀者的建議

- **邊讀邊試**：開啟終端機，跟著文章輸入程式碼
- **修改程式碼**：試著調整參數，觀察執行結果的變化
- **練習題目**：到 LeetCode 或 ZeroJudge 尋找相關題目練習
- **閱讀原始碼**：查看 Python 標準函式庫的原始碼，學習大師的實作方式
