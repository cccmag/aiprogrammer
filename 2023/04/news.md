# 本月新知

## 2023 年 4 月程式與 AI 技術動態

### 程式語言與框架

**Rust 1.69 發布**

Rust 團隊於 4 月 20 日發布 Rust 1.69 穩定版。此版本主要聚焦於 Cargo 的改進，新增了 `cargo remove` 和 `cargo add` 的原生支援，並提升了編譯器性能。此外，Rust 1.69 引入了對 `#[doc(cfg)]` 的原生支援，使得條件編譯的文件更加清晰。

**Python 3.12 alpha 進入測試階段**

Python 3.12 的首個 alpha 版本於本月釋出，帶來了不少語法改進。最值得關注的是更清晰的錯誤訊息和 `typing` 模組的增強。Python 社群對類型提示的生態持續擴大感到振奮。

**TypeScript 5.1 Beta**

微軟於本月發布 TypeScript 5.1 Beta，引入了 `@link` 支援和更好的自動補全功能。型別系統的強化使得大型專案的維護更加可靠。

**WebAssembly GC 提案進入新階段**

W3C 的 WebAssembly GC 提案在本月進入候選推薦階段，這將允許在瀏覽器中直接運行垃圾回收語言的程式碼。Java、Kotlin、Dart 等語言的 WASM 支援將因此大幅改善。

### AI 與機器學習

**GPT-4 持續發燒**

OpenAI 的 GPT-4 自 3 月發布以來持續引發熱潮。4 月間，OpenAI 開放了 GPT-4 的 API 給更多開發者，並發布了 ChatGPT Plugin 生態系統，讓 LLM 可以存取外部工具和資料庫。

**LLaMA 模型洩漏引發開源 AI 浪潮**

Meta 的 LLaMA 模型在 3 月遭到洩漏後，4 月間社群基於 LLaMA 展開了大量研究。Stanford 的 Alpaca、Berkeley 的 Vicuna 等基於 LLaMA 微調的模型陸續出現，引發了開源 LLM 的熱潮。

**AutoGPT 與自主 AI Agent**

本月最熱門的項目莫過於 AutoGPT。這是一個基於 GPT-4 的自主 AI Agent，能夠自動分解任務、執行子任務，並根據結果調整策略。AutoGPT 的出現在 GitHub 上迅速獲得大量星標，引發了 AI Agent 開發的熱潮。

**Midjourney V5 發布**

Midjourney 於本月發布 V5 版本，在圖像品質和提示詞理解上有顯著提升。新版本支援更高解析度輸出，並允許更細緻的風格控制。

**Stable Diffusion 2.1 更新**

Stability AI 持續更新 Stable Diffusion 模型，2.1 版本改善了人體解剖結構的生成品質，並減少了色偏問題。

### 開發工具與雲端服務

**GitHub Copilot X 宣布**

GitHub 於本月宣布 Copilot X 計畫，將整合 GPT-4 模型，並新增聊天介面、PR 描述生成、以及終端機命令建議等功能。這標誌著 AI 輔助程式設計進入新階段。

**Google Colab 支援 LLaMA**

Google Colab 宣布支援 LLaMA 模型的運行，讓研究者和開發者可以在免費的 GPU 環境中實驗開源 LLM。

**VS Code 1.78 發布**

微軟發布 VS Code 1.78，新增了對遠端開發的改進，並強化了下拉終端機的功能。

### 業界動態

- **GitHub 使用者突破 1 億**：GitHub 宣布全球開發者數量突破 1 億大關

- **Adobe 推出 Firefly 商用 AI**：Adobe 發布了可商用授權的 AI 生成工具

### 標準與規範

- **HTTP/3 採用率突破 30%**：全球流量中 HTTP/3 的佔比持續成長
- **WebGPU 發布候選版**：為 Web 上的高效 GPU 運算做好準備
