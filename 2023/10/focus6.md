# 核方法與再生核 Hilbert 空間

## 從線性到非線性

許多機器學習演算法（如感知機、SVM、線性回歸）本質上是線性的。但現實世界的問題很少是線性可分的。

**核方法**（Kernel Methods）提供了一個優雅的解決方案：**透過特徵映射將非線性問題轉化為線性問題**。

## 特徵映射

設 φ: X → H 是從輸入空間到特徵空間的映射。在特徵空間中學習線性模型等價於在原始空間中學習非線性模型。

**範例**：二次多項式映射

```
φ(x₁, x₂) = (x₁², x₂², √2 x₁x₂, x₁, x₂, 1)
```

原始空間中需要 6 個參數（超平面在 6 維空間中）。

## 核技巧

核技巧（Kernel Trick）的核心觀察是：許多學習演算法只需要計算**內積**⟨φ(x), φ(x')⟩，不需要 φ 的顯式表示。

**核函數**定義為：

```
k(x, x') = ⟨φ(x), φ(x')⟩
```

### 常見核函數

| 核 | 公式 | 超參數 |
|----|------|--------|
| 線性 | k(x,x') = ⟨x,x'⟩ | — |
| 多項式 | k(x,x') = (⟨x,x'⟩ + c)^d | d, c |
| RBF（高斯） | k(x,x') = exp(-γ||x-x'||²) | γ |
| Sigmoid | k(x,x') = tanh(α⟨x,x'⟩ + c) | α, c |

### 核矩陣

給定 n 個樣本，核矩陣 K ∈ ℝ^{n×n} 的元素為：

```
K_{ij} = k(x_i, x_j)
```

核矩陣必須是半正定（Positive Semi-definite）的，這保證了存在對應的 φ。

## 再生核 Hilbert 空間（RKHS）

RKHS 是核方法的理論基礎。RKHS H 是一個滿足以下條件的 Hilbert 空間：

1. **再生性質**：對於所有 x ∈ X 和 f ∈ H，有 f(x) = ⟨f, k(·, x)⟩
2. **核函數 k** 滿足 k(·, x) ∈ H
3. **內積** ⟨k(·, x), k(·, x')⟩ = k(x, x')

### RKHS 的意義

RKHS 提供了一個優雅的框架：

- 核函數隱式定義了一個特徵空間（RKHS）
- 在這個空間中，學習問題變成線性的
- 代表定理（Representer Theorem）保證最優解有有限表示

## 表示定理

表示定理（Representer Theorem）是核方法的基石：

**定理**：正則化經驗風險最小化問題的解可以表示為：

```
f*(x) = Σᵢ α_i k(x, x_i)
```

其中 α_i 是係數，x_i 是訓練樣本。

這意味著：
- 解存在於由訓練樣本張成的有限維子空間中
- 即使 RKHS 是無窮維的，最優解只需要 n 個參數
- 非參數問題變成了參數問題

## 核 SVM

在 SVM 中應用核技巧：

原始 SVM（對偶形式）：

```
max Σᵢ α_i - ½ ΣᵢΣⱼ α_i α_j y_i y_j ⟨x_i, x_j⟩
```

核化 SVM：

```
max Σᵢ α_i - ½ ΣᵢΣⱼ α_i α_j y_i y_j k(x_i, x_j)
```

分類函數：

```
f(x) = sign(Σᵢ α_i y_i k(x, x_i) + b)
```

## 核方法的優缺點

| 優點 | 缺點 |
|------|------|
| 優雅的非線性擴展 | 核矩陣計算 O(n²) |
| 存在理論保證 | 無法擴展到超大型資料 |
| 透過正則化控制複雜度 | 需要選擇合適的核 |
| 可直接操作結構化資料 | 對核參數敏感 |

## 核方法與深度學習

雖然深度學習在許多領域取代了核方法，但核方法仍然在以下方面具有優勢：

- **小樣本學習**：核方法的理論邊界更緊
- **可解釋性**：決策函數有明確的支援向量
- **理論保證**：泛化邊界和收斂性明確

## 小結

| 概念 | 定義 | 意義 |
|------|------|------|
| 特徵映射 φ | X → H | 非線性→線性 |
| 核函數 k | k(x,x') = ⟨φ(x), φ(x')⟩ | 隱式計算內積 |
| RKHS | 具有再生性質的函數空間 | 核方法的理論基礎 |
| 表示定理 | f(x) = Σα_i k(x, x_i) | 有限參數表示 |
