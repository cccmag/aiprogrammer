# 假說空間與歸納偏置

## 學習沒有「純粹的歸納」

機器學習的一個常見迷思是「讓資料自己說話」——讓演算法從資料中自動發現模式。但學習理論告訴我們：**沒有偏置就沒有學習**。

任何學習演算法都必須對目標函數做出假設，這些假設稱為**歸納偏置**（Inductive Bias）。

## 假說空間的數學定義

假說空間 H 是所有可能模型的集合。對於分類問題：

```
H = {h: X → Y  |  h 滿足某種形式}
```

常見的假說空間：

| 模型 | 假說空間 | VC 維度 |
|------|---------|---------|
| 線性分類器 | {sign(w·x + b)} | d+1 |
| 決策樹 | {軸對齊分割的組合} | O(d·depth) |
| 3 層神經網路 | {σ(W₂·σ(W₁x + b₁) + b₂)} | O(W log W) |

## 歸納偏置的類型

### 語言偏置（Language Bias）

語言偏置是**假說空間本身的限制**。例如：
- 線性模型假設決策邊界是線性的
- KNN 假設相似的輸入有相似的輸出
- 決策樹假設決策邊界是軸對齊的

### 偏好偏置（Preference Bias）

偏好偏置是**在假說空間中的搜索偏好**。例如：
- L1 正則化偏好稀疏解
- 決策樹偏好較淺的樹
- SVM 偏好最大間隔的超平面

這在數學上對應於正則化項 Ω(h) 的選擇。

### 最佳化偏置（Optimization Bias）

最佳化偏置來自於**學習演算法的搜尋過程**。例如：
- SGD 偏好低頻譜範數的解
- 梯度下降隱式偏好「簡單」的解
- 隨機初始化導致不同的收斂解

## 沒有免費的午餐

No Free Lunch（NFL）定理告訴我們：**沒有任何歸納偏置在所有問題上都優於其他偏置**。

```
平均而言，所有學習演算法在所有問題上的期待表現相同。
```

這意味著：
1. 選擇假說空間就是選擇偏置
2. 沒有「通用的最佳學習演算法」
3. 領域知識是選擇偏置的關鍵

## 歸納偏置在深度學習中

深度學習展現了與傳統學習理論不同的模式：

**傳統觀點**：對假說空間施加強偏置（如線性、核函數）
**深度學習**：使用巨大假說空間，依賴隱式偏置（SGD、架構）

這解釋了為什麼深度學習能夠成功——巨大的假說空間加上適當的隱式偏置：

```
深度網路 ≈ 龐大的 H + SGD（隱式正則化）
```

### 深度學習中的隱式偏置

1. **架構偏置**：CNN 的局部連接、RNN 的循環結構、Transformer 的自注意力
2. **演算法偏置**：SGD 傾向於低範數解、batch normalization 的穩定化效果
3. **初始化偏置**：Xavier/He 初始化控制了初始函數的頻譜

## 如何選擇假說空間

選擇假說空間是機器學習實踐中最關鍵的決策：

| 考量 | 問題 | 建議 |
|------|------|------|
| 樣本數 | 很少 | 小 H（線性、正則化強） |
| 樣本數 | 很多 | 大 H（深度網路、隨機森林） |
| 先驗知識 | 很強 | 客製化 H（領域特徵） |
| 可解釋性 | 需要 | 小 H（決策樹、線性） |
| 精度 | 優先 | 大 H（神經網路） |

## 小結

假說空間和歸納偏置是學習理論中經常被忽略但極為重要的概念：

- 所有學習演算法都有偏置——選擇演算法就是選擇偏置
- NFL 定理意味著沒有通用演算法——問題結構決定了偏置的選擇
- 理解偏置有助於選擇正確的演算法和調整超參數
