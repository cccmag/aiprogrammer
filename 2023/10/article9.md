# 高斯過程

## 從參數到函數

之前的文章討論的模型都是**參數化**的——用一組參數 θ 描述模型。高斯過程（Gaussian Process，GP）是**非參數**的——直接在函數空間上進行推斷。

## 什麼是高斯過程？

高斯過程是一個隨機過程，其中任意有限個點的聯合分布都是高斯分布。

```
f ∼ GP(m(x), k(x, x'))
```

其中：
- **均值函數** m(x) = E[f(x)]
- **協方差函數**（核函數）k(x, x') = Cov[f(x), f(x')]

### 高斯過程的直觀理解

- GP 定義了函數的分布（類似於高斯分布定義了向量的分布）
- 從 GP 中「取樣」得到的是完整的函數（不僅是數值）
- 協方差函數決定了函數的平滑性和結構

## GP 回歸

給定訓練資料 {(x_i, y_i)}，其中 y_i = f(x_i) + ε_i，ε_i ∼ N(0, σ²)。

### 聯合分布

訓練點和測試點的聯合分布：

```
[y]   ~ N( [m(X)] , [K(X,X) + σ²I   K(X, X*)]  )
[f*]      [m(X*)]    [K(X*, X)       K(X*, X*)]
```

### 預測分布

給定訓練資料下的條件分布（後驗預測分布）：

```
f* | X, y, X* ∼ N(μ*, Σ*)
```

其中：

```
μ* = m(X*) + K(X*, X)[K(X, X) + σ²I]^{-1}(y - m(X))
Σ* = K(X*, X*) - K(X*, X)[K(X, X) + σ²I]^{-1}K(X, X*)
```

**關鍵性質**：GP 回歸不僅給出預測值 μ*，還給出預測的不確定性 Σ*。

## 核函數的作用

在 GP 中，核函數（協方差函數）編碼了我們對函數的**先驗假設**：

| 核函數 | 產生的函數特性 | 超參數 |
|--------|---------------|--------|
| RBF | 無限平滑（C^∞）| length-scale ℓ |
| Matérn | (ν-1) 次可微 | ν, ℓ |
| 週期性 | 週期函數 | 週期 p, ℓ |
| 線性 | 線性函數 | 斜率的方差 |

### RBF 核的超參數

RBF 核 k(x, x') = σ²_f exp(-||x-x'||²/(2ℓ²))：

- **ℓ（length-scale）**：控制函數的平滑度。ℓ 越小函數變化越快
- **σ²_f（信號方差）**：控制函數的幅度

## 超參數學習

GP 的超參數 θ（如 ℓ、σ²_f、σ²）可以透過最大化邊際似然來學習：

```
log p(y | X, θ) = -½ y^T K_y^{-1} y - ½ log|K_y| - n/2 log(2π)
```

其中 K_y = K(X,X) + σ²I。

邊際似然的三項解釋：
1. **擬合項**：y^T K_y^{-1} y —— 模型對資料的擬合程度
2. **複雜度懲罰**：log|K_y| —— 模型的複雜度
3. **歸一化常數**：常數項

最大化邊際似然自動實現了**偏差-變異權衡**。

## GP 的計算複雜度

GP 的主要限制是計算複雜度：

| 操作 | 複雜度 | 瓶頸 |
|------|--------|------|
| 訓練 | O(n³) | 求逆 K_y^{-1} |
| 預測均值 | O(n) | 每個測試點 |
| 預測方差 | O(n²) | 每個測試點 |

### 大規模 GP 的近似方法

| 方法 | 複雜度 | 策略 |
|------|--------|------|
| 稀疏 GP | O(n·m²) | m 個誘導點 |
| 隨機傅立葉特徵 | O(n·D) | D 個隨機特徵 |
| KISS-GP | O(n) | 結構化核 |
| 深度 GP | O(n·L·m²) | 分層 GP |

## GP 的應用

### 貝氏最佳化

GP 是貝氏最佳化（Bayesian Optimization）的核心工具。GP 作為目標函數的代理模型，透過採集函數（Acquisition Function）選擇下一個取樣點。

### 自動超參數調優

許多 AutoML 框架（如 Hyperopt、Optuna）使用 GP 來建模超參數與模型表現之間的關係。

### 時間序列預測

GP 可以自然地處理不規則採樣的時間序列，並提供預測的不確定性區間。

## GP 與深度學習的比較

| 方面 | 高斯過程 | 深度學習 |
|------|---------|---------|
| 參數數量 | O(n)（非參數） | O(W)（參數） |
| 不確定性 | 自然給出 | 需特別處理 |
| 小樣本表現 | 優秀 | 依賴預訓練 |
| 大規模資料 | 困難 | 擅長 |
| 可解釋性 | 核函數決定 | 黑箱 |

## 小結

高斯過程是機器學習中最優雅的貝氏非參數方法：

- **函數空間視角**：直接在函數上進行推斷
- **不確定性量化**：自然給出預測信心區間
- **核函數作為先驗**：編碼對函數結構的假設
- **邊際似然學習**：自動的模型選擇機制
