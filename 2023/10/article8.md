# 核技巧與 SVM

## SVM：從最大間隔到對偶

支援向量機（SVM）是核方法最著名的應用。我們從最大間隔分類器開始。

### 原始問題

```
min ½||w||²    subject to y_i(w·x_i + b) ≥ 1  for all i
```

**幾何意義**：最大化分類間隔 margin = 2/||w||。

### 對偶問題

透過拉格朗日對偶性，得到對偶問題：

```
max Σα_i - ½ ΣΣ α_i α_j y_i y_j ⟨x_i, x_j⟩
subject to Σα_i y_i = 0, α_i ≥ 0
```

對偶形式的關鍵特徵：**目標函數只依賴於樣本間的內積** ⟨x_i, x_j⟩。

## 核技巧的引入

將內積替換為核函數：

```
⟨x_i, x_j⟩ → k(x_i, x_j)
```

對偶問題變為：

```
max Σα_i - ½ ΣΣ α_i α_j y_i y_j k(x_i, x_j)
```

決策函數：

```
f(x) = sign(Σ α_i y_i k(x, x_i) + b)
```

這就是**核 SVM**——完全在原始空間中計算，但等效於在高維特徵空間中學習線性分類器。

## 核函數的選擇

### 線性核

k(x, x') = ⟨x, x'⟩

- 無超參數
- 等於原始 SVM
- 適合高維稀疏資料（如文字分類）

### 多項式核

k(x, x') = (⟨x, x'⟩ + c)^d

- 超參數：c（偏移），d（次數）
- d = 1 退化為線性核
- d 越大，決策邊界越複雜

### RBF（高斯）核

k(x, x') = exp(-γ||x-x'||²)

最常用的核函數。γ 控制徑向基函數的寬度：

- γ 太小：接近線性核（高偏差）
- γ 太大：每個點都是支援向量（高變異）

**選擇建議**：通常先用 RBF 核，然後調參 γ 和 C。

### 字串核、圖核

針對結構化資料設計的專用核函數：

- **字串核**：基於共同子序列的數量
- **圖核**：基於隨機遊走或 WL 子樹模式
- **樹核**：基於共同子樹的數量

這些核讓 SVM 可以直接應用於序列、圖形和樹狀資料。

## Mercer 條件

一個函數 k(x, x') 是合法核函數的充分必要條件（Mercer 定理）：

對任意有限點集 {x_i}，核矩陣 K_{ij} = k(x_i, x_j) 是**半正定**的。

```
對任意向量 c: c^T K c ≥ 0
```

這保證了存在對應的特徵映射 φ 和 RKHS。

## SVM 的支援向量

SVM 的關鍵特性是**稀疏性**——只有支援向量（α_i > 0 的樣本）影響決策邊界。

```
f(x) = sign(Σ_{i∈SV} α_i y_i k(x, x_i) + b)
```

支援向量的數量決定了 SVM 的**推理複雜度**——與訓練集大小無關，只與支援向量數有關。

### 支援向量的意義

- 支援向量是距離決策邊界最近的樣本
- 移除非支援向量不改變分類器
- 支援向量越多，決策邊界越複雜

## 軟間隔 SVM

現實問題通常不是線性可分的。軟間隔 SVM 引入鬆弛變數 ξ_i：

```
min ½||w||² + C Σ ξ_i
subject to y_i(w·x_i + b) ≥ 1 - ξ_i, ξ_i ≥ 0
```

C 控制間隔最大化和誤差最小化之間的權衡：

- C 很大：強調分類正確（可能過擬合）
- C 很小：強調大間隔（可能欠擬合）

## SVR：支援向量回歸

SVM 可以推廣到回歸問題（SVR）。SVR 引入 ε-insensitive 損失函數：

```
L(y, f(x)) = max(0, |y - f(x)| - ε)
```

SVR 只對預測偏差超過 ε 的樣本進行懲罰，產生稀疏的支援向量表示。

## SVM 的優缺點

| 優點 | 缺點 |
|------|------|
| 理論完備（VC 維度保證） | 核矩陣 O(n²) 記憶體 |
| 對偶問題凸最佳化 | 對核參數敏感 |
| 稀疏解（支援向量） | 非機率輸出 |
| 核技巧擴展性強 | 大規模資料困難 |

## SVM 與深度學習

在深度學習時代，SVM 仍然在以下場景中不可替代：

- **小樣本學習**：SVM 在小樣本下泛化更強
- **高維稀疏資料**：如基因表達數據、文字分類
- **需要理論保證**：醫療診斷、安全關鍵系統
- **可解釋性要求高**：支援向量直觀展示了決策依據
