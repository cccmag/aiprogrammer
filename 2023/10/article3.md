# VC 維度計算範例

## 為什麼需要實際計算 VC 維度

VC 維度是衡量假說空間複雜度的關鍵指標，但直接計算並不總是直觀。本文透過多個實際範例展示如何計算不同假說空間的 VC 維度。

## 範例一：實數區間

假說空間 H = {I_{a,b}(x) = 1{a ≤ x ≤ b} | a, b ∈ ℝ, a < b}

**VC 維度：2**

**證明**：
- **下界（打散 2 個點）**：設 x₁ < x₂。選擇 a ∈ (x₁, x₂) 且 b ∈ (x₂, ∞) 可實現 (+,-)。選擇 a ∈ (-∞, x₁) 且 b ∈ (x₁, x₂) 可實現 (-,+)。透過調整區間可實現所有四種模式。
- **上界（不能打散 3 個點）**：設 x₁ < x₂ < x₃。模式 (+, -, +) 表示 x₁ 和 x₃ 在區間內但 x₂ 不在。但區間是連續的，不可能跳過中間點。因此 VC(H) = 2。

## 範例二：軸對齊矩形

H = {R_{a,b,c,d}(x,y) = 1{a ≤ x ≤ b, c ≤ y ≤ d}}

**VC 維度：4**

**證明思路**：
- 下界：4 個點成菱形排列（一個在中心，三個在外圍），可以實現所有 16 種標記
- 上界：5 個點無法全部打散。考慮 5 個點的凸包，如果它們構成一個凸五邊形，則無法實現「外部全負、內部全正」的模式

## 範例三：d 維感知機

H = {sign(w·x + b) | w ∈ ℝ^d, b ∈ ℝ}

**VC 維度：d + 1**

**證明**：

**下界**：考慮 d+1 個點 x₀ = 0, x₁ = e₁, ..., x_d = e_d（e_i 是標準基向量）。
對於任意標記 y₀, ..., y_d ∈ {+1, -1}，需要找到 w, b 使得：
- sign(w·0 + b) = sign(b) = y₀ → b = y₀
- sign(w·e_i + b) = sign(w_i + b) = y_i

令 w_i = y_i - b = y_i - y₀。由於 w_i 等於 0 或 ±2，sign(w_i + y₀) = sign(y_i) = y_i，因此所有標記都可實現。

**上界**：使用 Radon 定理。任意 d+2 個點在 ℝ^d 中一定存在非平凡仿射相關關係，這限制了可實現的標記模式。

## 範例四：決策樹樁（Decision Stump）

H = {sign(x_i - t) | i ∈ {1,...,d}, t ∈ ℝ}

**VC 維度：d**

**證明**：
- 每個決策樹樁只看一個特徵，等價於在該特徵上的二分法
- d 個特徵意味著最多能打散 d 個點（每個點在不同特徵上分離）
- 當點數超過 d 時，一定存在兩個點在所有特徵上的順序關係相同，無法區分

## 範例五：正弦分類器

H = {sin(ωx) | ω ∈ ℝ}

**VC 維度：∞**

**證明**：給定任意 n 個點 x₁ < ... < x_n，選擇足夠大的 ω 使得 sin(ωx) 在相鄰點之間振盪足夠多次，可以實現任意標記模式。因此 VC 維度無窮大。

這解釋了為什麼正弦函數族不是 PAC 可學習的。

## VC 維度總結

| 假說空間 | VC(H) | 關鍵洞見 |
|---------|-------|---------|
| 實數區間 | 2 | 連續性限制 |
| 軸對齊矩形 | 4 | 凸包限制 |
| 線性分類器 | d+1 | Radon 定理 |
| 決策樹樁 | d | 特徵獨立性 |
| 正弦 | ∞ | 無限振盪能力 |

## 計算 VC 維度的通用策略

1. **下界**：找到一組可打散的點集，構造實現所有標記的參數
2. **上界**：證明任意更大點集存在無法實現的標記模式
3. 通常下界容易找到，上界需要更深入的分析

## VC 維度的實用意義

- **指導模型選擇**：VC 維度較低的模型需要較少樣本
- **解釋正則化**：正則化降低了有效 VC 維度
- **理解過擬合**：當 n < VC(H) 時，理論上無法保證泛化
