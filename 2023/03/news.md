# 2023年3月科技新聞

## GPT-4 正式發表

2023年3月14日，OpenAI 發表了 GPT-4 多模態大型語言模型。GPT-4 支援文字與圖片輸入，在律師考試、SAT 等多項專業測驗中表現超越人類平均水準。此外，GPT-4 也改善了幻覺問題與安全性，成為當時最先進的對話式 AI 模型。https://www.google.com/search?q=GPT-4+2023+3+14

## 美國總統 Biden 發表晶片法案後續

拜登政府持續推動半導體本土製造政策，2023年3月宣布 Intel 將在俄亥俄州興建大型晶圓廠，投資金額超過 200 億美元。此舉旨在減少美國對亞洲半導體供應鏈的依賴，並強化本土晶片製造能力。https://www.google.com/search?q=Biden+CHIPS+Act+2023+March

## Rust 1.68 發布

Rust 程式語言於 2023 年 3 月 9 日發布 1.68 版。主要亮點包括 Cargo 稀疏協定（sparse protocol）正式穩定化，大幅提升了套件依賴的解析速度。此外也推出了 `--help` 輸出的改善與 `std::process::ExitCode` 等新 API。https://www.google.com/search?q=Rust+1.68+2023

## Python 3.12 首個 Alpha 版

Python 3.12 的首個 alpha 版本在 2023 年 3 月釋出。新版本包含更佳的錯誤訊息顯示、`typing` 模組的改進、以及 f-string 解析器的重寫。Python 3.12 預計在 2023 年 10 月正式發布。https://www.google.com/search?q=Python+3.12+alpha+2023

## Alpine.js 3.12 推出

輕量級前端框架 Alpine.js 發布 3.12 版，新增 Teleport 功能，讓開發者可以更方便地將元素渲染到 DOM 的其他位置。此外也強化了表單驗證與焦點管理機制。https://www.google.com/search?q=Alpine.js+3.12+2023

## Git 2.40 發布

Git 2.40 於 2023 年 3 月正式釋出，改善了合併衝突的顯示方式，讓開發者更容易理解衝突的來源。同時也對 diff 演算法進行了最佳化，並引入了新的 `git branch` 排序選項。https://www.google.com/search?q=Git+2.40+2023

## Google 推出 Bard 聊天機器人

2023年3月，Google 正式推出 Bard 聊天機器人（後更名為 Gemini），作為 ChatGPT 的競爭對手。初期僅在美國與英國開放測試，使用 LaMDA 語言模型作為基礎。https://www.google.com/search?q=Google+Bard+2023+March
