# AI 程式人雜誌 — 2023年3月號

## 本期主題：資料結構理論

資料結構是電腦科學中最核心的基礎知識之一。無論是演算法設計、系統開發或面試準備，扎實的資料結構背景都能讓你事半功倍。本期雜誌從抽象資料型別（ADT）出發，深入探討樹、平衡樹、堆積、圖論、圖走訪與雜湊等經典主題，並搭配完整的 Python 程式碼實作，讓理論與實務緊密結合。
