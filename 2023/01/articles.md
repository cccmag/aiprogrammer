# 文章集錦

## 計算機組織與架構專輯

### 程式相關（10 篇）

#### 1. [二進位與布林代數](article1.md)

二進位系統的數學基礎、布林代數的公理與定理、真值表與邏輯表達式化簡。包含 Python 實作的布林代數運算與卡諾圖（Karnaugh Map）化簡範例。

#### 2. [邏輯閘與組合電路](article2.md)

基本邏輯閘（AND、OR、NOT、NAND、NOR、XOR、XNOR）的結構與功能、組合電路的設計方法、多工器與解碼器的實作。包含邏輯閘層級的電路模擬程式碼。

#### 3. [加法器與 ALU 實作](article3.md)

半加器、全加器、漣波進位加法器、超前進位加法器的設計與比較。從加法器逐步建構 ALU 的完整過程，包含 4 位元 ALU 的 Python 模擬實作。

#### 4. [正反器與循序電路](article4.md)

SR 正反器、D 型正反器、JK 正反器、T 型正反器的結構與特性。暫存器、計數器、有限狀態機的設計。包含時序圖分析和 Python 模擬。

#### 5. [管線化冒險與解決](article5.md)

資料冒險、控制冒險、結構冒險的成因與分類。轉發（Forwarding）、管線停頓（Stall）、分支預測的解決方案。包含管線冒險的動畫式模擬程式碼。

#### 6. [快取記憶體策略](article6.md)

快取映射方式（直接映射、全關聯、組關聯）、替換策略（LRU、FIFO、隨機）、寫入策略（Write-through、Write-back）。包含不同快取配置的效能比較模擬。

#### 7. [虛擬記憶體](article7.md)

虛擬位址與實體位址的映射、分頁與分段機制、頁表結構、TLB 加速、頁面置換演算法（FIFO、LRU、時鐘演算法）。包含虛擬記憶體模擬器。

#### 8. [RISC vs CISC](article8.md)

RISC 與 CISC 的設計哲學對比、指令編碼格式、定址模式分析、歷史案例（MIPS、ARM、x86）。包含指令編碼的 Python 解析器。

#### 9. [GPU 架構導論](article9.md)

GPU 的平行計算模型、SIMT 執行模型、CUDA 核心架構、記憶體層次、GPU 程式設計模型。包含簡單的 GPU 模擬程式碼。

#### 10. [效能評估：CPI 與 Amdahl 定律](article10.md)

CPU 效能公式（Time = 指令數 × CPI × 時脈週期）、CPI 計算方法、Amdahl 定律的應用、SPEC 基準測試。包含 CPI 計算器和加速比模擬。
