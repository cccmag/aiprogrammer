# 本月新知

## 2023 年 9 月程式與 AI 技術動態

### AI 領域重大新聞

**OpenAI 推出 GPT-4V：多模態視覺能力**

2023 年 9 月，OpenAI 宣布 GPT-4 新增視覺理解能力（GPT-4V），可分析圖像內容並回答相關問題。這標誌著大型語言模型從純文字邁向多模態的重要里程碑。API 同時開放圖像輸入，開發者可建立圖像辨識、文件分析等應用。

**OpenAI 發表 DALL-E 3**

DALL-E 3 於 9 月發布，與 ChatGPT 深度整合，使用者可用自然語言對話方式生成和編輯圖像。DALL-E 3 在文字渲染和細節準確度上有顯著提升，並引入了內容安全過濾機制。

**Meta 釋出 Llama 2 商業可用版本**

Meta 在 9 月更新了 Llama 2 的授權條款，正式允許商業使用。Llama 2 提供 7B、13B 和 70B 三種規模，在多項基準測試中與 GPT-3.5 競爭。社群迅速建立了基於 Llama 2 的微調模型生態，包括 Code Llama（程式碼專用版）。

### 程式語言生態

**Python 3.12 候選版發布**

Python 3.12 RC 於 9 月推出，引入更高效的 f-string 解析、錯誤訊息改進（PEP 701）、TypedDict 的關鍵字支援（PEP 692），以及 Linux perf 分析器的支援。CPython 的內部架構持續最佳化，直譯器效能提升約 10-20%。

**Rust 1.72 穩定版發布**

Rust 1.72 於 9 月釋出，穩定了 `feature(const_trait_impl)` 的部份功能，使 const 泛型更加可用。新版本也改進了 borrow checker 的診斷訊息，並引入了 `#[doc(cfg)]` 功能來文件化平台特定 API。

**TypeScript 5.3 進入開發階段**

TypeScript 5.3 在 9 月進入 beta，引入 import attributes、`using` 宣告（TC39 stage 3）的自動資源管理，以及 narrowed type 的改善。TS 團隊持續聚焦於型別系統的精確度和編譯效能。

### 開發工具與平台

**GitHub Copilot 更新：支援多模型選擇**

GitHub 宣布 Copilot 將支援多種 AI 模型，不再僅限於 OpenAI。開發者可在 Copilot Chat 中選擇不同後端模型，包括 GitHub 自家的 Codex 模型和社群貢獻模型。

**JetBrains AI Assistant 正式版推出**

JetBrains 發布 AI Assistant 正式版，整合至 IntelliJ IDEA、PyCharm 等 IDE。支援程式碼生成、解釋、重構和測試撰寫，並可自訂提示模板。

### 業界動態

- **AWS 推出 Amazon Bedrock 正式版**：全託管的基礎模型服務正式可用
- **Google 更新 Bard**：整合 Google 應用生態，支援 Gmail、Docs 等服務
- **Adobe 發布 Firefly 2.0**：商用級 AI 圖像生成，引入向量圖形功能
- **Elon Musk 成立 xAI**：新創 AI 公司，目標「理解宇宙的真實本質」
- **Linux 基金會成立 Unified AI Accelerator 介面標準**：旨在統一不同 AI 晶片的驅動介面
- **Stack Overflow 2023 調查**：Rust 連續第八年獲得「最受歡迎語言」第一名，TypeScript 超越 Java 成為第三大語言
