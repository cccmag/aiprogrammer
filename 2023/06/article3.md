# 對角線論證法

## Cantor 的壯舉

對角線論證法（Diagonal Argument）是 Georg Cantor 在 1891 年提出的證明技巧，用於證明實數是不可數的。這個技巧後來成為計算理論中最核心的證明方法之一，深刻影響了 Gödel、Turing 和 Church 的成果。

## 實數不可數的證明

Cantor 的證明思路如下：

假設我們可以列出所有 (0, 1) 區間內的實數。將它們用二進位小數表示：

```
r₁ = 0.0 1 0 1 0 1 0 1 ...
r₂ = 0.1 0 1 0 1 0 1 0 ...
r₃ = 0.0 0 1 1 0 0 1 1 ...
r₄ = 0.1 1 0 0 1 1 0 0 ...
r₅ = 0.0 1 0 0 1 0 1 1 ...
...
```

現在構造一個新數 d：它的第 i 位小數與 rᵢ 的第 i 位不同（如果 rᵢ 的第 i 位是 0，d 的第 i 位就是 1；反之亦然）。

```
d = 0.1 0 0 1 0 ...
```

d 與列表中的每一個 rᵢ 都不同（因為它在第 i 位上與 rᵢ 不同）。因此 d 不在列表中——但我們假設列表包含了所有實數。矛盾！

這證明了實數比自然數「更多」。這種「不同大小的無限」的概念在當時是革命性的。

## 從 Cantor 到 Turing

Cantor 的對角線論證法是 Turing 證明停機問題不可判定的直接靈感來源。Turing 的證明步驟與 Cantor 的證明可以一一對應：

| Cantor 的證明 | Turing 的證明 |
|-------------|-------------|
| 列出所有實數 | 列出所有程式 |
| 構造一個與對角線不同的數 | 構造一個與對角線行為相反的程式 |
| 這個數不在列表中 | 這個程式不在列表中 |
| 實數不可數 | 停機問題不可判定 |

## 對角線技巧的通用模式

對角線論證法的通用模式是：

1. 假設一個集合 S 包含所有具有某種性質的元素
2. 對 S 中的元素進行編號：s₁, s₂, s₃, ...
3. 構造一個新元素 d，它在第 i 個屬性上與 sᵢ 不同
4. d 不在 S 中，但根據定義應該在 S 中 → 矛盾
5. 因此 S 不可能存在（或不可能被列舉）

## 對角線論證法在計算理論中的應用

### Gödel 不完全性定理

Gödel 使用類似對角線的技巧構造了一個自指陳述式：「這個陳述式無法被證明」。如果這個陳述式可以被證明，它就說自己無法被證明；如果無法被證明，它就是真的但無法被證明。

### 停機問題

Turing 構造了一個程式 paradox：它讀取一個程式的描述，然後在這個程式對自身進行分析時做出與分析結果相反的行為。

### 語言不可判定性

對角線論證法也可以直接用來證明某些形式語言是無法被圖靈機判定的：

```
L_d = { wᵢ | 圖靈機 Mᵢ 不接受 wᵢ }
```

L_d 就是「對角線語言」，它與每個圖靈機的接受語言在至少一個輸入上不同。

## 對角線論證的哲學意義

對角線論證法揭示了數學和計算中的一個深刻事實：任何試圖用有限描述來捕捉無限多樣性的系統都存在盲點。這直接導致了以下結論：

- 在任何強大的數學系統中，都存在無法被證明或否定的命題
- 在任何足夠強大的計算系統中，都存在無法被演算法解決的問題
- 自我指涉與對角線論證密不可分
