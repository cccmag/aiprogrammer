# 可歸約性與 Rice 定理

## 歸約的概念

歸約（Reduction）是計算理論中最強大的工具之一。簡而言之，歸約就是將一個問題轉換為另一個問題。

如果有問題 A 和問題 B，且存在一個轉換方法將 A 的任何實例轉換為 B 的實例，使得 A 的答案可以從 B 的答案中推導出來，我們就說「A 可以歸約到 B」。

符號表示：A ≤ B（A 可以歸約到 B）

歸約的關鍵性質：
- 如果 A 可以歸約到 B，且 B 是可判定的，那麼 A 也是可判定的
- 等價地：如果 A 可以歸約到 B，且 A 是不可判定的，那麼 B 也是不可判定的

## 使用歸約證明不可判定性

歸約是證明問題不可判定的核心技術。假設我們想證明問題 B 不可判定：

1. 找到一個已知的不可判定問題（如停機問題）
2. 將該已知問題歸約到 B
3. 如果 B 是可判定的，那麼該已知問題也是可判定的——矛盾！
4. 因此 B 不可判定

### 範例：空輸入停機問題

考慮「空輸入停機問題」：給定一個程式 P，P 在空輸入上是否會停機？

我們可以從標準停機問題歸約到這個問題：

```python
def reduce(P, x):
    # 構造新程式 Q，它在空輸入下模擬 P(x)
    def Q():
        simulate(P, x)
    return Q
```

如果我們能解決空輸入停機問題，我們就能解決標準停機問題——矛盾。因此空輸入停機問題不可判定。

## Rice 定理

Rice 定理是計算理論中的一個強大理論，它給出了程式行為不可判定的統一框架：

> **Rice 定理**：任何關於程式行為的非平凡語義性質都是不可判定的。

所謂「非平凡性質」是指：
- 有些程式具有該性質，有些程式不具有該性質（不為空）
- 不是所有程式都具有該性質（不為全集）

所謂「語義性質」是指與程式的具體實作無關，只與程式的輸入輸出行為有關的性質。

### Rice 定理的應用

以下性質都是不可判定的：

1. **程式是否會停機**（就是停機問題本身）
2. **程式是否會輸出特定值**（如「是否會輸出 42」）
3. **兩個程式是否等價**（對所有輸入產生相同輸出）
4. **程式是否為全函式**（對所有輸入都會停機）
5. **程式是否在特定輸入上傳回特定值**

相比之下，以下性質是可判定的（因為它們是語法性質，而非語義性質）：

1. **程式是否包含特定關鍵字**（可以在原始碼中直接檢查）
2. **程式的行數是否少於 100**
3. **程式是否使用遞迴**

### Rice 定理的證明思路

Rice 定理的證明也是透過歸約。假設 P 是一個非平凡的語義性質。取一個不具有性質 P 的程式 A，和一個具有性質 P 的程式 B。然後將停機問題歸約到「判斷程式是否具有性質 P」這個問題：

給定一個停機問題的實例 (M, w)，我們構造一個新程式 N：

```python
def N(x):
    simulate(M, w)
    return B(x)
```

如果 M(w) 會停機，那麼 N 的行為與 B 完全相同，因此 N 具有性質 P。
如果 M(w) 不會停機，那麼 N 對所有輸入都不會停機，因此 N 不具有性質 P。

這樣，判定 N 是否具有性質 P 就等價於判定 M(w) 是否會停機。

## 歸約的技巧總結

| 技巧 | 說明 | 範例 |
|------|------|------|
| 直接模擬 | 將問題 A 的計算包裝在問題 B 中 | 停機問題 → 空輸入停機問題 |
| 忽略輸入 | 在歸約過程中丟棄不需要的輸入 | 全函式判定 → 停機問題 |
| 程式變換 | 對程式進行語法轉換 | Rice 定理的證明 |
| 配對 | 同時考慮多個實例 | 兩個程式是否等價 |
