# 本月新知

## 2023 年 6 月程式與 AI 技術動態

### AI 與機器學習

**GPT-4 持續發燒，ChatGPT 插件生態成形**

OpenAI 在 2023 年 3 月發布 GPT-4 後持續推出更新。6 月 ChatGPT 插件系統正式對所有 Plus 用戶開放，開發者可以建立各種第三方工具整合，從程式碼執行器到圖片生成工具，打造 AI Agent 的生態雛形。同時期的 LangChain 框架迅速崛起，成為建構 LLM 應用的標準工具。

**Anthropic 發布 Claude 2**

Anthropic 於 6 月推出 Claude 2，支援最高 100K token 的上下文視窗，在程式碼理解、文檔摘要與多輪對話方面表現出色。Claude 2 的推出預示著長上下文能力的競賽開跑，也奠定了 Anthropic 在 AI 安全與對齊研究領域的領導地位。

**Llama 2 開源前夕**

Meta 的 LLaMA 模型於 2 月洩漏後引發開源 LLM 浪潮。6 月 Meta 宣布將於 7 月正式發布 Llama 2，支援商業使用，引發社群高度期待。這場開源 LLM 革命將徹底改變 AI 的生態格局。

**Stable Diffusion XL 登場**

Stability AI 於 6 月發布 Stable Diffusion XL 候選版本，參數量提升至 3.5B，在圖片品質、構圖與文字理解上有大幅進步。開源圖像生成模型正式進入高品質時代。

### 程式語言與框架

**Python 3.12 Beta 發布**

Python 3.12 進入 Beta 階段，引入更清晰的錯誤訊息、`typing` 模組大規模改進，以及針對 CPython 的效能優化。新的 f-string 語法支援巢狀引號，讓開發體驗更佳。

**Rust 1.70 穩定版**

Rust 發布 1.70 版本，正式穩定化 `OnceCell` 與 `OnceLock` 這兩個惰性初始化型別，並大幅改善了編譯器錯誤訊息的可讀性。Rust 在 Linux Kernel 中的採用持續增加。

**TypeScript 5.1 發布**

TypeScript 5.1 於 6 月發布，新增 `undefined` 回傳類型的自動推斷、`@link` JSDoc 標籤支援，以及效能與編輯器體驗的改善。TypeScript 的採用率持續攀升。

**React Server Components 穩定化**

React 團隊在 6 月宣布 React Server Components 進入穩定階段，Next.js 13.4 的 App Router 成為預設推薦。這項設計改變了前端開發的思維方式，將部分邏輯移至伺服器端執行。

### 開發工具與平台

**GitHub Copilot 普及化**

GitHub Copilot 於 6 月正式全面開放，支援 JetBrains IDE 與 Neovim。Copilot Chat 也進入公開預覽，開發者可以在 IDE 中直接與 AI 對話。這是 AI 輔助程式設計走向主流的重要里程碑。

**AWS 推出生成式 AI 工具**

AWS 於 6 月宣布多項 AI 服務：Amazon Bedrock 提供基礎模型 API 存取、Amazon CodeWhisperer 全面開放、Amazon SageMaker 新增 LLM 微調能力。雲端巨頭正式進入 AI 軍備競賽。

**Docker 發布 Docker Compose v2**

Docker Compose v2 正式取代 v1，支援 GPU 資源分配與更完善的服務依賴管理，為 AI 開發環境的容器化部署提供更好的支援。

### 標準與規範

- **ECMAScript 2023 正式發布**：納入 `Array.prototype.toSorted()`、`toReversed()` 等不可變陣列方法
- **WebGPU 進入 Chrome 穩定版**：為 Web 上的高效能 GPU 運算開啟大門
- **HTTP/3 採用率突破 30%**：QUIC 協定逐漸成為主流

### 學術與理論

- **Google DeepMind 發布 AlphaDev**：利用強化學習發現更快的排序演算法，這是 AI 協助演算法發現的里程碑
- **SAT 競賽 2023 結果出爐**：新的 CDCL 求解器在隨機實例上取得重大突破
- **P vs NP 問題**：本月沒有重大突破，但在 circuit complexity 領域有若干進展

### 業界動態

- **微軟與 OpenAI 合作深化**：Azure 成為 OpenAI API 的獨家雲端供應商
- **Hugging Face 獲 2.35 億美元融資**：估值達 45 億美元
- **PyTorch 2.0 採用度持續攀升**：torch.compile 動態圖優化為深度學習訓練帶來 40% 加速
