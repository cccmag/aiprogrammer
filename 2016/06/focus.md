# 本期焦點：正規語言與狀態機

## 什麼是正規語言？

正規語言（Regular Language）是形式語言理論中最基本的語言類別之一。它們可以用有限狀態機識別，也可以用正規表達式描述。

正規語言在電腦科學中無處不在：

- 詞彙分析（Lexical Analysis）
- 文字搜尋和替換
- 網頁表單驗證
- 網路協議解析

## 狀態機的直覺理解

狀態機是一種計算模型，根據輸入從一個狀態轉換到另一個狀態：

```
         a
    ┌─────────┐
    │         ▼
→ q0 ──────► q1 ◄──┐
    ▲         │    │
    │         │ b  │
    └─────────┴────┘
```

例如，檢測二進位字串是否包含 "01" 的狀態機：

1. 初始狀態 q0：未看到 "01"
2. 讀到 0，轉換到 q1：剛看到 0
3. 讀到 1，轉換到 q2：看到了 "01"
4. q2 是接受狀態

## 為何要學習正規語言理論？

### 理論價值

正規語言理論是計算理論的基石：

- **Chomsky 層級**：正規語言在最底層，之上是上下文無關語言、上下文有關語言、遞迴枚舉語言
- **可計算性理論**：理解什麼可以被計算
- **複雜度理論**：理解計算的資源消耗

### 實用價值

- 編譯器設計
- 文字處理
- 電路設計
- 形態分析

## 形式語言的層級

```
 Chomsky 層級：

 第 0 層：無限制文法（任意規則）
           ↓ Turing 機可識別
 第 1 層：上下文有關文法
           ↓ 線性有界自動機
 第 2 層：上下文無關文法
           ↓ Pushdown 自動機
 第 3 層：正規文法
           ↓ 有限自動機
```

## 我們從哪裡來，要往哪裡去

1956 年，Noam Chomsky 提出了形式語言的層級理論。這個理論為編譯器設計和計算理論奠定了基礎。
