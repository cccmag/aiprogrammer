# 主題一：正規語言基礎

## 字母表

**字母表（Alphabet）** 是符號的有窮集合。

常用字母表：
- Σ = {0, 1}  -- 二進位字母表
- Σ = {a, b, c, ..., z}  -- 拉丁字母表
- Σ = {0, 1, 2, ..., 9}  -- 數字
- ASCII 字元集
- Unicode 字元集

## 字串

**字串（String）** 是字母表中符號的有窮序列。

### 基本定義

- **空字串**：長度為 0 的字串，記為 ε（epsilon）
- **字串長度**：|w| 表示字串 w 中符號的數量
- **字串連接**：vw 表示將字串 v 和 w 連接

### 字串運算

```python
# 字串長度
w = "hello"
print(len(w))  # 5

# 字串連接
v = "world"
print(w + v)  # "helloworld"

# 空字串
empty = ""
print("hello" + "" + "world")  # "helloworld"
```

### 字串的冪

```python
def power(s, n):
    """s 的 n 次冪：n 個 s 連接"""
    if n == 0:
        return ""
    return s + power(s, n - 1)

print(power("ab", 3))  # "ababab"
```

## 語言

**語言（Language）** 是某個字母表上字串的集合。

### 形式定義

如果 Σ 是字母表，則語言 L 是 Σ* 的子集：

```
L ⊆ Σ*
```

### 語言範例

```python
# 二進位字母表
Σ = {'0', '1'}

# 偶數長度的字串
L1 = {ε, "00", "11", "0000", "0101", ...}

# 以 1 開頭的二進位數
L2 = {"1", "10", "11", "100", "101", "110", "111", ...}

# 包含 "00" 的字串
L3 = {"00", "001", "100", "000", ...}
```

## 語言運算

### 三大運算

假設 L 和 M 是語言：

1. **並集**：L ∪ M = {w | w ∈ L 或 w ∈ M}
2. **連接**：LM = {vw | v ∈ L 且 w ∈ M}
3. **克林閉包**：L* = {w | w = ε 或 w = x1x2...xn，其中 xi ∈ L}

### 克林閉包 vs 正閉包

```python
Σ = {"a", "b"}

# 克林閉包 L*：包含空字串
L_star = {"", "a", "b", "aa", "ab", "ba", "bb", "aaa", ...}

# 正閉包 L+：不包含空字串
L_plus = {"a", "b", "aa", "ab", "ba", "bb", "aaa", ...}
```

### 語言的交集

```python
# L1 ∩ L2：同時屬於 L1 和 L2 的字串
L1 = {"0", "1", "00", "01", "10", "11"}
L2 = {"0", "00", "000", "0000", ...}

intersection = {"0", "00"}  # 以 0 開頭的純 0 字串
```

## 正規語言

**正規語言（Regular Language）** 是可用以下方式定義的語言：

1. **有限自動機（DFA 或 NFA）**
2. **正規表達式**
3. **正規文法（3 型文法）**

### 正規語言的封閉性

正規語言對以下運算封閉：

- 並集
- 連接
- 克林閉包
- 交集
- 補集
- 反轉

```python
# 正規語言的封閉性示例
# 如果 L1 和 L2 是正規的，則 L1 ∪ L2 也是正規的
# 如果 L 是正規的，則 L* 也是正規的
```

## 語言的表示法

### 列舉法

```python
L = {"00", "01", "10", "11"}
```

### 描述法

```python
# L = {w | w 是長度為 2 的二進位字串}
L = {w for w in ["00", "01", "10", "11"] if len(w) == 2}
```

### 語法法（正規表達式）

```python
# L = (0|1)(0|1)  -- 長度為 2 的二進位字串
import re
pattern = r"^[01]{2}$"
strings = ["00", "01", "10", "11", "010"]
matching = [s for s in strings if re.match(pattern, s)]
```

## 語言的勢

### 有窮 vs 無窮

- 有窮語言：只有有限個字串
- 無窮語言：有無限多個字串

```python
# 有窮語言
L1 = {"hello", "world"}

# 無窮語言
L2 = {"0", "00", "000", "0000", ...}  # 以 0 開頭，任意長度的純 0 字串
```

## 與自動機的關係

語言 L 是正規的當且僅當存在 DFA M 使得 L = L(M)。

```python
# 對應的 DFA：接受所有長度為 2 的二進位字串
# q0 --0--> q1 --0--> q2 (accept)
# q0 --1--> q1 --1--> q2 (accept)
```
