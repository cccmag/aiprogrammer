# 本月新知

## 2016 年 6 月程式與 AI 技術動態

### 程式語言與編譯器

**Swift 3.0 Preview 發布**

Apple 在 2016 年 WWDC 發布 Swift 3.0 Preview，帶來大幅度的 API 改動。Swift 從 Objective-C 繼承的語法正在被重新設計，使其更加現代化和一致。Swift 正式成為跨平臺語言，支援 Linux。

**ECMAScript 2016（ES7）正式發布**

2016 年 6 月，ECMAScript 2016 正式成為標準。這是首個根據新流程（小版本、快速發布）制定的 ECMAScript 版本。新特性包括：

- 指數運算子（`**`）
- `Array.prototype.includes()`

**TypeScript 2.0 開發進展**

Microsoft 的 TypeScript 2.0 持續開發中，預計將引入非空類型、更好的類型推斷等特性。Angular 2 的快速發展推動了 TypeScript 的普及。

**Rust 1.10 發布**

Rust 1.10 持續改進編譯時間和 Cargo 功能。Rust 的生態系統持續成長，crates.io 上的套件數量突破 10,000 大關。

### 人工智慧

**AlphaGo 論文發表**

DeepMind 在 Nature 發表論文，詳細介紹 AlphaGo 的技術細節。策略網路（Policy Network）和價值網路（Value Network）的設計借鑒了深度學習和強化學習的結合。

**深度學習框架競爭**

- TensorFlow 1.0 正式發布，成為最受歡迎的開源 ML 框架
- Facebook 開源 Caffe2
- Amazon 宣佈支援 MXNet
- Microsoft 持續改進 CNTK

**強化學習突破**

DeepMind 展示如何讓 AI 從零開始學習 Atari 遊戲，無需人類知識。這種「通用」強化學習方法展示了 AI 的潛力。

### 開發工具與框架

**React 15.2 發布**

React 15.2 帶來了更好的錯誤處理和效能改進。React 生態系統持續繁榮，Redux、React Router 等工具成為標準配置。

**Angular 2 Beta 版**

Angular 2 進入 Beta 階段，採用 TypeScript 和全新的元件架構。離線編譯和更好的行動裝置支援是主要焦點。

**GitHub 持續改進**

GitHub 推出 Projects、Insights 等新功能，改善專案管理和分析能力。

### 雲端與運算

**AWS Lambda 更新**

AWS Lambda 新增 Python 支援和更多觸發器。Serverless 架構越來越受歡迎。

**容器化持續發展**

Kubernetes 1.3 發布，改善了 federated clusters 和 PetSets 的支援。Docker 1.12 整合了 Swarm Mode。
