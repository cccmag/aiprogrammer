# 程式碼說明 — 密碼強度檢測腳本

## 功能概述

`_code/pw_check.py` 是一個密碼強度檢測工具，幫助開發者與使用者評估密碼的安全性。腳本會檢查密碼的長度、字元組成、是否包含常見詞彙、以及是否在已知外洩資料庫中。

## 檢測項目

### 1. 基本長度檢查

密碼長度是安全性最基本的要素。建議最少 8 個字元，理想情況下應該 12 個字元以上。

### 2. 字元組成檢查

混合不同類型的字元可以大幅增加密碼空間：
- 小寫字母（26 個字元）
- 大寫字母（26 個字元）
- 數字（10 個字元）
- 特殊符號（32 個常見符號）

字元類型越多，暴力破解難度越高。

### 3. 暴力破解時間估算

基於假設攻擊者每秒可嘗試 10 億次（現代硬體），計算密碼被破解的最長時間。

### 4. 簡易模式比對

檢查密碼是否包含常見詞彙或模式，這些密碼在字典攻擊中會被快速破解。

## 執行方式

```bash
cd _code
python3 pw_check.py
```

## 建議

即使密碼看起來很強，仍建議：
- 不要在多個網站使用同一個密碼
- 使用密碼管理器產生並儲存唯一密碼
- 啟用雙因素認證
- 定期更換密碼（特別是在資料外洩事件後）
