# 2016 年 11 月 — DevOps 與持續交付新聞

## Jenkins 2.0 長期支援版發布

Jenkins 2.0 LTS 版本正式發布，帶來 Pipeline as Code 原生支援與大幅改善的使用者體驗。

## Docker 1.12 Swarm Mode

Docker 1.12 將 Swarm Mode 整合為原生功能，提供內建的容器叢集管理與服務發現。

## Kubernetes 1.4 發布

Google 發布 Kubernetes 1.4，簡化安裝流程，新增聯邦（Federation）支援，大幅提升大規模部署能力。

## AWS 容器服務更新

Amazon ECS 支援 Docker Compose，簡化容器化應用的部署流程。

## GitHub Enterprise 2.8

GitHub Enterprise 新增多方改進，包括更好的 CI/CD 整合與程式碼審查工具。

## DevOps 成熟度報告

根據 2016 年 DevOps 報告，高效能組織的部署頻率提升 30 倍，恢復時間縮短 99%。

## Terraform 0.7 發布

HashiCorp 發布 Terraform 0.7，增強 Azure 支援與模組化基礎設施管理。

## 自動化測試趨勢

容器化測試環境成為主流，測試速度平均提升 60%。

## 開源 CI 工具生態

GitLab CI、CircleCI、Travis CI 持續競爭，推動 CI/CD 工具的快速創新。
