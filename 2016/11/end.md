# 結語

## DevOps：文化、自動化與持續改進

DevOps 不只是一套工具，更是一種文化、一種思考方式。它打破開發與維運的壁壘，讓整個團隊共同為軟體的品質與交付負責。

### 關鍵學習

1. **CI/CD 是基礎** — 自動化建置、測試、部署是 DevOps 的核心
2. **容器化簡化交付** — Docker 讓「一次建置，到處執行」成真
3. **IaC 讓基礎設施可控** — Terraform、Ansible 讓基礎設施版本化
4. **監控與日誌是眼睛** — 看不見就無法管理
5. **安全是每個人的責任** — DevSecOps 將安全整合到每個環節

### 下一步行動

- 評估現有 CI/CD 流程，找出瓶頸
- 學習一種 IaC 工具（Terraform 或 Ansible）
- 建立監控與日誌基礎設施
- 推動團隊建立 blameless post-mortem 文化
