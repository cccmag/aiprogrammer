# 2016 年 10 月 — 軟體測試新聞

## Jest 16.0 發布

Facebook 發布 Jest 16.0，引進平行測試執行與改進的失敗訊息，大幅提升 JavaScript 測試效率。

## Kubernetes 1.4 增強監控功能

Google 發布 Kubernetes 1.4，增強 Prometheus 整合，讓容器化應用的監控與測試更加便利。

## Docker 1.12 支援健康檢查

Docker 1.12 正式支援容器健康檢查（HEALTHCHECK指令），為容器化應用的品質確保提供原生支援。

## GitHub 企業版 2.8 發布

GitHub Enterprise 2.8 增強了程式碼審查與 CI/CD 整合功能，進一步推動開發流程自動化。

## Jenkins 2.0 長期支援版

Jenkins 2.0 LTS 成為企業 CI/CD 標配，Pipeline as Code 概念獲得廣泛採用。

## Selenium 3.0 測試框架

Selenium 3.0 發布，放棄對 Firefox 舊版支援，全面支援 WebDriver 協議。

## 單元測試覆蓋率工具進化

Codecov 與 Coveralls 等程式碼覆蓋率服務持續改進，提供更精細的測試報告與趨勢分析。

## 軟體品質管理趨勢

根據 2016 年軟體品質報告，越來越多企業採用「測試左移」策略，在開發早期階段就投入測試資源。
