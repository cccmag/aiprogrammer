# 本月科技新聞 — 2016年2月

## Docker 1.10 正式發布

2016 年 2 月 Docker 發布了 1.10 版本，這是史上最大幅度的更新之一。新的 Swarm Mode 整合原生叢集管理，讓多台 Docker 主機可以形成一個叢集，統一管理容器部署與調度。網路功能全面翻新，支援 CNM（Container Network Model）與 overlay 網路，容器之間的服務發現不再需要依賴外部工具。

## CoreOS 與 Docker 合作

CoreOS 與 Docker 宣布在 Kubernetes 與容器標準上展開合作。CoreOS 長期推動開源容器標準，包括 App Container Image（appc）規範與 rkt 容器執行環境。這次合作被視為回應社群對開放入場標準的呼聲。

## Kubernetes 1.2 發布

Google 發布 Kubernetes 1.2，增加了多項期待已久的功能，包括 Federation（跨叢集管理）、更完善的水平自動擴展、以及簡化的部署方式。Kubernetes 在容器編排領域的影響力持續擴大。

## 容器化安全議題受關注

多起基於容器映像的安全漏洞被發現，促使業界更重視容器安全。Docker 官方開始推動官方映象的簽名機制，並提供安全掃描服務。企業在採用容器技術時，安全合規成為必須考量的因素。

## 雲端容器服務競爭激烈

AWS 推出 EC2 Container Service（ECS）的重大更新，Google Container Engine（GKE）支援 Kubernetes 1.2，Azure 也強化了 Azure Container Service。三大雲端平台在容器編排服務上的競爭趨於白熱化。
