# 回顧與結語

## 2016 年 7 月號結語

### 本月主題回顧

七月是電腦圖學與遊戲程式發展的重要月份。本期雜誌聚焦了 3D 渲染技術、遊戲引擎架構和 WebGL 程式設計的理論與實戰。

#### 電腦圖學的啟示

電腦圖學是遊戲、VR/AR 和視覺化的核心基礎。本期的七篇焦點文章系統性地介紹了圖學的核心知識：

1. **電腦圖學導論**：從歷史演進到現代渲染技術，建立了對 3D 圖學的整體認識
2. **OpenGL 與渲染管線**：從固定功能管線到可程式化管線的演進
3. **WebGL 與瀏覽器 3D**：用 JavaScript 在瀏覽器中實現 3D 渲染
4. **3D 數學基礎**：向量、矩陣和四元數是圖學的數學工具
5. **著色器程式設計**：GLSL 是 GPU 程式設計的語言
6. **遊戲引擎架構**：元件系統、渲染架構和物理系統
7. **效能優化與陰影**：陰影映射、延遲渲染和 instancing

#### 本月技術亮點

**Unity 5.4** 的 VR 支援改進讓虛擬實境開發更加普及。原生支援 Oculus Rift 和 HTC Vive，以及多執行緒渲染增強，使 Unity 繼續保持領先的遊戲引擎地位。

**WebGL 2.0** 的逐步落地為瀏覽器 3D 開發帶來了新可能。3D 紋理、多渲染目標和增強的實例化渲染，讓 WebGL 應用的視覺效果達到接近原生的水準。

**NVIDIA Pascal 架構** 的發布是遊戲顯示卡的重大進步。GTX 1080 和 1070 的效能提升，為即時光線追蹤的未來應用奠定了硬體基礎。

**Vulkan 1.0** 生態系統的持續成長，預示著未來遊戲將有更多的圖形 API 選擇。這種開放標準的推廣，有助於打破專有 API 的壟斷。

**Oculus Rift 消費者版出貨** 標誌著 VR 正式進入消費市場。這開啟了一個全新的互動體驗時代，對遊戲和應用開發者提出了新的挑戰。

### 未來展望

**VR/AR 的普及**：隨著硬體成本下降和技術成熟，虛擬實境和擴增實境將逐漸走向大眾。

**即時光線追蹤**：NVIDIA RTX 顯示卡開啟了消費級即時光線追蹤的可能性，未來遊戲的視覺效果將更加逼真。

**WebGL 持續進化**：瀏覽器中的 3D 圖形能力將越來越強，可能成為重要的遊戲平台。

**跨平台開發**：一套程式碼，多平台部署將成為常態，減少開發者的維護成本。
