# 本期焦點

## 電腦圖學與遊戲程式：從 OpenGL 到 WebGL

### 引言

電腦圖學（Computer Graphics）是計算機科學中最令人著迷的領域之一。從 1960 年代的第一個 3D 演算法，到今天的即時光線追蹤，電腦圖學經歷了驚人的發展。而遊戲程式設計，則是這些理論與技術的最佳實踐場域。

2016 年是遊戲技術的重要一年：VR 進入消費市場、WebGL 2.0 逐步落地、DirectX 12 和 Vulkan 等新一代圖形 API 開始普及。本期雜誌將帶領讀者探索電腦圖學與遊戲程式的核心技術。

## 濃縮回顧

### 為什麼要學電腦圖學

電腦圖學是遊戲、VR/AR、科學視覺化、影視特效等領域的基礎。理解圖學原理能讓你：

- **開發自己的遊戲引擎或渲染器**
- **最佳化現有遊戲的效能**
- **解決圖學相關的技術問題**
- **創新視覺效果**

### 圖學流水線的演進

從固定功能管線（Fixed Function Pipeline）到可程式化管線（Programmable Pipeline），再到今天的完全可自訂渲染管線，開發者拥有的控制權越來越大，但同時也需要更多的專業知識。

### WebGL 的崛起

WebGL 讓瀏覽器成為一個強大的 3D 平台。透過 Three.js、Babylon.js 等框架，即使不深入了解 WebGL API，也能快速開發出精美的 3D 網頁應用。

### 未來趨勢

- **VR/AR 整合**：遊戲與沉浸式體驗的界線日益模糊
- **光線追蹤硬體**：NVIDIA RTX 顯示卡開啟即時光追新時代
- **跨平台開發**：一套程式碼，多種平台

## 結論與展望

電腦圖學與遊戲程式是一個既有趣又具挑戰性的領域。它需要數學、物理、程式設計和藝術感的結合。2016 年的技術發展顯示，這個領域仍在快速進步中。

無論你是想開發自己的遊戲，還是希望在 VR/AR 領域發展，電腦圖學的基礎知識都是不可或缺的。本期的七篇焦點文章將帶你從基礎到實作，全面掌握這個領域的核心技術。
