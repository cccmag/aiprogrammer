# 本月科技新聞 — 2016年1月

## AWS 2015 re:Invent 回顧：新服務與功能

2015 年 11 月舉辦的 AWS re:Invent 大會發布了多項重要更新。Lambda 支援 Python、涵蓋更多觸發事件來源；EC2 推出 T2 Unlimited 讓小型執行個體可彈性超越基準效能；S3 發布 Life Cycle Migration 與 Cross-Region Replication 提升資料管理彈性；Snowball 邊緣運算資料傳輸服務正式上市。這些功能在 2016 年初已全面可用，成為企業採用雲端的重要推力。

## Docker 1.10 重大更新

Docker 在 2016 年 2 月發布 1.10 版本，帶來多項實用功能。新的 Swarm Mode 整合原生叢集管理，不再需要額外安裝 Docker Swarm；網路功能大翻新，支援 overlay 網路與 service discovery；映像層級的內容位址儲存提升安全性與效能。這次更新讓容器編排能力大幅躍進。

## 雲端採用持續成長

根據 RightScale 2016 年雲端狀況報告，93% 的受訪企業已採用雲端策略，其中公有雲採用率達 82%，混合雲達 71%。AWS 仍是市場領導者，但 Azure 與 GCP 的採用率持續上升。值得注意的是，企業開始更重視雲端成本最佳化，平均有 30% 的雲端支出被浪費。

## 安全性話題持續發燒

1 月傳出多家大型企業的 AWS S3 儲存桶因設定錯誤而暴露敏感資料，包括美國國防部的情報資料外洩事件。這些事件凸顯雲端安全設定的重要性，促使雲端供應商加強安全預設值，同時帶動雲端資安培訓需求。

## 物聯網與雲端結合

Google Cloud IoT 和 AWS IoT Platform 在年初持續擴張。邊緣運算的概念開始受到關注，雲端不再只是資料中心，而是擴展到物聯網閘道與終端裝置。這種「雲端到邊緣」的架構趨勢，為 2016 年的技術發展奠定基礎。
