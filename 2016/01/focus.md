# 主題總覽：雲端運算與基礎設施

雲端運算徹底改變了資訊產業的遊戲規則。從傳統自行架設伺服器的高成本、高維護負擔，到現在只要幾分鐘就能啟動一台虛擬機器，雲端讓新創公司也能使用過去只有大型企業才能負擔的運算資源。本期將帶領讀者了解雲端運算的核心概念與主要平台的特色。

## 核心概念

1. **隨選自助服務**：使用者可隨時透過網頁或指令介面取得運算資源，無需人工介入審核。這大幅縮短了環境建置的時間，從數週縮短到數分鐘。

2. **廣泛網路存取**：透過標準網路協定的 API，使用者可從任何地點、任何裝置存取雲端資源。RESTful API 的普及讓系統整合更加容易。

3. **資源池化**：雲端供應商將實體資源虛擬化後形成資源池，多個使用者共享同一組硬體，透過隔離機制確保安全性與效能。這種方式提升了硬體利用率，也降低了單位成本。

4. **快速彈性**：雲端資源可根據負載快速擴充或縮減。電子商務在促銷活動期間可自動擴展容量，活動結束後再縮回，不需為尖峰負載準備永久性硬體。

5. **服務計量**：依實際使用量計費，而非傳統的固定月租費。這種模式讓小型專案與新創公司能以極低成本開始，隨著業務成長再逐步擴展。

## 三大服務模型

雲端服務依抽象層級可分為 IaaS、PaaS、SaaS 三種模型，各有適合的應用場景。IaaS 提供最基礎的虛擬化伺服器，給予使用者最大的控制權與彈性；PaaS 在基礎設施之上再封裝一層平台服務，簡化開發與部署流程；SaaS 則是完整的應用程式服務，使用者無需關心底層技術細節。

## 主要平台比較

AWS 是市場領導者，服務種類最齊全，生態系成熟；Google Cloud Platform 在大資料處理與機器學習領域有獨特優勢；Microsoft Azure 與企業既有 IT 架構整合最佳，適合已經使用 Microsoft 產品的組織。
