# 5. Microsoft Azure 企業應用

## Azure 的定位

Microsoft Azure 的優勢在於與企業既有 IT 架構的緊密整合。對於已經使用 Windows Server、SQL Server、Active Directory、SharePoint 的企業，Azure 提供幾乎無縫的遷移路徑。同時 Azure 也是唯一原生支援 SAP 的三大雲端平台，這對大型企業非常重要。

## Windows 與 .NET 生態系

**Virtual Machines**：支援 Windows 與 Linux 雙平台。Windows VM 與地端的 Active Directory、Group Policy 可完美整合。

**App Service**：PaaS 應用程式服務，支援 .NET、Java、Node.js、PHP、Python。與 Visual Studio 整合，.NET 開發者可快速部署 Web 應用程式。

**Azure DevOps**：原名 Visual Studio Online，提供程式碼托管、持續整合、持續部署、敏捷開發工具。與 GitHub 形成互補。

## 企業級資料庫

**SQL Database**：托管的 SQL Server 相容資料庫服務。支援自動備份、長期保留、威脅偵測效能調校。相容於地端 SQL Server 的 T-SQL 語法，遷移最簡單。

**SQL Server on VM**：想要完全控制 SQL Server 但不想管理硬體，可選擇在 VM 中執行 SQL Server。

**Azure Cosmos DB**：全球分散式 NoSQL 資料庫，支援文件、鍵值、圖形、單欄資料模型。提供 99.999% 可用性 SLA。

## 混合雲能力

**Azure Stack**：讓企業在自己的資料中心執行與 Azure 一致的服務。對於有資料主權要求或網路延遲考量的場景，Azure Stack 提供一致的開發與管理體驗。

**ExpressRoute**：專線連接服務，建立从企业网络到 Azure 的私有高速连接。繞過公共互聯網，確保傳輸穩定性與安全性。

**Azure AD**：雲端身份識別服務，與地端 Active Directory 同步。實現單一登入（SSO）跨雲端與地端應用程式。

## 物聯網與企業解決方案

**Azure IoT Hub**：完整的 IoT 解決方案，支援千萬級別的裝置連接、管理、資料收集。與 Azure Functions、Stream Analytics、Power BI 整合形成完整的 IoT 資料處理流程。

**Azure Machine Learning**：企業級機器學習平台，從資料準備、特徵工程、模型訓練、部署提供完整工具鏈。

**Power BI**：商業智慧視覺化工具，與 Azure 資料服務緊密整合，可快速將資料轉換為互動式報表。

## 對企業的吸引力

Azure 最大的價值在於降低企業遷移到雲端的風險。熟悉的工具、相同的管理概念、與地端系統的緊密整合，讓企業可以在不完全顛覆現有架構的情況下逐步採用雲端。對於大型組織，這種漸進式轉型往往比全面重新架構更實際。
