# 結語

本期介紹了雲端運算的基礎知識與 AWS 核心服務的使用方法。從最早的隨選自助服務到現在完整的生態系，雲端運算已經彻底改变了軟體開發與部署的方式。

## 關鍵 takeaways

1. **雲端已成為標準**：無論是新創公司還是大型企業，雲端運算已成為基礎建設的默認選擇。理解雲端概念不再只是 DevOps 的工作，每位開發者都應該具備基本的雲端知識。

2. **善用免費方案學習**：AWS、GCP、Azure 都提供豐富的免費方案，這是學習雲端技術的最佳起點。建議讀者在閱讀本期雜誌後，立即申請帳戶並實際操作。

3. **安全是第一步**：從帳戶申請開始就要建立良好的安全習慣，包括啟用 MFA、使用 IAM 最小權限原則、設定帳單警示。這些看起來瑣碎的安全措施，卻是保護雲端資源的第一道防線。

4. **自動化勝於手動操作**：雲端環境的規模與複雜度很容易超過手動管理的能力範圍。從一開始就養成使用 CLI 與 Script 的習慣，為未來的基礎設施即程式碼打下基礎。
