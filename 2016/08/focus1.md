# 數位邏輯基礎

## 什麼是數位系統？

數位系統（Digital System）處理的是離散的訊號，通常用 0 和 1 來表示。這與連續變化的類比訊號形成對比。

## 數位與類比

### 類比訊號

- 連續變化的電壓或電流
- 例如：溫度、光線強度、音波
- 優點：自然、直觀
- 缺點：容易受雜訊干擾、難以精確保存

### 數位訊號

- 離散的二進位值（0 或 1）
- 用電壓高低表示：例如 3.3V = 1，0V = 0
- 優點：抗雜訊能力強、易於處理和儲存
- 缺點：需要轉換、有限精度

```
類比訊號：~~~~~/\~~~~/\
數位訊號：_____/    \__
          0  1  0  1  0
```

## 進位制

### 二進位（Binary）

計算機最基本的進位制，只有 0 和 1 兩個數字。

```
十進位 0 = 二進位 0000
十進位 1 = 二進位 0001
十進位 2 = 二進位 0010
十進位 3 = 二進位 0011
...
十進位 15 = 二進位 1111
```

### 十六進位（Hexadecimal）

為了方便表示二進位，使用 0-9 和 A-F 十六個字符。

```
二進位    十六進位  十進位
0000         0        0
0001         1        1
0010         2        2
...
1000         8        8
1001         9        9
1010         A       10
1011         B       11
1100         C       12
1101         D       13
1110         E       14
1111         F       15
```

### 進位轉換

**二進位轉十進位**

```
10110 = 1×2⁴ + 0×2³ + 1×2² + 1×2¹ + 0×2⁰
      = 16 + 0 + 4 + 2 + 0
      = 22
```

**十進位轉二進位**

```
22 ÷ 2 = 11 餘 0
11 ÷ 2 = 5  餘 1
 5 ÷ 2 = 2  餘 1
 2 ÷ 2 = 1  餘 0
 1 ÷ 2 = 0  餘 1
結果：10110
```

## 二補數（Two's Complement）

用於表示有號數的方法。

### 正數

保持不變：+5 = 00000101

### 負數

取反後加一：-5 = 11111011

### 優點

- 0 只有一種表示：00000000
- 減法可以變成加法
- 符號位自動參與運算

## 布林變數

布林代數中的變數只有兩個可能值：TRUE（1）或 FALSE（0）。

## 基本邏輯值

| 符號 | 意義 |
|-----|------|
| 0 | 邏輯 FALSE / 低電位 |
| 1 | 邏輯 TRUE / 高電位 |
| X | 不定值（未知） |
| Z | 高阻抗（浮接） |

## 數位電路分類

### 組合邏輯電路（Combinational Logic）

輸出只取決於當前輸入，無記憶能力。

```
輸入 ──→ [組合邏輯電路] ──→ 輸出
```

### 序向邏輯電路（Sequential Logic）

輸出取決於當前輸入和過去狀態，有記憶能力。

```
           ┌──────────────┐
輸入 ──────→│ 組合邏輯電路 │──→ 輸出
           │              │
           └──←─ 狀態 ←──┘
               (記憶元素)
```

## 時脈與同步

### 時脈訊號

規律變化的方波，用於同步序向邏輯電路。

```
      ┌─┐   ┌─┐   ┌─┐
      │ │   │ │   │ │
──────┘ └───┘ └───┘ └───
   ↑     ↑     ↑
  上升   下降   上升
  邊緣   邊緣   邊緣
```

### 同步 vs 異步

- **同步**：所有正反器由同一時脈驅動
- **異步**：不同元件有不同時脈（少用，但設計複雜）

## 重要的時間參數

| 參數 | 說明 |
|-----|------|
| t_setup | 輸入在時脈邊緣前必須穩定的最小時間 |
| t_hold | 輸入在時脈邊緣後必須保持的最小時間 |
| t_clk→Q | 時脈邊緣到輸出改變的延遲 |
| t_pd | 邏輯閘傳播延遲 |

## 積體電路家族

| 家族 | 全名 | 特點 |
|-----|------|-----|
| TTL | Transistor-Transistor Logic | 速度快、功耗高 |
| CMOS | Complementary MOS | 低功耗、整合度高 |
| ECL | Emitter-Coupled Logic | 超高速 |
