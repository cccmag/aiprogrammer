# 現場可程式化邏輯閘陣列

## FPGA 簡介

現成可程式化邏輯閘陣列（Field-Programmable Gate Array, FPGA）是一種可由使用者程式化的積體電路。

## FPGA 結構

### 基本架構

```
┌────────────────────────────────────────────────────────────┐
│                         FPGA 晶片                           │
├────────────────────────────────────────────────────────────┤
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐        │

│       │            │            │            │             │
│  ┌────┴────┐  ┌────┴────┐  ┌────┴────┐  ┌────┴────┐        │
│  │  CLB    │  │  CLB    │  │  CLB    │  │  CLB    │        │
```

### CLB / Slice / LAB

- **Xilinx**：CLB → Slice → LUT + FF
- **Intel**：LAB → ALM → LUT + FF

### LUT（查表表）

LUT 是 FPGA 的核心，用於實現任意布林函數。

```
輸入 A,B,C → LUT (內容可程式化) → 輸出
```

N 輸入 LUT 可以實現 2^N 種可能的函數。

## FPGA vs CPLD

| 特性 | FPGA | CPLD |
|-----|------|-----|
| 邏輯密度 | 高（數十萬 LUT） | 低（數千園林） |
| 記憶體 | 丰富（Block RAM） | 有限 |
| 觸發器 | 多 | 少 |
| 功耗 | 較高 | 較低 |
| 速度 | 高 | 中 |
| 成本 | 高 | 低 |

## 主要廠商和系列

### Xilinx

| 系列 | 定位 | 應用 |
|-----|------|-----|
| Virtex | 高效能 | 資料中心、國防 |
| Kintex | 高性價比 | 通訊、醫療 |
| Artix | 低功耗 | 嵌入式 |
| Spartan | 低成本 | 消費電子、教育 |

### Intel（原 Altera）

| 系列 | 定位 | 應用 |
|-----|------|-----|
| Stratix | 高效能 | 資料中心、網路 |
| Arria | 中端 | 通訊、軍事 |
| Cyclone | 低成本 | 嵌入式、IoT |
| MAX | 低功耗 | 消費電子 |

### Lattice

- **ECP 系列**：通用
- **iCE 系列**：超低功耗
- **Certus 系列**：汽車級

## 應用領域

1. **通訊**：5G 基站、資料中心交換機
2. **汽車**：ADAS、資訊娛樂
3. **醫療**：醫療影像、監護設備
4. **軍事**：雷達、飛彈導引
5. **運算加速**：AI 推理、區塊鏈
6. **原型驗證**：晶片設計驗證
