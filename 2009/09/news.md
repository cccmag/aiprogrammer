# 本月新知

## 2009 年 9 月程式與 AI 技術動態

### 資料庫技術

**MongoDB 1.0 正式發布**

2009 年 8 月，MongoDB 1.0 正式發布，這是文件資料庫領域的重大里程碑。MongoDB 以其 JSON 風格的文件的模型和水平擴展能力，迅速獲得開發者青睞。

**CouchDB 0.10 發布**

CouchDB 0.10 強化了離線優先的能力，支援更多檢視功能，並改善了效能。

**Redis 2.0 開發中**

Redis 持續發展，2.0 版本將帶來虛擬記憶體和更多的資料結構支援。

### JavaScript 生態

**Node.js 0.2 發布**

Node.js 0.2 帶來了更多的核心模組和效能改進，JavaScript 全端開發的趨勢越來越明顯。

**npm 早期發展**

npm（Node Package Manager）正在快速成長，為 Node.js 生態提供了強大的套件管理能力。

### 雲端運算

**雲端安全議題**

2009 年，雲端運算的安全性成為熱門話題。資料隱私、多租戶隔離、服務可用性等問題受到廣泛關注。

**Google App Engine 支援 Java**

Google 宣布 App Engine 支援 Java，這是繼 Python 之後的第二個語言支援。

### AI 與機器學習

**scikit-learn 活躍發展**

Python 機器學習庫 scikit-learn 持續活跃開發，成為 Python 生態中最重要的 ML 工具。

**詞嵌入技術研究**

2009 年，詞嵌入技術開始受到學術界關注，為後來的 Word2Vec 奠定了基礎。

### 版本控制

**GitHub 持續成長**

GitHub 用戶數持續快速成長，分散式版本控制已成為開源專案的標準。

### 業界動態

- **Twitter API 1.0 穩定**：第三方應用開發指南
- **Facebook Open Graph**：社交圖譜 API 推出
- **Google Wave 發布**：協作平台新想像

### 標準與規範

- **HTML 5 規格討論持續**：Web 應用標準演進
- **OAuth 1.0a 確認**：開放授權標準
- **RESTful API 成為主流**：Web 服務設計模式
