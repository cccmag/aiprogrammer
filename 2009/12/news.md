# 本月新知

## 2009 年技術年度回顧

### 年度總結

2009 年是技術變革的關鍵一年。以下是這一年的重要事件回顧：

### 雲端運算

**雲端運算爆發**
- Amazon AWS 持續成長，EC2 和 S3 成為標準
- Google App Engine 支援 Java
- Microsoft Azure 正式上路
- VMware 進軍雲端

**雲端安全**
- 資料隱私問題受到關注
- 多租戶安全成為議題
- 合規性標準討論

### 資料庫

**NoSQL 運動興起**
- MongoDB 1.0 正式發布（8月）
- CouchDB 0.10 發布
- Redis 2.0 開發中
- Cassandra 0.4 穩定

**關聯式資料庫**
- PostgreSQL 8.4 發布
- MySQL 5.4 發布

### 程式語言

**Ruby**
- Ruby 1.9.1 發布（YARV 效能提升）
- Rails 3.0 開發中（整合 Merb）

**JavaScript**
- Node.js 0.1 發布
- jQuery 1.4 發布

**Python**
- Python 3.1 發布
- Django 1.1 發布

**其他**
- Scala 2.8 開發中
- Go 語言發布（11月）

### 前端技術

**HTML/CSS/JavaScript**
- HTML 5 進入最後呼籲階段
- CSS 3 瀏覽器支援增加
- jQuery 1.4 效能提升 3 倍

### 版本控制

**Git 與 GitHub**
- Git 1.6.5 發布
- GitHub 突破 100 萬用戶
- Git 成為開源專案標準

### 開放平台

**API 與 OAuth**
- OAuth 1.0a 標準化
- Twitter API 1.0 穩定
- Facebook Open Graph 推出
- Stripe、Twilio 成立

### AI 與機器學習

**機器學習**
- scikit-learn 持續發展
- 詞嵌入技術研究興起

**電腦視覺**
- OpenCV 2.0 開發中
- 臉部偵測普及

### 業界動態

- Twitter 快速成長
- Facebook 突破 3 億用戶
- iPhone 3GS 發布
- Android 1.6 發布
- Google Wave 發布

### 標準與規範

- HTML 5 規格討論
- ECMAScript 5 完成
- RESTful API 成為主流
- OAuth 1.0a 標準化
