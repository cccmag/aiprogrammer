### 引言

本期歷史回顧將帶領讀者探索開放平台的概念、設計原則和商業價值。

## 濃縮回顧

### OAuth 的價值

```
OAuth 價值：
- 用戶不需要分享密碼
- 有限授權範圍
- 隨時撤回權限
- 標準化接口
```

### REST 的原則

```
REST 核心原則：
1. 資源導向（everything is a resource）
2. 統一接口（HTTP methods）
3. 無狀態（Stateless）
4. 層次化系統
```
