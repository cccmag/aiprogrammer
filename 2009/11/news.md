# 本月新知

## 2009 年 11 月程式與 AI 技術動態

### 開放平台

**OAuth 1.0a 確認**

OAuth 1.0a 正式成為開放授權的標準，解決了安全 API 訪問的問題。

**Twitter API 1.0**

Twitter API 1.0 提供穩定的 API 接口，第三方應用蓬勃發展。

**Facebook Open Graph**

Facebook 推出 Open Graph API，建立了社交網路的標準。

### 雲端運算

**Google App Engine 支援 Java**

Google 宣佈 App Engine 支援 Java，擴大了平台的語言支援。

**Salesforce Platform**

Salesforce 的 AppExchange 平台持續壯大，展示了企業 SaaS 的可能性。

### 支付和通信

**Stripe 成立**

Stripe 成立於 2009 年，為開發者提供簡單的支付 API。

**Twilio 雲端通信**

Twilio 提供雲端電話和簡訊 API，改變了通信應用的開發方式。

### AI 技術

**機器學習 API**

各种机器学习 API 开始出现，提供图像识别、自然语言处理等能力。

**大數據處理**

Hadoop 和相關技術持續發展，大數據處理成為熱門話題。

### 業界動態

- **雲端運算投資增加**
- **移動應用爆發**
- **社交媒體持續成長**

### 標準與規範

- **OAuth 1.0a 標準化**
- **RESTful API 成為主流**
- **HTML 5 規格討論持續**
