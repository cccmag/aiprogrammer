# 回顧與結語

## 2009 年 11 月號結語

### 本月主題回顧

十一月是開放平台與 API 經濟興起的月份。本期雜誌聚焦了 OAuth、REST API 與平台化策略，以及當月最重要的程式與 AI 技術動態。

#### API 經濟的崛起

2009 年見證了 API 作為商業模式的興起。

```
API 商業模式：

1. 平台中介（Platform）
   - Facebook, Twitter, eBay
   - 吸引第三方開發者
   - 創造網路效應

2. 資料即服務（Data-as-a-Service）
   - Google Maps, AWS
   - 按用量收費
   - 雲端交付

3. 統一接口
   - REST 成為標準
   - 跨平台互通
```

#### OAuth 的成熟

OAuth 1.0 的確立和 OAuth 2.0 的討論，為開放平台提供了安全的授權框架。

### 本月技術亮點

**Twitter API 1.0** 的穩定和 Facebook Open Graph 的推出，顯示了社交平台 API 的成熟。

**Google App Engine** 突破 100 萬應用的里程碑，證明了 PaaS 模式的可行性。

**Salesforce CRM** 的 AppExchange 展示了企業級平台的運作方式。

### 未來展望

**API 管理工具的興起**：Mashery, Apigee 等

**OAuth 2.0 標準化**：更簡單的授權流程

**微服務架構**：小型、專注的 API 服務
