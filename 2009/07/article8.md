# Wolfram Alpha 上線：知識引擎的崛起

## 前言

2009 年 5 月，Wolfram Alpha 正式上線，這是一個與傳統搜尋引擎完全不同的「知識引擎」。它不是搜尋網頁，而是透過結構化知識庫回答用戶的自然語言查詢。

## 什麼是 Wolfram Alpha？

### 與傳統搜尋引擎的比較

```
傳統搜尋引擎 vs Wolfram Alpha：

Google：
- 輸入：「台北人口」
- 輸出：數十億個網頁連結

Wolfram Alpha：
- 輸入：「台北人口」
- 輸出：「台北人口約 264 萬人（2009年）」
         「世界排名第 43 位」
         「人口密度每平方公里 9,700 人」
```

### 核心概念

```markdown
Wolfram Alpha 的特點：

1. 計算知識
   - 不只是檢索，而是計算
   - 使用內建的知識庫和演算法

2. 自然語言輸入
   - 「台灣的 GDP 是多少？」
   - 「蘋果的卡路里」
   - 「地球和火星的距離」

3. 結構化輸出
   - 不只是文字
   - 包含圖表、圖形、表格

4. 即時計算
   - 天氣預報
   - 股票報價
   - 數學計算
```

### API 存取

```javascript
// Wolfram Alpha API 使用示例
var query = '台北的人口';
var url = 'http://api.wolframalpha.com/v1/result?appid=YOUR_ID&input=' +
          encodeURIComponent(query);

fetch(url)
  .then(response => response.json())
  .then(data => {
    console.log(data.result);
  });
```

## 局限性

### 2009 年的限制

```markdown
Wolfram Alpha 的不足：

1. 覆蓋範圍
   - 主要集中於數學、科學、統計
   - 人文、藝術領域不足

2. 自然語言理解
   - 複雜問題處理不佳
   - 歧義處理有限

3. 即時資訊
   - 天氣、金融有延遲
   - 新聞完全不支援

4. 創作能力
   - 只能回答，不能創作
   - 沒有對話能力
```

## 結語

Wolfram Alpha 的上線代表了一種全新的資訊獲取方式。它不是要取代 Google，而是提供了一個互補的服務——直接回答問題而非提供網頁連結。
