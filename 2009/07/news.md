# 本月新知

## 2009 年 7 月程式與網頁技術動態

### 程式語言與框架

**HTML 5 規格接近完成**

W3C 於本月宣布 HTML 5 規格進入「最後呼籲」（Last Call）階段，預計於 2010 年正式定案。HTML 5 引入了多項重要新特性：原生 `<video>` 和 `<audio>` 標籤、`<canvas>` 繪圖、離線儲存、Web Workers、Geolocation API 等。瀏覽器廠商已開始積極實作這些功能。

**jQuery 1.4 發布**

jQuery 團隊於本月發布 jQuery 1.4，這是近年來最大幅度的版本更新。新版本重寫了 DOM 操作核心，效能提升 2-3 倍。全新推出的 `jQuery.fn.animate()` 支援硬體加速動畫，並引入了延遲物件（Deferred Objects）模式。

**Google Chrome OS 發布**

Google 宣布開發 Chrome OS——一款以瀏覽器為核心的雲端作業系統。Chrome OS 基於 Linux 核心，所有應用程式都在 Web 瀏覽器中執行。預計於 2010 年下半年推出，瞄準 Netbook 市場。

**Node.js 0.1.23 發布**

Node.js 在本月發布了 0.1.23 版本，這個事件驅動的伺服器端 JavaScript 框架正在快速成長。Node.js 使用 Google V8 引擎，單執行緒非阻塞 I/O 模型特別適合即時網頁應用。

### 前端技術

**瀏覽器大戰持續**

2009 年 7 月的瀏覽器市場競爭激烈：
- Internet Explorer 8 市佔率約 65%
- Firefox 3.5 發布，市佔率突破 25%
- Google Chrome 3.0 Beta 推出
- Safari 4.0 正式版發布

各家瀏覽器廠商積極實作 CSS 3 新特性，這是 CSS 3 獲得廣泛採用的關鍵時期。

**CSS 3 採用状況**

2009 年是 CSS 3 採用元年。主流瀏覽器開始支援多項 CSS 3 模組：
- Firefox 3.5：支援 text-shadow、opacity、rgba
- Safari 4.0：支援 transform、transition、Web Fonts
- Chrome 3.0：支援 border-radius、box-shadow
- Opera 10：支援 CSS 3 動畫

### AI 與網頁技術

**Google 語音搜尋 API 發布**

Google 發布了語音搜尋 API，讓開發者可以將語音辨識整合到網頁應用中。這項技術基於 Google 的語音辨識模型，支援超過 30 種語言。

**Facebook 人臉辨識功能**

Facebook 推出了人臉偵測和標籤建議功能。這個功能使用機器學習演算法自動偵測照片中的人臉，並建議可以標記的朋友。引發了隱私權的熱烈討論。

**Wolfram Alpha 上線**

由 Stephen Wolfram 創立的 Wolfram Alpha 于2009年5月上線，本月持續獲得關注。這個「知識引擎」不是搜尋網頁，而是透過結構化知識庫回答自然語言查詢。

**Google Wave 發布**

Google 發布了 Google Wave——一個結合影交、即時編輯和社交網路的新型協作平台。Wave 使用 HTML 5 WebSocket 實現即時通訊，被視為電子郵件的未來。
