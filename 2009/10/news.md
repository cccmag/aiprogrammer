# 本月新知

## 2009 年 10 月程式與 AI 技術動態

### Ruby 生態

**Rails 3.0 開發中**

Rails 3.0 正在積極開發中，整合了 Merb 的模組化設計。這將是 Rails 自 1.0 以來最大的版本更新。

**Ruby 1.9.2 預覽**

Ruby 1.9.2 添加了新的功能，包括更好的字串處理和標準庫更新。

**Bundler 0.9**

Bundler 0.9 發布，為 Ruby 專案提供了更好的依賴管理方案。

### Web 開發

**GitHub 持續成長**

GitHub 用戶突破 150 萬，成為全球最大的 Git 代碼托管平台。

**Node.js 生態擴展**

npm（Node Package Manager）開始快速成長，為 Node.js 生態提供了套件管理能力。

### 雲端運算

**雲端 IDE 興起**

多個雲端 IDE 開始出現，包括 Cloud9 IDE 的早期版本。

**Heroku 支援更多語言**

Heroku 宣佈支援 Node.js 和 Java，進一步擴大了平台能力。

### AI 與機器學習

**機器學習框架比較**

各種開源機器學習框架開始受到關注，包括 scikit-learn 的持續發展。

**遊戲引擎進化**

HTML 5 遊戲引擎開始出現，為 Web 遊戲開發帶來新可能。

### 業界動態

- **Google 雲端運算動態**
- **Amazon AWS 新功能**
- **Facebook 開發者大會**

### 標準與規範
