## Ruby 生態系：從語言到平台

### 引言

2009 年是 Ruby 生態系豐收的一年。Ruby 1.9.1 的效能提升、Rails 3.0 的開發、以及活躍的社群，讓 Ruby 繼續在 Web 開發領域保持重要地位。

本期歷史回顧將帶領讀者探索 Ruby 生態系的各個面向。

## 濃縮回顧

### Ruby 發展時間線

```
2004年：Rails 1.0 發布
     ↓
2007年：Rails 2.0
     ↓
2008年：Merb 興起
     ↓
2009年：Rails 3.0 開發中
     ↓
2010年：Rails 3.0 正式發布
```

### Ruby 1.9 的意義

```
Ruby 1.9 vs 1.8：

執行速度：2-5x 提升
記憶體：略微增加
相容性：部分破壞
生態系統：逐步適應
```
