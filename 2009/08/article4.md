# Eclipse 3.5：現代 IDE 的標竿

## 前言

2009 年 6 月，Eclipse 3.5 Galileo 發布，這是 Eclipse 史上最重要的版本之一。新版本帶來了更好的效能、Git 支援增強、以及對多種程式語言的改進支援。

## Eclipse 3.5 的新功能

### 安裝和更新改進

```markdown
Eclipse 3.5 的軟體安裝：

1. 更快的安裝
   - 並行下載
   - 增量更新

2. 新軟體鏡像
   - 更可靠的更新站點

3. 依賴解析改進
   - 自動解決衝突
   - 更清晰的錯誤訊息
```

### Git 支援

```bash
# Eclipse 3.5 內建 Git 支援（EGit）

Team → Share Project → Git

# 功能：
- 提交、拉取、推送
- 分支管理
- 歷史檢視
- 衝突解決
```
