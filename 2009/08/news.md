# 本月新知

## 2009 年 8 月程式與 AI 技術動態

### 程式語言與框架

**Ruby 1.9.1 發布**

2009 年 1 月，Ruby 1.9.1 正式發布，這是 Ruby 1.9 系列的首個穩定版本。Ruby 1.9 採用了 YARV（Yet Another Ruby Virtualmachine）虛擬機，執行效能較 Ruby 1.8 提升了 2-5 倍。

**Python 3.0 發布**

2008 年 12 月，Python 3.0 正式發布，這是 Python 語言自 2000 年以來最大的版本改變。Python 3.0 引入了許多破壞性改變（如 print成為函數、Unicode 預設），推動了語言的現代化。

**Git 1.6.5 發布**

Git 在 2009 年持續快速發展。Git 的分散式版本控制模型已被各大開源專案採用，包括 Linux 核心、Android、Ruby on Rails 等。

### 測試與品質

**QUnit 2.0 開發中**

jQuery 團隊持續改進 QUnit 測試框架，為 JavaScript 單元測試提供更好的工具。

**Jenkins 成立**

2009 年，Kohsuke Kawaguchi 創建了 Jenkins，這是 Hudson 的分支。Jenkins 迅速成為最受歡迎的持續整合工具之一。

### 雲端運算

**雲端安全問題凸顯**

2009 年，隨著雲端運算的普及，安全問題開始受到關注。資料隱私、服務可用性、供應商鎖定等問題成為焦點。

**EC2 價格下降**

Amazon EC2 在 2009 年多次調降價格，推動了雲端運算的普及。

### AI 與機器學習

**scikit-learn 發展**

scikit-learn 在 2009 年持續發展，成為 Python 生態系統中最重要的機器學習庫。

**詞嵌入技術興起**

2009 年，詞嵌入（Word Embedding）技術開始受到關注。Mikolov 等人的研究預示了 Word2Vec 的誕生。

### 開發工具

**Eclipse 3.5 Galileo 發布**

Eclipse 3.5 帶來了更好的效能、Git 支援改進、以及對多種程式語言的增強支援。

**Visual Studio 2010 Beta**

Microsoft 發布了 Visual Studio 2010 Beta，帶來了新的 C++0x 支援和更好的除錯工具。

### 業界動態

- **Google 收購 AdMob**：行動廣告市場整合
- **Twitter 發布 API 1.0**：開放平台的標準化
- **Facebook 突破 3 億用戶**：社交網路的持續成長

### 標準與規範

- **HTML 5 規格討論熱烈**：Web 應用標準持續演進
- **ECMAScript 5 草案完成**：JavaScript 標準化前進
- **RESTful API 成為主流**：Web 服務設計模式
