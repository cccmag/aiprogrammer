# 本月科技新聞 — 2026年1月

## NVIDIA RTX 5090 正式發表

NVIDIA 在 CES 2026 上發表了新一代旗艦顯示卡 RTX 5090，搭載 Blackwell 架構，具備超過 200 TFLOPS 的 AI 運算能力，配備 32GB GDDR7 記憶體，頻寬超過 1.5 TB/s。對 AI 開發者而言，這張卡可以在單卡上訓練 70B 參數的量化模型，大幅降低中型模型訓練的硬體門檻。售價約 1,999 美元，預計 2 月出貨。

## Python 3.14 進入 Beta 階段

Python 3.14 全新改進了錯誤訊息，讓語法錯誤與型態錯誤的提示更具可讀性，直接指出錯誤發生的精確位置。新的 pattern matching 最佳化讓結構化資料處理更快，asyncio 的效能提升則對 AI 推理服務的並發處理有正面影響。建議開發者開始測試套件相容性，特別是底層依賴 C 擴充的科學計算套件。

## Docker 推出 AI 開發工具包

Docker 官方推出 AI Development Kit，整合了 CUDA 12.6、PyTorch 2.5 與 TensorFlow 2.17 的容器映像，並提供了預先配置好的 docker-compose 範本。開發者只需要一行指令就能啟動完整的 GPU 加速開發環境，大幅簡化環境建置流程。支援 VS Code Dev Containers 整合。

## CUDA 12.8 發布

NVIDIA 發布 CUDA 12.8，支援更高效的記憶體管理、更快的編譯速度，並新增了對 Blackwell 架構的 PTX 指令最佳化。對於 Transformer 模型的訓練與推理有 15–20% 的效能提升。建議使用 RTX 40 系列以上的開發者升級。

## VS Code 整合 GitHub Copilot Agent Mode

微軟在 VS Code 中推出 Copilot Agent Mode，不再只是被動的程式碼補全，而是能自動執行多步驟任務，直接操作終端與檔案系統。開發者可以用自然語言描述需求，Agent 自動完成環境建置、套件安裝、除錯與測試。這對 AI 開發者來說，可以大幅減少環境設定與樣板程式碼的撰寫時間。
