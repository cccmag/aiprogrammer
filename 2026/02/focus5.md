# 神經網路直覺

## 生物神經元的啟發

神經網路的概念最初來自生物神經系統的簡化模型。人類大腦由約 860 億個神經元組成，每個神經元接收信號、處理後傳遞給其他神經元。

### 人工神經元：感知器

1958 年，Frank Rosenblatt 提出了感知器（Perceptron），這是第一個人工神經元模型：

```
輸入              權重
x₁ ────── w₁ ──┐
                │
x₂ ────── w₂ ──┼── Σ ── 激活函式 ── 輸出
                │
x₃ ────── w₃ ──┘
```

感知器計算輸入加權和，通過激活函式產生輸出：

```
output = activation(w₁x₁ + w₂x₂ + w₃x₃ + b)
```

## 多層感知器（MLP）

單一感知器只能學習線性決策邊界。為了解決更複雜的問題，我們將多個感知器堆疊成多層結構：

```
輸入層    隱藏層    輸出層
x₁ ──── ● ──── ●
       ╱ ╲ ╱     ╲
x₂ ──── ● ──── ● ──── ● 輸出
       ╲ ╱ ╲     ╱
x₃ ──── ● ──── ●
```

- **輸入層**：接收原始特徵
- **隱藏層**：中間計算層，學習抽象特徵
- **輸出層**：產生最終預測

每層之間的每個連接都有權重，每個神經元都有偏置。

## 激活函式

激活函式引入非線性，讓神經網路能夠學習複雜模式：

```python
import numpy as np

def relu(x):        # ReLU：最常用的激活函式
    return np.maximum(0, x)

def sigmoid(x):     # Sigmoid：輸出範圍 [0,1]
    return 1 / (1 + np.exp(-x))

def tanh(x):        # Tanh：輸出範圍 [-1,1]
    return np.tanh(x)
```

- **ReLU**：計算簡單，有效緩解梯度消失問題
- **Sigmoid**：常用於二元分類的輸出層
- **Tanh**：常用於隱藏層，輸出以零為中心

## 反向傳播

神經網路的訓練依賴反向傳播（Backpropagation）演算法：

1. **前向傳播**：輸入資料從輸入層傳到輸出層，計算預測值
2. **計算損失**：比較預測值與真實值
3. **反向傳播**：從輸出層向輸入層，計算損失對每個權重的梯度
4. **更新權重**：使用梯度下降更新權重

這個過程就是鏈鎖法則（Chain Rule）的應用——將誤差從輸出層逐層分配回前面各層。

## 神經網路的表示能力

**通用近似定理**：只要隱藏層有足夠多的神經元，一個前饋神經網路可以逼近任何連續函式。

這意味著神經網路理論上可以學習任何模式——問題在於如何找到合適的權重。

## 小結

神經網路的關鍵概念：
- **感知器**：最基本的神經元單元
- **多層結構**：透過堆疊提升表示能力
- **激活函式**：引入非線性
- **反向傳播**：梯度計算的核心

這些概念構成了深度學習的基礎。下一篇文章將進一步探討當模型從淺層走向深層時，發生了什麼質的變化。
