# 分類問題：邏輯迴歸

## 二元分類

分類是監督式學習的另一大類任務，目標是將輸入分配到離散的類別中。最簡單的案例是二元分類，例如：
- 一封郵件是垃圾郵件還是正常郵件
- 一個病人是否患有特定疾病
- 一筆交易是否為詐騙

邏輯迴歸（Logistic Regression）雖然名字中有「迴歸」，實際上是最常用的二元分類演算法之一。

## 從線性迴歸到邏輯迴歸

線性迴歸輸出連續值，範圍是 (-∞, +∞)，這不適合直接做分類。邏輯迴歸的核心想法是：先計算線性組合，再通過 Sigmoid 函式將結果映射到 [0, 1] 區間，作為屬於某個類別的機率。

```python
import numpy as np

def sigmoid(z):
    return 1 / (1 + np.exp(-z))
```

Sigmoid 函式的形狀如 S 曲線：

```
輸出 ↑
  1 ──────────────
    |            /
    |          /
  0.5 ──────/────
    |     /
    |   /
  0 ───/────────────→ 輸入
```

當輸入趨近於 +∞ 時，輸出趨近於 1；當輸入趨近於 -∞ 時，輸出趨近於 0；當輸入為 0 時，輸出為 0.5。

## 邏輯迴歸模型

```
P(y=1|x) = σ(w·x + b)

其中 σ(z) = 1 / (1 + e⁻ᶻ)
```

模型輸出的是樣本屬於類別 1 的機率。通常以 0.5 為決策邊界：
- 若 P(y=1|x) ≥ 0.5，預測為類別 1
- 若 P(y=1|x) < 0.5，預測為類別 0

決策邊界對應的是 w·x + b = 0 的超平面。

## 損失函式：交叉熵

線性迴歸使用均方誤差，但邏輯迴歸更適合使用交叉熵損失（也稱為對數損失）：

```
L = -[y·log(ŷ) + (1-y)·log(1-ŷ)]
```

這個損失函式有良好的特性：當預測越接近真實值時損失越小；當預測完全錯誤時損失趨近於無窮大。

```python
def cross_entropy_loss(y_true, y_pred):
    eps = 1e-15
    y_pred = np.clip(y_pred, eps, 1 - eps)
    return -np.mean(y_true * np.log(y_pred) + (1 - y_true) * np.log(1 - y_pred))
```

## 多類別分類

邏輯迴歸可以擴展到多類別分類，稱為 Softmax 迴歸。Softmax 函式將多個輸出轉換為機率分布，確保所有類別的機率總和為 1：

```
P(y=k|x) = exp(wₖ·x) / Σⱼ exp(wⱼ·x)
```

## 邏輯迴歸的優缺點

**優點**：
- 計算效率高，訓練速度快
- 輸出具有機率解釋
- 正則化技術可防止過擬合
- 特徵重要性易於解釋

**缺點**：
- 只能學習線性決策邊界
- 需要手動進行特徵工程
- 對異常值敏感

當資料不是線性可分時，可以透過多項式特徵或核方法提升邏輯迴歸的能力。
