# 從 ML 到深度學習

## 深層革命的開端

機器學習與深度學習之間的最大區別在於「深度」。傳統機器學習演算法（如決策樹、SVM）通常使用淺層結構，而深度學習透過多層神經網路自動學習層次化的特徵表示。

### 為什麼「深」很重要？

淺層網路只能學習簡單的模式。隨著層數增加，網路能夠學習越來越抽象的特徵：

```
輸入 → [邊緣] → [形狀] → [物件部分] → [完整物件] → 預測
      層1       層2       層3          層4          輸出
```

在影像辨識中：
- 第一層學習邊緣和角落
- 第二層學習紋理和形狀
- 第三層學習物體組件（如眼睛、輪子）
- 更深層學習完整物體（如臉部、汽車）

## 卷積神經網路（CNN）

CNN 是專門為處理網格狀資料（如影像）設計的網路架構：

```
輸入影像   卷積層   池化層   全連接層   輸出
  ┌──┐     ┌──┐    ┌──┐    ┌──┐     ┌──┐
  │  │ →  │  │ →  │  │ →  │  │ →  │  │
  └──┘     └──┘    └──┘    └──┘     └──┘
```

### 關鍵元件

- **卷積層**：使用濾波器（Kernel）掃描影像，提取局部特徵
- **池化層**：降取樣，減少參數數量
- **全連接層**：在特徵提取後進行分類決策

CNN 的權重共享機制大幅減少了參數數量，使深層網路的訓練成為可能。

## 循環神經網路（RNN）

RNN 專為序列資料設計，如文字、語音、時間序列：

```
輸出 ←── 隱藏狀態 ←── 隱藏狀態
         ↑            ↑
         x₁           x₂
```

每個時間步的隱藏狀態不僅取決於當前輸入，還取決於上一步的隱藏狀態，形成記憶效應。

但傳統 RNN 有梯度消失問題——長距離的資訊難以傳遞。這引出了 LSTM 和 GRU 等改進架構。

## Transformer：改變一切

2017 年，Google 發表了《Attention Is All You Need》，提出 Transformer 架構。Transformer 的核心是注意力機制：

```python
def attention(Q, K, V):
    scores = Q @ K.T / sqrt(d_k)
    weights = softmax(scores)
    return weights @ V
```

### Transformer 的優勢

- **並行計算**：不需像 RNN 逐步處理
- **長距離依賴**：直接連接所有位置
- **可擴展性**：更多資料、更大模型帶來更好表現

Transformer 催生了 BERT、GPT 系列等大型語言模型（LLM），徹底改變了 NLP 領域。

## 深度學習的關鍵推動因素

### 資料

大規模標記資料集（ImageNet、Common Crawl）讓深層網路得以充分訓練。

### 運算資源

GPU 的並行計算能力使深層網路的訓練從不可能變為可能。TPU、NVLink 等專用硬體進一步加速。

### 演算法進步

- **ReLU 激活函式**：緩解梯度消失
- **Batch Normalization**：穩定訓練過程
- **Dropout**：防止過擬合
- **Adam 最佳化器**：自適應學習率
- **ResNet 殘差連接**：讓訓練更深網路成為可能

## 現代 AI 的基石

深度學習不僅僅是更「深」的神經網路，它代表了從特徵工程到表示學習的典範轉移。傳統 ML 需要專家手動設計特徵；深度學習讓模型自動學習特徵表示。

從 ML 到深度學習的轉變，本質上是從「人類告訴模型學什麼」到「模型自己發現該學什麼」的躍進。
