# 監督式學習：線性迴歸

## 迴歸問題定義

迴歸是監督式學習中的一類任務，目標是預測一個連續的數值。例如：
- 根據房屋坪數和地理位置預測房價
- 根據廣告支出預測銷售額
- 根據學習時數預測考試成績

最簡單也是最經典的迴歸方法就是線性迴歸。

## 線性迴歸模型

線性迴歸假設目標值與特徵之間存在線性關係。對於單一特徵的情況：

```
y = wx + b
```

其中 `w` 是權重（斜率），`b` 是偏置（截距），`x` 是輸入特徵，`y` 是預測值。

### 視覺化理解

```
y ↑                ·    
  │              ·
  │           ·    ← 預測值 ŷ = wx + b
  │        ·
  │     ·    ← 實際值 y
  │  ·
  ·——————————————→ x
     特徵
```

目標是找到一條直線，使得所有資料點到直線的距離總和最小。

## 最小平方法（Ordinary Least Squares）

最常用的誤差度量是均方誤差（MSE）：

```
MSE = (1/n) Σ(yᵢ - ŷᵢ)²
```

其中 `yᵢ` 是實際值，`ŷᵢ` 是預測值，`n` 是樣本數。最小平方法就是尋找使 MSE 最小的 `w` 和 `b`。

### 最佳解公式

對於單一特徵的線性迴歸，最佳解有封閉形式：

```
w = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²
b = ȳ - w × x̄
```

## 梯度下降

當特徵數量很多時，最小平方法的直接計算可能很耗時。梯度下降是一種迭代最佳化演算法：

1. 隨機初始化 `w` 和 `b`
2. 計算損失函式對參數的梯度
3. 向梯度反方向更新參數
4. 重複直到收斂

```python
import numpy as np

def gradient_descent(X, y, lr=0.01, epochs=1000):
    n = len(X)
    w, b = 0, 0
    for _ in range(epochs):
        y_pred = w * X + b
        dw = (-2/n) * np.sum(X * (y - y_pred))
        db = (-2/n) * np.sum(y - y_pred)
        w -= lr * dw
        b -= lr * db
    return w, b
```

## 多變量線性迴歸

當有多個特徵時，模型擴展為：

```
y = w₁x₁ + w₂x₂ + ... + wₙxₙ + b
```

這可以寫成矩陣形式：`y = Xw + b`

## 線性迴歸的假設

- **線性**：特徵與目標之間存在線性關係
- **獨立性**：觀測值之間相互獨立
- **同方差性**：殘差的變異數恆定
- **常態性**：殘差呈常態分布

違反這些假設時，可能需要使用更複雜的模型或對資料進行轉換。
