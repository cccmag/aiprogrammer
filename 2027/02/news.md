# 2027 年 2 月 — 遊戲開發新聞

## Bevy 2.8 正式發布

2027 年 1 月 15 日，Bevy 2.8 版發布。主要更新包括：

- **GPU 驅動 ECS 排程**：利用 compute shader 在 GPU 上並行執行 System，CPU 開銷降低 40%
- **新的 UI 框架**：bevy_ui 2.0 引入 flexbox 佈局與 CSS-like 樣式系統
- **改進的資產管線**：支援非同步載入與熱重載的資產系統
- **WebGPU 後端穩定**：在 Chrome、Firefox、Safari 上達到原生效能

參考：[Bevy 2.8 release notes](https://www.google.com/search?q=Bevy+2.8+release+notes+2027)

## Steam 上 Rust 遊戲突破 200 款

截至 2027 年 1 月，Steam 平台上有超過 200 款使用 Rust 開發的遊戲，其中 47 款使用 Bevy 引擎。知名作品包括：

- **Tiny Glade**（2024）：使用 Bevy 的程式化建築生成遊戲
- **Veloren**：開源 RPG，使用自家 Rust 引擎
- **Fish Fight**：使用 Bevy 的多人派對遊戲

參考：[Rust game development 2027](https://www.google.com/search?q=Rust+games+on+Steam+2027)

## Rust 基金會宣布遊戲開發工作小組

Rust 基金會與 Bevy 核心團隊共同成立遊戲開發工作小組，目標包括：

- 標準化 ECS 介面
- 改善遊戲相關 crate 的協作
- 提供遊戲開發的官方教學資源

## wgpu 22.0 支援 DirectX 12 光追

wgpu 22.0 加入了對 DirectX 12 光線追蹤的實驗性支援，使 Rust 遊戲能夠使用硬體加速的光追效果。

## Unreal Engine 推出 Rust 綁定

Epic Games 在 UE 5.8 中實驗性地加入了 Rust 綁定，允許使用 Rust 編寫遊戲邏輯並與 UE 的 Blueprint 系統互通。

參考：[Unreal Engine Rust](https://www.google.com/search?q=Unreal+Engine+Rust+binding)
