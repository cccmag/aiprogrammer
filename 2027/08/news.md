# 2027 年 8 月 — 多 Agent 新聞

## AutoGen 1.0 正式發布

Microsoft 的 AutoGen 框架在 2027 年 7 月達到 1.0 穩定版。新增功能：

- **可插拔 Agent 模型**：支援 OpenAI、Claude、Llama、Gemini
- **群組聊天模式（GroupChat）**：多 Agent 即時協作
- **內建記憶系統**：支援向量資料庫和檔案記憶
- **A2A 協議整合**：跨執行期 Agent 通訊

參考：[AutoGen 1.0](https://www.google.com/search?q=AutoGen+1.0+Microsoft+2027)

## A2A 通訊協議成為標準

W3C 在 2027 年 6 月發布了 Agent-to-Agent（A2A）通訊協議的第一個工作草案，定義了：

- 統一的訊息格式
- 服務發現機制
- 能力宣告標準
- 安全性與認證

## Agent 即服務（AaaS）市場興起

多家新創公司推出 Agent 市集，開發者可以發布專業化 Agent（程式碼審查 Agent、資料分析 Agent、客服 Agent），按使用量計費。

## OpenAI 發布 Agent SDK

OpenAI 在 2027 年 7 月發布了官方的 Agent SDK，整合了函數呼叫、記憶管理、多步驟推理，並支援與 AutoGen 和 LangGraph 互通。

## LangGraph 達到 10 萬 GitHub Stars

LangChain 的 Agent 框架 LangGraph 在 2027 年達到 10 萬顆星。其圖形化的 Agent 流程設計成為多 Agent 系統的事實標準之一。

## 安全漏洞：首起跨 Agent 注入攻擊

安全研究人員展示了首起跨 Agent 提示詞注入攻擊：惡意文件內容透過一個 Agent 感染同一個系統中的其他 Agent。這促使了 Agent 隔離標準的討論。
