# 2027 年 7 月 — MLOps 新聞

## LLMOps 工具生態成熟

2027 年上半年，LLMOps 工具市場經歷了爆發式成長。LangSmith 發布 3.0 版本，加入即時提示詞監控與自動回滾功能。Weights & Biases 推出 LLM 專用評估儀表板，支援 LLM-as-Judge 自動評分。業界共識：LLMOps 已從「新興領域」進入「標準實踐」階段。

參考：[LLMOps tools 2027](https://www.google.com/search?q=LLMOps+tools+2027+LangSmith+Weights+Biases)

## Google 開源 Vertex AI Agent Builder

Google 在 2027 年 6 月開源了 Vertex AI Agent Builder 的核心元件，包括 Agent 可觀測性 SDK 與提示詞管理 API。此舉被視為 Google 對抗開源 LLMOps 生態的戰略布局。

參考：[Vertex AI Agent Builder open source](https://www.google.com/search?q=Vertex+AI+Agent+Builder+open+source+2027)

## AI 監控標準 OpenTelemetry 新增 LLM 規範

CNCF 的 OpenTelemetry 專案在 2027 年 5 月正式發佈 LLM 監控規範（Semantic Conventions for LLM Applications），定義了提示詞、生成、RAG 檢索等元件的標準追蹤格式。這標誌著 AI 可觀測性的標準化邁出關鍵一步。

參考：[OpenTelemetry LLM semantic conventions](https://www.google.com/search?q=OpenTelemetry+LLM+semantic+conventions+2027)

## Prompt Registry 成為企業必備基礎設施

根據 Gartner 2027 年度報告，超過 65% 的企業已導入 Prompt Registry 系統管理提示詞資產。開源方案 PromptLayer、Agenta 與商業方案共同推動了市場成長。提示詞的版本控制、A/B 測試和變更審計已成為標準流程。

## MLflow 4.0 加入 LLM 支援

MLflow 在 2027 年 4 月發布 4.0 大版本更新，原生支援 LLM 評估、提示詞追蹤與 RAG 管線記錄。新功能包含 `mlflow.log_prompt()`、`mlflow.evaluate_llm()` 等 API，將傳統 MLOps 平台無縫擴展到 LLMOps。

## Databricks 收購開源向量資料庫專案

Databricks 在 2027 年 6 月收購了開源向量資料庫專案 Chroma，計劃將其深度整合到 Unity Catalog 與 MLflow 生態系統中。此舉強化了 Databricks 在 RAG 基礎設施領域的競爭力。

## NVIDIA 推出 GPU 監控與自動擴展方案

NVIDIA 發布了 AI 基礎設施監控平台 `NVIDIA AIOps`，提供 GPU 使用率、模型推論延遲、記憶體頻寬的即時監控，並支援 Kubernetes 的自動擴展策略。該平台與 Prometheus/Grafana 生態相容。

參考：[NVIDIA AIOps GPU monitoring](https://www.google.com/search?q=NVIDIA+AIOps+GPU+monitoring+2027)

## AWS 推出 SageMaker LLMOps 模組

AWS 在 re:Invent 2027 預告了 SageMaker 的全新 LLMOps 模組，包含提示詞管理、模型評估與 RAG 監控三大功能。該模組與 SageMaker Pipelines 深度整合，提供從提示詞開發到部署的完整工作流程。

參考：[AWS SageMaker LLMOps](https://www.google.com/search?q=AWS+SageMaker+LLMOps+2027)
