# 強化學習基礎概念

強化學習有其獨特的術語和概念。

## 1. 核心術語

**Agent（智能體）**：
- 學習和決策的主體

**Environment（環境）**：
- Agent 互動的外部系統

**State（狀態）**：
- 環境的當前情況描述

**Action（動作）**：
- Agent 可執行的操作

**Reward（獎勵）**：
- 環境給予的反饋信號

## 2. 策略（Policy）

π(a|s) = P(a|s)

在狀態 s 下執行動作 a 的機率。

## 3. 價值函數

**V(s)**：狀態價值
**Q(s,a)**：狀態-動作價值

## 4. 探索與利用

- 探索：嘗試新動作
- 利用：選擇已知最佳動作
