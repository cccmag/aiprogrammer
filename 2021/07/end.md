# 回顧與結語

## 本期回顧

2021 年 7 月號的《AI 程式人雜誌》帶領讀者深入探索了自然語言處理的前沿技術。

### 本期主題：自然語言處理的突破

我們從多個角度分析了 NLP 的最新進展：

**Transformer 架構**：
- 注意力機制替代了傳統 RNN
- 完全并行計算開啟了新可能
- 成為 NLP 和多領域的基礎架構

**預訓練語言模型**：
- BERT 開創了雙向預訓練時代
- GPT 系列展示了生成式 AI 的潛力
- GPT-3 的 Few-shot Learning 能力震驚業界

**基礎技術**：
- Tokenization 從詞粒度到子詞
- 詞向量從靜態到動態上下文相關

### 技術脈絡

從 Word2Vec（2013）到 Transformer（2017），再到 BERT（2018）和 GPT-3（2020），NLP 領域經歷了爆發式增長。2021 年，這些技術已經成熟並開始广泛应用。

### 產業影響

- Google 將 BERT 應用於搜尋引擎
- OpenAI 提供 GPT-3 API
- Hugging Face 估值突破 20 億美元
- 各種 NLP 應用層出不窮
