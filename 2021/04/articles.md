# 文章總覽

本期共十篇技術文章，從實務經驗到深入原理，全面涵蓋 Python 資料科學工具鏈。

## 第一期：從 pandas 轉換到 Polars 的實務經驗

分享將既有 pandas 程式碼遷移到 Polars 的實務心得。討論兩者的語法差異、效能比較、以及如何逐步遷移而不影響業務。提供常見陷阱和對應解決方案。[閱讀全文](article1.md)

## 第二期：NumPy 廣播機制深入解析

剖析 NumPy 廣播的運作原理、解釋何時使用廣播、如何避免意外的記憶體拷貝。透過視覺化方式展示形狀匹配規則，幫助讀者建立正確的直覺。[閱讀全文](article2.md)

## 第三期：Pandas GroupBy 效能優化

深入探討 pandas groupby 的效能瓶頸和優化策略。介紹分組鍵類型選擇、聚合方法比較、以及如何利用多程序加速。涵蓋新舊 API 的效能差異。[閱讀全文](article3.md)

## 第四期：Dask 分散式訓練實戰

展示如何使用 Dask 加速大規模資料處理。從本機多程序到真正分散式叢集，逐步說明部署和調優過程。討論常見的效能瓶頸和監控方法。[閱讀全文](article4.md)

## 第五期：資料處理中的常見陷阱與解決方案

整理 pandas 使用中的常見錯誤： SettingWithCopyWarning、記憶體持續飆升、字串和數字的混淆、日期時區處理等。提供正確的寫法模式和除錯技巧。[閱讀全文](article5.md)

## 第六期：Parquet 格式與資料湖架構

介紹列式儲存格式 Parquet 的優勢，以及如何用 pyarrow 和 pandas 處理 Parquet 檔案。討論資料湖架構的設計原則和 Parquet 在其中的角色。[閱讀全文](article6.md)

## 第七期：PyArrow 與跨語言資料互通

探討 Apache Arrow 如何實現 Python、Ruby、R、JavaScript 等語言間的高效資料交換。介紹 pyarrow 的核心功能，以及如何用於構建高性能資料管線。[閱讀全文](article7.md)

## 第八期：Pandas 與 SQL 的轉換技巧

整理 pandas 操作與 SQL 查詢的對應關係。從簡單的 SELECT、WHERE 到複雜的 JOIN、GROUP BY，提供轉換指南和雙向轉換的實用技巧。[閱讀全文](article8.md)

## 第九期：處理超大型資料集的策略

面對記憶體不足的挑戰，介紹多種策略：探樣計算、記憶體映射、分散式處理、資料分頁等。討論何時用哪種策略，以及如何組合使用。[閱讀全文](article9.md)

## 第十期：2021 年資料科學工具發展趨勢

回顧 2021 年 Python 資料科學生態的重要進展。預測未來發展方向，討論哪些工具值得學習，哪些可能退潮。提供學習路徑建議。[閱讀全文](article10.md)
