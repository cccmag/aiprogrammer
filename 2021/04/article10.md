# Article 10：2021 年資料科學工具發展趨勢

## Rust 的崛起

2021 年見證了 Rust 在資料處理領域的爆發。Polars、Ballista、RisingWave 等專案選擇 Rust，基於其記憶體安全和高效能特性。這一趨勢可能持續，越來越多的核心資料庫將用 Rust 重寫。

## 雲端原生架構

越來越多工具原生支援雲端。分散式處理的門檻持續降低工程師可在不搭建基礎架構的情況下，使用 Dask Cloud、Coiled 等服務快速擴展。

## 即時分析的需求

對即時分析的需求持續成長。串流處理框架如 Flink、Kafka Streams獲得更多關注。傳統的批次處理加即時檢視的 Lambda 架構正在被更簡單的方案取代。

## AutoML 的普及

AutoML 工具如 Auto-sklearn、H2O AutoML、FLAML 更加成熟，降低了機器學習的進用壁壘。沒有機器學習背景的從業人員也能用這些工具构建不错的模型。

## 建議的學習路徑

1. 打好 NumPy 和 pandas 基礎
2. 學習一種深度學習框架（PyTorch 或 TensorFlow）
3. 掌握一種分散式處理工具（Dask 或 Spark）
4. 了解一種雲端機器學習平台
