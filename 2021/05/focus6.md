# Focus 6：殘差連接與參數初始化

## 殘差連接的由來

ResNet（Deep Residual Network）在 2015 年提出時，解決了深度網路訓練的一個根本困難：隨著網路加深，訓練誤差反而上升。這不是過擬合（訓練誤差不變但測試誤差上升），而是「退化」問題。殘差連接的解決方案簡單優雅：讓網路學習殘差而非完整映射。

## 殘差學習的數學

若希望網路學習映射 H(x)，傳統方法讓每層學習 F(x) = H(x)。殘差方法讓每層學習 F(x) = H(x) - x，輸出變為 F(x) + x = H(x)。這使得即使 H(x) 難以學習，F(x) 更容易被逼為零，使網路至少能實現恒等映射。

## Xavier/He 初始化

初始化對深度網路訓練至關重要。Xavier 初始化適用於線性和 Sigmoid/Tanh 激活：W ~ N(0, 2/(fan_in + fan_out))。He 初始化適用於 ReLU 激活：W ~ N(0, 2/fan_in)。錯誤的初始化會導致梯度消失或爆炸。

## 殘差連接的理論意義

殘差連接不僅是工程技巧，更有深刻理論意義。它相當於為網路添加了「捷徑」，使得梯度可以繞過中間層直接傳播。這解釋了為何殘差網路可以訓練這麼深。也說明深層網路的訓練困難主要來自短路徑的破壞，而非簡單的梯度消失。

## 實務建議

1. 幾乎所有現代架構都使用殘差連接
2. 殘差分支通常包含多層，全連接或卷積
3. 確保殘差分支和捷徑的維度匹配，或使用線性投影
4. 預訓練/微調場景，殘差連接使模型更具彈性
