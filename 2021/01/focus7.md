# 未來展望

## GPT-3 之後的發展方向

### 時間線回顧

```
2020/05 - GPT-3 發表（175B 參數）
2020/06 - API 開放 Beta 測試
2021/01 - API 存取持續擴大
2022/03 - GPT-3.5 / ChatGPT
2023/03 - GPT-4 發表
```

### 即將到來的發展

#### GPT-3.5 與後續模型

OpenAI 在 GPT-3 之後，持續改進：
- **更好的遵循指令**：InstructGPT
- **更長的上下文**：從 2048 擴展到更多
- **程式碼能力**：Codex 整合

#### 多模態擴展

未來的大型模型將整合：
- **文字**：現在已經成熟
- **圖像**：DALL-E、CLIP
- **音頻**：Whisper
- **影片**：可能的發展方向

### AGI 的距離

大型語言模型展示了一些通用智慧的跡象：
- 跨任務遷移能力
- Few-shot 學習
- 某種程度的推理

但仍缺少：
- 真正的理解
- 持續學習能力
- 物理世界的互動

### 開放議題

1. **規模瓶頸**：是否繼續擴大就能更聰明？
2. **能耗問題**：永續發展的 AI
3. **監管需求**：如何確保安全
4. **存取公平性**：避免少數公司壟斷

### 對開發者的建議

- 掌握 Prompt Engineering 技能
- 關注模型能力與限制
- 設計人類在迴路中的系統
- 關注 AI 倫理和責任
