# 本月新知

## 2021 年 1 月程式與 AI 技術動態

### 程式語言與框架

**Python 3.9 穩定運行**

Python 3.9 於 2020 年 10 月發布後，在 2021 年 1 月已成為主流選擇。新版本帶來多項實用功能：字典合併運算子（|`）、類型提示改進、以及更清晰的語法。許多大型專案已開始遷移到 3.9 系列。

**Rust 2021 發展方向確定**

Rust 語言團隊公佈 2021 年的發展藍圖，重點包括：更快的編譯速度、改進的錯誤訊息、以及 async/await 語法的穩定性提升。Rust 連續五年被評選為最受歡迎的程式語言。

### AI 與機器學習

**OpenAI 擴大 GPT-3 API 存取**

2021 年 1 月，OpenAI 宣佈進一步擴大 GPT-3 API 的存取範圍。GPT-3 自 2020 年 6 月開放 Beta 測試以來，已吸引數千名開發者申請。API 提供多種模型規格，從 350M 到 175B 參數，讓開發者能根據任務需求選擇合適的模型。

**Google 發表 Switch Transformer**

Google Research 在 2021 年 1 月發表 Switch Transformer 模型，這是一種新型態的專家混合模型（Mixture of Experts）。該模型採用「稀疏激活」技術，在保持計算效率的同時，大幅增加模型參數量，達到 1.6 兆參數的規模。

**DeepMind 發表 Perceiver IO**

DeepMind 發表 Perceiver IO，這是一種能處理多種資料類型（文字、圖像、音頻）的通用架構。該模型延續了 Perceiver 的設計理念，使用潛在空間進行處理，大幅降低計算複雜度。

**Meta 開源 PyTorch Geometric**

Meta 持續擴充 PyTorch 生態系，於本月開源 PyTorch Geometric 1.7 版本，新增對圖神經網路的強化支援。該庫已成為圖機器學習領域的標準工具。

### 開發工具

**VS Code 1.53 發布**

Visual Studio Code 發布 1.53 版本，主要改進包括：更快的啟動時間、更好的終端支援、以及 Jupyter Notebook 的整合增強。GitHub Copilot 擴展的預覽版也持續優化。

**Docker 支援 Apple M1**

Docker 正式宣佈支援 Apple M1 晶片，開發者可以在新世代 Mac 上原生運行容器化應用。這解決了長期以來ARM架構相容性的問題。

### 標準與規範

- **WebAssembly 支援 Rust**：WASM 生態系持續成熟，Rust 成為首選的 WASM 編譯目標
- **TypeScript 4.2 開發中**：預計帶來更智慧的類型推導
- **WebGPU 標準進展**：新一代 Web 圖形 API 持續制定中

### 業界動態

- **Intel 新任 CEO 上任**：Pat Gelsinger 接掌 Intel，強調重新奪取製程領先
- **Sony 與微軟合作雲遊戲**：PlayStation 與 Xbox 在雲端領域展開合作
- **比特幣突破 3 萬美元**：加密貨幣熱潮持續，相關技術備受關注
