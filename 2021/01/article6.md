# GPT 系列模型演進

## 模型比較

| 版本 | 參數量 | 發布時間 | 關鍵創新 |
|------|--------|----------|----------|
| GPT-1 | 110M | 2018/06 | 預訓練 + 微調 |
| GPT-2 | 1.5B | 2019/02 | 無監督多任務 |
| GPT-3 | 175B | 2020/05 | Few-shot learning |

## GPT-1：開創性工作

- 首創「預訓練 + 微調」範式
- 使用 Book Corpus 資料集
- 為後續版本奠定基礎

## GPT-2：規模與通用性

- 強調「通用語言模型」
- 可處理多種任務而無需微調
- 因安全疑慮延後發布

## GPT-3：規模的突破

- 1750 億參數
- 強大的 Few-shot 能力
- 開放 API 商業化

## 技術演進

```
GPT-1 → GPT-2 → GPT-3
        ↓
       規模
        ↓
       100倍
```
