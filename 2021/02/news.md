# 本月新知

## 2021 年 2 月程式與 AI 技術動態

### 程式語言與框架

**Python 3.9 生態系成熟**

Python 3.9 發布後，相關生態系持續完善。Jupyter Notebook、Lconda 等工具已全面支援新版本特性。

**Rust 語言持續成長**

Rust 在 2021 年初持續受到關注，Linux 核心開始逐步整合 Rust 程式碼，開創了系統程式設計的新局面。

### AI 與機器學習

**PyTorch 1.8 發布**

PyTorch 發布 1.8 版本（實際為 2021 年 3 月正式發布，2 月是 RC 版本），帶來多項重要更新：
- 分解的 `torch.linalg` 模組
- 更好的分散式訓練支援
- 行動裝置推理效能提升

**Hugging Face Transformers 4.4 發布**

Hugging Face 發布 Transformers 4.4 版本，新增多語言模型的支援，並最佳化了推理速度。

**TensorFlow Lite 支援 GPU 加速**

Google 宣佈 TensorFlow Lite 現在支援 GPU 加速，使得在行動裝置上的推論更加高效。

### 開發工具

**VS Code 1.54 發布**

Visual Studio Code 發布新版本，強化了 Python 擴充功能，包括更好的類型檢查和偵錯支援。

**Docker 20.10 發布**

Docker 更新到 20.10 版本，強化了網路功能和安全性，特別是對 Apple M1 的更好支援。

### 標準與規範

- **Python 3.10 開發中**：新版本將引入 pattern matching 語法
- **WebAssembly 更新**：提供更多 Web API 存取能力
- **TypeScript 4.3 Beta**：改進型別推導

### 業界動態

- **Intel 任命新 CEO**：提出 IDM 2.0 策略，強調重新奪取製程領先
- **NVIDIA 發布 RTX 3060**：供應依舊緊張，顯示卡短缺持續
- **亞馬遜收購 Zoox**：布局自動駕駛技術
