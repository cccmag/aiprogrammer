# 回顧與結語

## 2021 年 3 月號結語

### 本月主題回顧

本期我們聚焦在「GPU 運算與 CUDA 新發展」，涵蓋了從硬體架構到軟體工具的完整 GPU 運算生態系。

#### 焦點回顧

**NVIDIA Ampere 架構**展示了我們已經進入了一個新的 GPU 效能時代。A100 的 542 億晶體管和 1.6 TB/s 的記憶體頻寬，支撐著越來越大的深度學習模型。

**CUDA 程式模型**讓我們理解 GPU 平行運算的核心概念：執行緒、Warp、共享記憶體和全域記憶體的層級結構。

**ROCm 與 AMD GPU** 的發展代表了一個重要的趨勢：開放運算平台正在挑戰 NVIDIA 的 CUDA 生態壟斷。

#### 文章亮點

本期的 10 篇文章從 CUDA 基礎到邊緣部署，提供了 GPU 運算開發的完整技能地圖：
- 記憶體配置 → 核函數 → 共享記憶體 → Warp → Streams
- 硬體架構 → TensorCore → 分散式訓練 → 邊緣部署

### 程式碼實作

本期附帶的 CUDA 範例（約 150 行 C++）展示了：
- 基本的記憶體管理
- 核函數設計
- 共享記憶體優化

### 未來展望

GPU 運算在 2021 年正處於快速發展階段：

- **更大的頻寬**：HBM3、GDDR6X
- **更多的核心**：更精細的平行化
- **更專用的硬體**：Transformer Engine

同時也面臨挑戰：
- 晶片短缺持續
- 功耗牆限制
- 軟體生態的持續競爭
