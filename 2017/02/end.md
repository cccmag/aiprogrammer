# 回顧與結語

## 本期總結

2017 年 2 月號的「AI 程式人雜誌」聚焦於 TensorFlow 1.0 與深度學習框架。這個議題在 2017 年顯得格外重要——TensorFlow 1.0 的正式發布標誌著深度學習框架進入成熟階段。

### 主要內容回顧

1. **TensorFlow 1.0 正式發布**
   - API 穩定性承諾
   - XLA 編譯器帶來效能提升
   - Keras 整合進入 TensorFlow

2. **計算圖與 Session**
   - 理解 TensorFlow 的核心概念
   - 宣告式計算模型
   - Session 執行機制

3. **Keras 與高層 API**
   - 降低深度學習門檻
   - Sequential 和 Functional API
   - tf.keras 的官方支援

4. **PyTorch 的崛起**
   - 動態計算圖的優勢
   - 與 TensorFlow 的比較
   - 研究領域的人氣

5. **深度學習應用**
   - 電腦視覺：物體偵測、語義分割
   - 語音辨識：CTC 網路、端到端模型
   - 自然語言處理：Word2Vec、Seq2Seq、Attention

## 下期預告

在深度學習大放異彩的背後，傳統機器學習仍然是 AI 的重要基礎。本期將深入探討：

- scikit-learn 的使用
- 機器學習的核心概念
- 訓練/測試/驗證流程
- 特徵工程基礎
- 常見演算法詳解
