# 本月新知

## 2017 年 2 月程式與 AI 技術動態

### 重大事件：TensorFlow 1.0 正式發布

**2017 年 2 月 15 日，TensorFlow 1.0 正式發布！**

這是深度學習領域的重大里程碑。TensorFlow 1.0 帶來了：

- **Python API 穩定性承諾**：1.0 的 Python API 將保持穩定，不會有破壞性變更
- **，改進效能**：通過 XLA 編譯器和核心優化，效能提升約 2 倍
- **Keras 整合**：從 TensorFlow 1.1 開始，Keras 將整合進 TensorFlow
- **Java 和 Go API**：擴展了多語言支援
- **tf.layers、tf.metrics、tf.losses**：新的高層 API 模組

TensorFlow 從 2015 年開源以來，已成為最受欢迎的深度學習框架。1.0 版本標誌著其邁向 production-ready 的重要一步。

### 程式語言與框架

**Keras 2.0 開發中**

Francois Chollet 宣佈 Keras 2.0 正在開發中，新版本將帶來：
- 與 TensorFlow 更好地整合
- 更清晰的 API 設計
- tf.keras 將成為官方推薦的高層 API

**PyTorch 0.2 發布**

PyTorch 在 2017 年 1 月發布了 0.2 版本，主要改進包括：
- nn.DataParallel 的改進
- 更豐富的 torch.optim 功能
- 更好的 CUDA 記憶體管理

### AI 與機器學習

**AlphaGo 對決柯洁：世紀大戰確定**

AlphaGo 與世界排名第一的柯洁九段的對決確認於 2017 年 5 月在浙江烏鎮舉行。這將是 2016 年 AlphaGo 戰勝李世乭後最受關注的對局。

**DeepMind 發布星際爭霸 II 研究環境**

DeepMind 宣佈將與 Blizzard 合作發布星際爭霸 II 的 AI 研究環境，這為強化學習研究提供了新的測試平台。

**GAN 的最新進展**

Ian Goodfellow 等人發布了 GAN 的新訓練技巧，包括：
- 技術的實際應用案例
- 訓練穩定性的改進方法

### 業界動態

- **Google 收購 ML 平台**：強化其雲端 AI 服務
- **Amazon 發布 SageMaker**：機器學習平台即服務
- **Microsoft 認知服務更新**：新增多個 AI API
- **NVIDIA 發布 Volta 架構傳聞**：為深度學習優化的新一代 GPU

### 標準與規範

- **ONNX 預覽版發布**：Facebook 和 Microsoft 合作開發的標準模型格式
- **TensorFlow 1.0 API 穩定**：確定了官方支援的 API 列表
