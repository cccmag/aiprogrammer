# 本期焦點：電腦視覺與物體偵測

## 概述

電腦視覺（Computer Vision）讓電腦能夠理解和處理視覺資訊。從影 片、影像中提取有意義的資訊，是人工智慧的重要應用領域。

## 歷史背景

- 1966 年：MIT 啟動「夏季視覺計畫」，嘗試建立視覺系統
- 1990 年代：基於幾何的視覺方法興起
- 2000 年代：機器學習方法廣泛應用
- 2012 年：AlexNet 開啟深度學習視覺時代
- 2015-2017 年：物體偵測快速发展，RCNN、YOLO、SSD 相繼出現

## 核心技術流程

```
輸入圖像
    ↓
前處理 → 調整大小、歸一化、資料增強
    ↓
特徵提取 → CNN / 傳統特徵（SIFT、HOG）
    ↓
物體偵測 → 分類 + 定位
    ↓
後處理 → NMS、篩選
    ↓
輸出結果
```
