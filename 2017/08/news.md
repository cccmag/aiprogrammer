# 2017 年 8 月號新聞摘要

## 電腦視覺發展

- Facebook AI 發布 Detectron 系統，基於 Caffe2 實現 Faster R-CNN 與 Mask R-CNN，在 COCO 資料集上達到業界領先水準。

- Google Cloud Vision API 增加新的標籤偵測模型，準確率提升 15%。同時發布 AutoML Vision 預覽版，讓無機器學習背景的開發者也能訓練自訂視覺模型。

- Apple 發布 Core ML，為 iOS 裝置上的機器學習提供硬體加速支援。 Vision 框架增強人臉偵測與追蹤功能。

## 深度學習框架

- TensorFlow 1.3 發布，新增 Keras 整合與資料集 API。TensorBoard 可視化功能大幅改進。

- PyTorch 0.2.0 發布，支援 Python 3.6 與高階 Gradient 計算。分散式訓練支援改善。

- Caffe2 正式發布，強調行動裝置與嵌入式系統的部署效率。

## 物體偵測進展

- YOLO (You Only Look Once) 在 2016 年提出後持續更新 v2 版本，在 Pascal VOC 與 COCO 資料集上表現優異，實現即時偵測。

- Faster R-CNN (2015) 已成為目標偵測的標準方法，ResNet 骨幹網路進一步提升準確率。

- Single Shot MultiBox Detector (SSD) 與 YOLO 競爭，提供另一種快速的物體偵測選擇。

## 自駕車視覺

- Waymo 發布自驾車感測技術論文，介紹結合雷射雷達與相機的感知系統。

- NVIDIA 發布 DRIVE PX 平台，用於自駕車的視覺處理與深度學習推論。

## 開源工具

- OpenCV 3.3 發布，正式支援深度神經網路模組（DNN），可在 CPU 上執行常見的 CNN 模型。

- Darknet 開源專案持續更新 YOLO，社群活躍。

## 學術界動態

- CVPR 2017（電腦視覺與模式識別會議）在夏威夷舉行，發表多篇物體偵測與語意分割論文。GAN 與風格遷移也是熱門話題。

- ICCV 2017 接收論文涵蓋深度學習在視覺的各類應用。
