# 2017 年 8 月號結語

## 本月回顧

2017 年 8 月號的《AI 程式人雜誌》聚焦於**電腦視覺與物體偵測**領域。這是深度學習應用最成功的方向之一。

### 重點回顧

1. **電腦視覺基礎** - 從傳統圖像處理到深度學習的完整技術棧
2. **CNN 卷積神經網路** - 視覺深度學習的核心架構
3. **物體偵測** - R-CNN、Fast R-CNN、Faster R-CNN 演變
4. **YOLO** - 一階段偵測，實現即時處理的突破
5. **語義分割** - U-Net、FCN、Mask R-CNN 等分割技術

## 技術發展觀察

### 2017 年的電腦視覺格局

這一年：
- Faster R-CNN 已經成熟
- YOLO v2 剛發布不久
- Mask R-CNN 即將發表
- OpenCV 3.3 加入 DNN 模組

### 框架生態

- **TensorFlow**：視覺模型丰富，生態完整
- **Caffe/Caffe2**：研究領域流行
- **Darknet**：YOLO 專用，輕量高效
- **OpenCV**：傳統視覺 + 深度學習部署

## 學習建議

### 入門者

從 CNN 基礎開始，理解卷積、池化、全連接的原理。使用 Keras 實作簡單的圖像分類。

### 进阶者

深入 Faster R-CNN 與 YOLO 的架構原理，嘗試使用預訓練模型進行偵測任務。

### 實務者

掌握 OpenCV 與深度學習框架的結合，了解模型部署的最佳實踐。
