# 本期焦點：強化學習基礎

## 概述

強化學習（Reinforcement Learning, RL）是機器學習的三大分支之一，與監督學習、非監督學習並列。不同於監督學習從標註資料學習，強化學習透過與環境互動、從獎懲中學習最佳決策。

## 什麼是強化學習？

强化學習的核心概念是**代理（Agent）在環境中透過試誤學習最佳策略**。

```
代理人 → 動作 → 環境
    ↑              ↓
    ← 觀察 + 獎賞 ←
```

## 與其他學習方式的區別

| | 監督學習 | 非監督學習 | 強化學習 |
|---|---------|-----------|---------|
| 學習方式 | 從標籤學習 | 發現模式 | 試誤學習 |
| 資料來源 | 獨立資料 | 獨立資料 | 互動產生 |
| 目標 | 預測 | 聚類/降維 | 最大化長期回報 |
| 應用 | 分類/迴歸 | 異常檢測 | 決策控制 |

## 歷史背景

- **1950s**：Bellman 提出動態規劃
- **1980s**：Q-learning 演算法出現
- **1992**：TD-Gammon 透過強化學習達到西洋棋大師水準
- **2013-2015**：DQN 開創深度強化學習
- **2016**：AlphaGo 擊敗李世石
- **2017**：AlphaGo Zero 從零自學

## 核心問題

强化學習要解決的核心問題：
- **Exploration（探索）**：嘗試新動作
- **Exploitation（利用）**：使用已知最佳策略

這是 RL 的根本權衡（Exploration-Exploitation Trade-off）。
