# 本月新知

## 2017 年 11 月程式與 AI 技術動態

### 程式語言與框架

**Python 3.6 正式發布**

Python 3.6 於 2016 年 12 月發布，在 2017 年初已獲得廣泛採用。這個版本帶來了多項重要新特性：

- **f-strings**：更簡潔的字串格式化語法
- **安全增強**：新型態提示系统和改進的隱私
- **模組改進**：asyncio 的諸多改進

```python
# f-strings 示例
name = "Alice"
age = 30
print(f"Name: {name}, Age: {age}")
```

**Node.js 8 LTS**

Node.js 8 於 2017 年 5 月發布，11 月進入 LTS（長期支援）狀態。主要改進包括：

- **HTTP/2 原生支援**：提升 Web 效能
- **N-API**：稳定的 Node 原生介面
- **npm 5**：更快的套件安裝

**Rust 1.20 穩定版**

Rust 1.20 於 2017 年 8 月發布，引入了 Associated Constants 和其他改進。Rust 的記憶體安全特性使其成為系統程式設計的熱門選擇。

**Go 1.9 發布**

Go 1.9 於 2017 年 8 月發布，主要特性包括：

- **Type Aliases**：更靈活的類型定義
- **Concurrent Map**：併發安全的 map 類型
- **testing 包的改進**：更方便的單元測試

### AI 與機器學習

**深度學習框架生態成熟**

2017 年下半年，深度學習框架生態持續成熟：

- **TensorFlow 1.4**：加入 Eager Execution 雛形
- **PyTorch 0.3**：更好的效能和更多功能
- **Caffe2**：Facebook 開源的跨平台框架

**ResNet 的影響持續擴大**

2015 年發表的 ResNet 架構已成為電腦視覺的標準骨幹。其殘差連接設計被廣泛採用，影響了後續許多模型架構。

**Batch Normalization 普及**

Batch Normalization 已成為訓練深度神經網路的標準技術。2017 年，大多數新的深度學習架構都默認使用 BatchNorm。

**優化器研究持續**

SGD、Adam、RMSprop 等優化器的比較和選擇仍是研究熱點。2017 年出現了更多關於學習率調整和優化策略的研究。

### 開發工具與雲端服務

**Google Cloud ML Engine**

Google 持續強化其雲端機器學習平台，提供更便捷的模型訓練和部署服務。

**AWS SageMaker**

Amazon SageMaker 於 2017 年中發布，簡化了機器學習工作流程的各個環節。

**Microsoft Azure AI**

Azure 提供涵蓋認知服務、機器學習服務的完整 AI 雲端解決方案。

- **IBM 發布 Power9**：針對 AI 工作負載優化的處理器

### 標準與規範

- **ONNX 0.1**：Facebook 和 Microsoft 共同發布模型交換格式
