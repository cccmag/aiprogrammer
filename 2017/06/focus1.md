# 焦點文章 1：RNN 簡介：處理序列資料

## 前言

循環神經網路（RNN）是一類專門用於處理序列資料的神經網路。與傳統的前饋神經網路不同，RNN 具有環路結構，能夠保持內部狀態，記住之前的輸入資訊。

## 序列資料的特點

序列資料的關鍵特點是：順序至關重要。

- **時間序列**：股票價格、氣溫變化
- **文字**：句子、對話、程式碼
- **音頻**：語音、音樂
- **影片**：連續的影像幀

## RNN 的基本結構

RNN 的核心是一個循環單元：

```
時刻 t:
  輸入: x_t
  隱藏狀態: h_t
  輸出: o_t

公式:
  h_t = f(W × x_t + U × h_{t-1} + b)
  o_t = g(V × h_t + c)
```

## 時間步展開

RNN 在每個時間步展開後相當於一個共享權重的前饋網路：

```
x_0 → [RNN] → h_0 → [RNN] → h_1 → [RNN] → h_2 → ...
       ↓         ↓         ↓
       o_0       o_1       o_2
```

權重 W、U、V 在所有時間步之間共享，這是 RNN 的核心特性。

## RNN 的類型

### 1. 一對一（One-to-One）
- 固定輸入，固定輸出
- 應用：影像分類

### 2. 一對多（One-to-Many）
- 固定輸入，序列輸出
- 應用：影像標題生成

### 3. 多對一（Many-to-One）
- 序列輸入，固定輸出
- 應用：情感分類

### 4. 多對多（Many-to-Many）
- 序列輸入，序列輸出
- 應用：機器翻譯、語音辨識

## RNN 的應用領域

| 領域 | 應用 | 輸入 | 輸出 |
|------|------|------|------|
| NLP | 情感分析 | 文字 | 情感類別 |
| NLP | 語言模型 | 詞序列 | 下一詞預測 |
| NLP | 機器翻譯 | 源語言語句 | 目標語言語句 |
| 語音 | 語音辨識 | 音頻 | 文字 |
| 語音 | 語音合成 | 文字 | 音頻 |
| 時間序列 | 預測 | 歷史資料 | 未來預測 |

## 為什麼 RNN 重要

1. **處理變長序列**：能處理任意長度的輸入
2. **記憶能力**：隱藏狀態編碼了之前的資訊
3. **參數共享**：權重在時間步之間共享
4. **可處理多種類型的序列資料**

## 總結

RNN 是處理序列資料的強大工具。其循環結構與權重共享機制使其能有效處理時間序列、文字、音頻等序列資料。
