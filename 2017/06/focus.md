# 本期焦點：循環神經網路 (RNN/LSTM)

## 概述

循環神經網路（Recurrent Neural Network, RNN）是專門處理序列資料的神經網路架構。與前饋神經網路不同，RNN 具有「記憶」能力，能夠處理任意長度的序列輸入。

## 為什麼需要 RNN

傳統神經網路無法處理序列關係：

- 影像分類：每張圖独立
- 序列問題：上下文至關重要

序列資料示例：
- 文字：「今天 天氣 很好」
- 語音：「你好」
- 股票：「100, 101, 99, 102」

## RNN 的核心思想

「循環」表示網路在時間步之間有連接：

```
h_t = f(W × x_t + U × h_{t-1} + b)
```

每個時間步的隱藏狀態 h_t 取決於當前輸入與上一個隱藏狀態。
