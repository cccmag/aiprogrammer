### 深度學習發展歷程

```
2012: AlexNet - 深度學習復興
2013: OverFeat, ZFNet - 改進 CNN
2014: VGGNet, Inception - 更深更寬的網路
2015: ResNet - 殘差連接
2016: DenseNet, Faster R-CNN - 密集連接, 區域提議網路
2017: Xception, YOLO9000, Transformer, AlphaGo Zero
```

### 框架生態

|------|--------|------|

| MXNet | Amazon | AWS 默認, 多語言支援 |
| CNTK | Microsoft | 企业级, 高效 |

## 為什麼 2017 年重要？

1. **AI 公眾意識覺醒**：AlphaGo 比賽讓大眾了解 AI 能力
2. **框架成熟**：從實驗到生產的轉變
3. **硬體突破**：專用 AI 晶片開始普及
4. **應用落地**：AI 開始進入日常生活
