# 本月新知：2017 年 AI 技術重大事件

## 2017 年 AI 領域年度總結

2017 年是人工智慧領域爆發性成長的一年。從 AlphaGo Zero 的歷史性突破到深度學習框架的成熟，從硬體創新到應用普及，AI 正在以前所未有的速度改變我們的世界。

### 年度重大事件回顧

**一月：PyTorch 正式發布**

Facebook AI Research 於 2017 年 1 月發布 PyTorch，這個基於動態計算圖的深度學習框架迅速獲得研究社群青睐。PyTorch 的 Python 優先設計和除錯便利性，使其成為深度學習研究的首選工具之一。

**二月：TensorFlow 1.0 發布**

Google 於 2017 年 2 月發布 TensorFlow 1.0，標誌著這個世界上最流行的深度學習框架達到重要里程碑。1.0 版本帶來了穩定的 API 和更好的效能。

**五月：AlphaGo 擊敗柯潔**

2017 年 5 月，AlphaGo 在中國烏鎮舉辦的「中國圍棋天才 VS AlphaGo」比賽中以 3:0 擊敗世界排名第一的棋士柯潔。這場比賽吸引了全球關注，展示了深度學習在複雜策略遊戲中的強大能力。

**五月：NVIDIA Volta 架構發布**

NVIDIA 於 2017 年 5 月的 GTC 大会上發布了基於 Volta 架構的 Tesla V100 加速器。這款專為深度學習優化的 GPU 配備 640 個 Tensor Core，提供高達 120 TFLOPS 的深度學習效能。

**六月：Apple 發布 Core ML**

Apple 在 WWDC 2017 發布了 Core ML，這是 iOS 11 的機器學習框架。Core ML 讓開發者能夠輕鬆地將機器學習模型整合到 iOS 應用中，實現影像識別、自然語言處理等功能。

**七月：Caffe2 開源**

Facebook 於 2017 年 4 月將 Caffe2 開源，並在接下來的幾個月持續改進這個輕量級深度學習框架。Caffe2 強調跨平台部署能力，支援從雲端到邊緣的各種場景。

**十月：AlphaGo Zero 震驚世界**

2017 年 10 月，DeepMind 發布了 AlphaGo Zero，這個版本從零開始學習，只透過自我對弈，不需要任何人類棋譜。在 40 天的訓練後，AlphaGo Zero 超越了之前所有 AlphaGo 版本的能力。

**十月：Transformer 論文發表**

Google Brain 在 NeurIPS 2017 發表了《Attention Is All You Need》論文，提出了 Transformer 架構。這種完全基於注意力機制的模型，日後成為大型語言模型的基礎。

**十一月：DeepMind 發布 WaveNet**

DeepMind 發布了 WaveNet，這是一種能夠生成極為逼真人類語音的深度神經網路。WaveNet 採用 Dilated Causal Convolutions，展示了深度學習在音訊生成領域的潛力。
