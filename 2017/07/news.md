# 2017 年 7 月號新聞摘要

## 自然語言處理發展

- Google 在 2017 年發布 Transformer 架構論文「Attention Is All You Need」，提出基於注意力機制的翻譯模型，為日後 BERT 等模型奠定基礎。

- TensorFlow 在 2017 年 8 月發布 1.3 版本，LSTM 等 NLP 常用元件效能持續優化。Keras 整合度提升，文字處理範例更加豐富。

- Facebook AI Research 發布 fastText 庫，使用層級 Softmax 與字元級 n-gram，大幅加速文字分類任務，在標準資料集上達到業界領先水準。

## 深度學習框架

- PyTorch 0.2.0 在 2017 年 7 月發布，支援 RNN/LSTM 的更高效實現，反向傳播速度提升約 30%。動態計算圖特性受到研究者青睞。

- Microsoft 發布 CNTK 2.0，增加 Python 3.6 支援與更好的 LSTM 实现。Azure 機器學習服務整合度提升。

## 開源專案

- spaCy 2.0 alpha 版發布，提供更快的斷詞與實體識別效能。支援多語言的命名實體識別。

- NLTK 3.2.5 穩定版發布，修復若干問題並更新若干語料庫。

- Stanford CoreNLP 3.8.0 發布，改進依存關係解析與情感分析模組。

## 雲端服務

- AWS Comprehend 在 2017 年上線，提供文字情感的持續訓練模型，支援多語言。企業無需自行訓練即可做文字分析。

- Google Cloud Natural Language API 增加語法解析與情感分析功能。

## 學術界動態

- ACL 2017（計算語言學年會）在溫哥華舉行，發表多篇 Word2Vec 改進與應用論文。Embedded Representations 仍然是熱門話題。

- ICML 2017 在雪梨舉行，生成對抗網路（GAN）與序列生成模型是討論焦點。
