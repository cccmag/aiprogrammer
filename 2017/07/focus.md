# 本期焦點：自然語言處理初探

## 概述

自然語言處理（Natural Language Processing, NLP）是人工智慧與語言學的交叉領域，目標是讓電腦夠理解、解釋與生成人類語言。2017 年的 NLP 領域正經歷深度學習帶來的巨大轉變，從傳統計量方法轉向神經網路模型。

## 歷史背景

NLP 的發展可追溯至 1950 年代，當時學者嘗試使用規則系統處理機器翻譯。1990 年代，統計式方法興起，語言模型開始使用機率模型。2013 年，Word2Vec 的發表開創了詞嵌入時代，讓電腦能以稠密向量表示語義。

## 核心技術地圖

```
文字輸入
    ↓
文字預處理 → 斷詞 → 詞性標注 → 命名實體識別
    ↓
詞彙表建立
    ↓
詞嵌入（Word2Vec / GloVe）
    ↓
特徵表示
    ↓
模型訓練 → 分類 / 序列標注 / 生成
```
