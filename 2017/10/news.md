# 本月新知

## 2017 年 10 月程式與 AI 技術動態

### 程式語言與框架

**Python 生態持續成長**

Python 語言在 2017 年持續快速發展。根據 Stack Overflow 的調查，Python 已成為最受人歡迎的程式語言之一。隨著資料科學和機器學習的興起，Python 的生態系統持續壯大，NumPy、Pandas、Scikit-learn 等庫的更新為開發者帶來更多便利。

**TypeScript 2.0 登場**

微軟於 2017 年 9 月發布 TypeScript 2.0，這是近年來最重要的版本更新之一。新版本加入了非空類型（non-nullable types）、更智慧的類型推斷，以及更好的錯誤檢測能力。開發者現在可以在編譯時避免更多的 null 引用錯誤，大幅提升程式碼品質。

**WebAssembly 標準化進展**

2017 年 10 月，W3C 正式宣佈 WebAssembly 成為標準規範。WebAssembly 是一種低層級的位元組碼格式，可以在瀏覽器中以接近原生的速度執行代碼。這項技術為 Web 應用帶來了前所未有的效能提升，也为邊緣運算和客戶端 AI 推理開闢了新的可能性。

**Kotlin 1.2 發布**

JetBrains 於 2017 年 11 月發布 Kotlin 1.2，引入了多平台程式設計（Multiplatform Projects）的概念。現在開發者可以用同一套程式碼同時開發 Android、JVM 和 JavaScript 應用。這項創新使得 Kotlin 成为跨平台開發的強力選擇。

### AI 與機器學習

**AlphaGo Zero 橫空出世**

Google DeepMind 於 2017 年 10 月發布了 AlphaGo Zero，這是一個里程碑式的人工智慧系統。不同於之前需要人類棋譜訓練的 AlphaGo 版本，AlphaGo Zero 從零開始，僅透過自我對弈學習，在 40 天內就超越了之前所有 AlphaGo 的版本。這證明了即使沒有人類知識，AI 也能從頭學習並達到超人類水平。

**NVIDIA Volta 架構 V100 加速器**

NVIDIA 於 2017 年 5 月發布了基於 Volta 架構的 Tesla V100 加速器，這是當時世界上最快的 GPU。V100 配備 640 個 Tensor 核心，專為深度學習優化，可提供 120 TFLOPS 的深度學習效能。這款硬體的推出大幅加速了 AI 研究的進展。

**PyTorch 生態蓬勃發展**

Facebook AI Research (FAIR) 於 2017 年 1 月發布 PyTorch 後，這個基於動態計算圖的深度學習框架迅速獲得廣泛採用。PyTorch 的 Python 優先設計和除錯便利性使其成為研究人员的首選工具。到 2017 年 10 月，PyTorch 生態系統已涵蓋視覺、自然語言處理、強化學習等多個領域。

**TensorFlow 1.0 滿週年生態持續壯大**

Google 於 2017 年 2 月發布 TensorFlow 1.0，經過近一年的發展，生態系統已相當成熟。TensorFlow Lite 的發布使得在行動裝置上部署模型成為可能，而 TensorBoard 的視覺化工具也讓模型訓練過程更加透明。TensorFlow 仍是最廣泛使用的深度學習框架之一。

**DeepMind 發布 WaveNet**

DeepMind 於 2017 年發布了 WaveNet，這是一種用於原始音訊生成的深度神經網路。WaveNet 能夠生成極為逼真的人類語音，為文字轉語音（TTS）系統帶來革命性提升。該技術後來被 Google 用於 Google Assistant 的語音合成。

### 開發工具與雲端服務

**Google Cloud Platform 強化 AI 服務**

Google 在 2017 年持續擴展其雲端 AI 服務。Cloud Machine Learning Engine 讓開發者能夠輕鬆訓練和部署機器學習模型，而 Vision API、Speech API 等預訓練服務也持續改進，為企業提供開箱即用的 AI 能力。

**AWS 擴展 AI 服務**

Amazon Web Services 在 2017 年推出了 SageMaker，這是一個端到端的機器學習平台。SageMaker 大幅簡化了模型訓練和部署的流程，讓更多開發者能夠使用機器學習技術。

**Microsoft 推出 Azure AI**

Microsoft 在 2017 年強化了其 Azure 雲端平台上的 AI 服務。Azure Cognitive Services 提供了涵蓋視覺、語音、語言、搜尋等多個類別的 API，使得開發者能夠輕鬆整合 AI 能力到應用程式中。

### 業界動態

- **Google 收購 HTC 手機部門**：強化硬體與 AI 整合能力
- **Apple 發布 Core ML**：在 iOS 11 中加入機器學習框架
- **Intel 發布 Nervana 神經網路處理器**：進軍 AI 硬體市場
- **Amazon 發布 Alexa 將支援多語言語音辨識**

### 標準與規範

- **WebAssembly 成為 W3C 正式標準**：開啟 Web 效能新時代
- **HTTP/2 廣泛採用**：提升 Web 傳輸效率
- **JSON Schema 正式發布**：為 API 設計提供規範
