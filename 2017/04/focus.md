# 本期焦點：類神經網路導論

## 概述

類神經網路（Neural Network）是受人腦神經元結構啟發的計算模型，是深度學習與現代人工智慧的核心技術。本期將帶領讀者從感知器的基礎開始，逐步深入了解多層感知器、激活函數與反向傳播演算法的原理。

## 主題文章

1. **神經網路的起源：感知器** - 了解第一代神經網路模型
2. **多層感知器與深度學習** - 從單層到多層的演進
3. **激活函數：神經網路的非線性変換** - Sigmoid、ReLU 等函數詳解
4. **反向傳播演算法** - 深度學習的核心訓練演算法
5. **梯度下降與學習率** - 優化神經網路的關鍵技術
6. **神經網路的訓練技巧** - 正則化、批次正規化等
7. **從零開始建構神經網路** - 實作自己的 MLP

## 感知器模型

```
輸入 → 權重 → 求和 → 激活 → 輸出
```

感知器是最簡單的神經網路單元，於 1957 年由 Frank Rosenblatt 提出。它可以解決線性可分的分類問題，是理解複雜神經網路的基礎。

## 多層感知器（MLP）

MLP 由輸入層、多個隱藏層和輸出層組成，能夠學習非線性決策邊界。誤差反向傳播演算法使 MLP 能夠有效訓練深層網路。
