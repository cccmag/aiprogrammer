# 焦點文章 5：梯度下降與學習率

## 前言

梯度下降（Gradient Descent）是優化神經網路參數的核心演算法。學習率（Learning Rate）則是影響訓練效果的關鍵超參數。本章節介紹梯度下降的原理與學習率的設定策略。

## 梯度下降原理

梯度下降的目標是最小化損失函數：

```
θ = θ - η ∇L(θ)
```

其中 θ 為參數，η 為學習率，∇L(θ) 為損失函數的梯度。

想像在山坡上找到下山最快的方向，學習率決定每一步邁出的距離。

## 梯度下降類型

### 批量梯度下降（Batch GD）

- 每個迭代使用全部訓練數據計算梯度
- 收斂穩定，但計算成本高
- 不適合大數據集

### 隨機梯度下降（SGD）

- 每個迭代只用一個樣本計算梯度
- 計算快，添加噪聲有正则化效果
- 收斂振盪，可能跳過最優解

### 小批量梯度下降（Mini-batch GD）

- 每個迭代使用一小批樣本（通常 32-256 個）
- 平衡計算效率與收斂穩定性
- 深度學習的主流選擇

## 學習率設定

學習率是最重要的超參數之一：

| 學習率 | 效果 |
|--------|------|
| 過大 | 振盪發散，無法收斂 |
| 過小 | 收斂緩慢，容易卡在局部最優 |
| 適中 | 穩定收斂 |

## 學習率衰減策略

### 固定衰減

```
η_t = η_0 / (1 + t)
```

### 指數衰減

```
η_t = η_0 × γ^t，γ 通常為 0.95
```

### 台階衰減

```
η_t = η_0 × γ^(floor(t / p))
```

每 p 個 epoch 學習率乘以 γ。

## 動量（Momentum）

動量累積過去的梯度方向，加速收斂並減少振盪：

```
v_t = βv_{t-1} + (1-β)∇L(θ)
θ = θ - ηv_t
```

通常 β = 0.9。

## 自適應學習率方法

### AdaGrad

對每個參數使用不同的學習率：

```
θ_i = θ_i - η / (√(G_ii + ε)) × ∇L/∂θ_i
```

其中 G_ii 為歷史梯度的累計。

### RMSprop

使用指數移動平均計算適應性學習率：

```
E[g²]_t = βE[g²]_{t-1} + (1-β)g²_t
θ = θ - η / √(E[g²]_t + ε) × g_t
```

### Adam

結合動量與 RMSprop：

```
m_t = β₁m_{t-1} + (1-β₁)g_t        # 動量估計
v_t = β₂v_{t-1} + (1-β₂)g²_t       # 梯度平方估計
m̂_t = m_t / (1 - β₁^t)             # 偏差修正
v̂_t = v_t / (1 - β₂^t)
θ = θ - η × m̂_t / √(v̂_t + ε)
```

默認參數：β₁=0.9, β₂=0.999, ε=10⁻⁸

## 總結

學習率影響深遠，需要仔細調整。現代訓練通常使用 Adam 或 SGD with Momentum，它們對學習率的初始值較不敏感。
