# 2017 年 4 月號新聞摘要

## 深度學習框架

- Google 發布 TensorFlow 1.2 版，新增 CUDA 8 支援與分布式訓練改進。TensorFlow 已成为深度學習研究的主流框架之一。

- Microsoft 推出 Azure Cognitive Services 新功能，提供預訓練的電腦視覺與自然語言處理模型。

- Amazon SageMaker 持續簡化機器學習模型訓練流程，降低企業導入 AI 的門檻。

## 人工智慧發展

- AlphaGo 在 2017 年初的比赛中连胜 60 场，击败了包括聂卫平在内的多位围棋高手。AlphaGo 展示了深度學習在複雜策略遊戲中的驚人能力。

- Facebook AI 研究院發布 PyTorch 0.2 版本，增強自動微分功能與多 GPU 訓練效率。

- 各大學術會議發表大量深度學習論文，神經網路架構創新持續涌现。

## Python 生態系

- Python 3.6 生态系逐渐成熟，f-string 格式化与类型提示成為新專案標準。

- Jupyter Notebook 在資料科學領域的採用率持續上升，成為機器學習實驗的首選工具。

## 雲端與基礎設施

- NVIDIA 發布 Volta 架構 GPU，為深度學習提供更強大的運算能力。

- 各雲端服務商加強 GPU 運算實例，推動 AI 開發民主化。
