# 焦點文章 2：多層感知器與深度學習

## 前言

多層感知器（Multi-Layer Perceptron, MLP）是將多個感知器組織成層狀結構的網路，能夠解決非線性問題。本章節介紹 MLP 的結構與原理。

## MLP 結構

MLP 由三種層次組成：

- **輸入層**：接收原始特徵輸入
- **隱藏層**：特徵轉換與抽象
- **輸出層**：產生最終預測結果

```
輸入層 → 隱藏層1 → 隱藏層2 → ... → 輸出層
```

## 前向傳播

資料由輸入層向輸出層流動：

```
a⁽⁰⁾ = x                    # 輸入層
z⁽ˡ⁾ = W⁽ˡ⁾a⁽ˡ⁻¹⁾ + b⁽ˡ⁾  # 第 l 層加權和
a⁽ˡ⁾ = σ(z⁽ˡ⁾)             # 第 l 層輸出
```

其中 W 為權重矩陣，b 為偏置向量，σ 為激活函數。

## 通用逼近定理

通用逼近定理（Universal Approximation Theorem）指出：一個具有足夠多神經元的單隱藏層前饋網路，可以逼近任意連續函數。

這定理保證了神經網路的表達能力，但也暗示了深層網路的優勢：深層網路可能用更少的參數達到同樣的表達能力。

## 深層網路的優勢

| 淺層網路 | 深層網路 |
|----------|----------|
| 需要大量神經元 | 層級結構減少參數 |
| 容易過擬合 | 特徵學習更有效 |
| 難以遷移學習 | 可預訓練底層特徵 |

## 深度學習的興起

2012 年 AlexNet 在 ImageNet 比賽中突破性勝出，標志著深度學習時代的來臨。此後：
- 2014: VGGNet、GoogLeNet
- 2015: ResNet 引入殘差連接
- 2016-2017: 各類變體與應用爆發

## 層級特徵學習

深度網路能自動學習多層次特徵：

- **底層**：邊緣、紋理等簡單特徵
- **中層**：形狀、物體部件
- **高層**：完整物體、語義概念

這種層級結構與人腦視覺皮層組織方式相似。

## 總結

MLP 從單層感知器演進而來，透過多層結構獲得強大的表達能力。深度學習的突破在於：大量資料 + 強大算力 + 更好的訓練技巧，使訓練深層網路成為可能。
