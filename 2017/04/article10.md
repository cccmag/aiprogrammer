# 文章 10：神經網路學習資源

## 前言

本篇文章整理了學習神經網路與深度學習的優質資源，幫助讀者系統性學習。

## 線上課程

### Coursera - Machine Learning (Andrew Ng)
https://www.google.com/search?q=Coursera+Andrew+Ng+machine+learning

機器學習經典入門課程，涵蓋監督學習、非監督學習與類神經網路基礎。

### deeplearning.ai (Andrew Ng)
https://www.google.com/search?q=deeplearning.ai+neural+networks+deep+learning

吳恩達的深度學習專項課程，包含：
- 神經網路基礎
- 改善深度網路
- 結構化機器學習項目
- 卷積神經網路
- 序列模型

### fast.ai
https://www.google.com/search?q=fast.ai+deep+learning+course

實踐導向的深度學習課程，強調「由上而下」的學習方式。

## 書籍

### 《Deep Learning》 (Ian Goodfellow)
https://www.google.com/search?q=Deep+Learning+Ian+Goodfellow

深度學習領域的經典教科書，數學嚴謹，涵蓋廣泛。

### 《Neural Networks and Deep Learning》 (Michael Nielsen)
http://neuralnetworksanddeeplearning.com/

免費線上書籍，適合初學者從零理解神經網路。

### 《Hands-On Machine Learning》 (Aurélien Géron)
https://www.google.com/search?q=Hands-On+Machine+Learning+Aurelien+Geron

實踐導向，結合理論與程式碼。
