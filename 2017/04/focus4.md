# 焦點文章 4：反向傳播演算法

## 前言

反向傳播（Backpropagation）是訓練神經網路的核心演算法，於 1986 年由 Rumelhart、Hinton 和 Williams 正式提出。本章節詳細解說這個革命性的學習演算法。

## 核心思想

反向傳播利用**鏈式法則**（Chain Rule）計算損失函數對每個參數的梯度：

```
∂L/∂w = ∂L/∂a × ∂a/∂z × ∂z/∂w
```

將梯度由輸出層向輸入層反向傳播，因此得名「反向傳播」。

## 符號定義

- L：損失函數
- W⁽ˡ⁾：第 l 層的權重矩陣
- b⁽ˡ⁾：第 l 層的偏置向量
- a⁽ˡ⁾：第 l 層的輸出（激活值）
- z⁽ˡ⁾：第 l 層的加權和
- σ：激活函數

## 前向傳播

```
a⁽⁰⁾ = x
for l = 1 to L:
    z⁽ˡ⁾ = W⁽ˡ⁾a⁽ˡ⁻¹⁾ + b⁽ˡ⁾
    a⁽ˡ⁾ = σ(z⁽ˡ⁾)
```

最後計算損失 L(y, ŷ)。

## 反向傳播步驟

### 1. 輸出層誤差

對於均方誤差損失：
```
δ⁽ᴸ⁾ = ∂L/∂a⁽ᴸ⁾ ⊙ σ'(z⁽ᴸ⁾)
```

其中 ⊙ 表示元素對應相乘（Hadamard 積）。

### 2. 隱藏層誤差遞迴

```
δ⁽ˡ⁾ = (W⁽ˡ⁺¹⁾)ᵀ δ⁽ˡ⁺¹⁾ ⊙ σ'(z⁽ˡ⁾)
```

### 3. 梯度計算

```
∂L/∂W⁽ˡ⁾ = δ⁽ˡ⁾ (a⁽ˡ⁻¹⁾)ᵀ
∂L/∂b⁽ˡ⁾ = δ⁽ˡ⁾
```

### 4. 參數更新

```
W⁽ˡ⁾ = W⁽ˡ⁾ - η ∂L/∂W⁽ˡ⁾
b⁽ˡ⁾ = b⁽ˡ⁾ - η ∂L/∂b⁽ˡ⁾
```

## 梯度消失問題

早期反向傳播难以训练深层网络的主要原因：

- Sigmoid 和 Tanh 的梯度在兩端趨近於 0
- 誤差信號在反向傳播時逐層衰减
- 導致底層權重幾乎不更新

解決方案：
- 使用 ReLU 激活函數
- 殘差連接（ResNet）
- 批次正規化
- 更好的初始化方法

## 計算圖視角

現代深度學習框架（如 TensorFlow、PyTorch）使用**動態計算圖**：

1. 定義前向傳播運算
2. 框架自動建立計算圖
3. 反向傳播時自動微分

這大大簡化了神經網路的開發與調試。

## 總結

反向傳播是深度學習的基石。透過鏈式法則，它能高效計算深層網路中每個參數的梯度。現代框架的自動微分功能使我們無需手動推導梯度。
