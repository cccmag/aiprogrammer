# 2017 年 5 月號新聞摘要

## 人工智慧重大事件

- **AlphaGo 與柯洁對戰**：2017 年 5 月，AlphaGo 在為期 5 天的三番棋比賽中以 3-0 擊敗世界排名第一的中國圍棋棋手柯洁。這標志著 AI 在複雜策略遊戲領域已達到超越人類的水平。DeepMind 隨後宣布 AlphaGo 將不再參加比賽，將專注於醫療等實際應用。

- Google 收購機器學習競賽平台 Kaggle，進一步擴大其在 AI 開源生態系的影響力。

## 深度學習框架

- TensorFlow 1.2 版持續優化效能，新增 Python 3.6 支援。Keras 已成為 TensorFlow 官方高階 API。

- Facebook AI 研究院發布 PyTorch 0.2，增加維度操作與稀疏張量支援。

- Microsoft 推出 Cognitive Toolkit (CNTK) 2.0，強化分散式訓練能力。

## 電腦視覺發展

- ImageNet 比賽持續推動影像辨識技術進步，錯誤率已降至 3% 以下。

- GAN（生成對抗網路）研究火熱進展，新的變體如 WGAN、DCGAN 不斷湧現。

- 影像風格遷移技術變得更為普及，Prisma 等應用掀起熱潮。

## Python 生態系

- Python 3.6 生態系持續成熟，type hints 採用率上升。

- scikit-image 0.13 版發布，新增深度學習整合功能。
