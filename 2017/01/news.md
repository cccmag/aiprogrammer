# 本月新知

## 2017 年 1 月程式與 AI 技術動態

### 程式語言與框架

**Python 3.6 正式版发布（2016 年 12 月）**

Python 3.6 於 2016 年 12 月正式发布，為 Python 生態系帶來了多項重要新特性。最引人注目的是 **f-string**（格式化字串字面量），這種新的字串格式化語法比傳統的 `%format` 和 `.format()` 方法更加簡潔直觀。此外，Python 3.6 還帶來了改進的協程（asyncio 模組增強）、安全的類別屬性 Creation Order，以及新生代的 `secrets` 模組。效能方面，Python 3.6 透過字典實現優化，整體性能提升了 5-10%。

**PyPI 套件數量突破 10 萬大關**

Python Package Index（PyPI）在 2016 年底達到了 10 萬套件的里程碑。這標誌著 Python 生態系的繁榮與成熟。熱門套件包括 Django、Flask、requests、numpy、pandas、scikit-learn 等。社群開發的套件涵蓋網頁開發、資料處理、機器學習、網路爬蟲、自動化等各個領域。

**Anaconda 4.3 發布**

Anaconda 發行版於 2017 年 1 月發布 4.3 版本，這個版本默認使用 Python 3.6，並包含了 conda 4.3 的新功能。Anaconda 是資料科學家最喜愛的 Python 發行版，預裝了 195+ 個科學計算套件。

**Visual Studio Code Python 擴展**

微軟持續強化 VS Code 對 Python 的支援。2017 年 1 月的更新帶來了更好的 IntelliSense、語法高亮、最佳化了的偵錯功能。VS Code 正在成為 Python 開發者的熱門選擇。

### AI 與機器學習

**AlphaGo 進化：戰勝頂尖職業棋手**

2016 年 12 月，AlphaGo 的升級版本以 60-0 的成績戰勝了各國頂尖職業棋士。這次對局展示了深度強化學習在複雜策略遊戲中的驚人能力。AlphaGo 使用了深度神經網路與蒙地卡羅樹搜索的結合，標誌著 AI 技術的重大突破。這個成果為 2017 年 5 月 AlphaGo 與世界冠軍柯洁的世紀對決鋪平了道路。

**TensorFlow 動態計算圖：TensorFlow Fold**

Google 在 2017 年 1 月開源了 TensorFlow Fold，這是一個針對動態計算圖的庫。傳統 TensorFlow 在處理變長輸入時效率較低，TensorFlow Fold 通過依賴圖分析實現了動態批次處理，大幅提升了處理效率。這對於自然語言處理和影像處理等場景特別重要。

**NVIDIA DIGITS 深度學習訓練系統更新**

NVIDIA 在 2017 年 1 月更新了 DIGITS（Deep Learning GPU Training System），這是一個用於訓練深度學習模型的圖形化工具。新版本增加了對最新 GPU 架構的支援，並優化了多 GPU 訓練流程。

**Microsoft CNTK 2.0 發布**

Microsoft Cognitive Toolkit（CNTK）在 2017 年 1 月發布了 2.0 版本（beta）。CNTK 是微軟的開源深度學習框架，以高效能和分散式訓練能力著稱。2.0 版本帶來了 Python API 改進與效能最佳化。

### 業界動態

- **Python 登上 TIOBE 冠軍**：Python 在 TIOBE 指數中持續攀升，2017 年初已成為最受歡迎的程式語言之一
- **Django 1.11 開發中**：最受歡迎的 Python Web 框架持續更新，LTS 版本持續獲得安全支援
- **PyTorch 生態成長**：雖然 PyTorch 還很新，但其在研究社群的人氣持續上升

- **PEP 498**（f-string）：格式化字串字面量，讓字串格式化更加直觀

- **Python Taiwan 社群活躍**：台灣 Python 社群持續成長，舉辦多次meetup
- **Django Girls 訓練營全球擴展**： Django Girls 在 2017 年初已在 50+ 國家舉辦過培訓
