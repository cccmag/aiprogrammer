# GPT-3 傳聞：更大語言模型的風聲

## 前言

到 2019 年 8 月，關於 GPT-3（OpenAI 的下一代語言模型）的傳聞開始在社群中流傳。雖然 GPT-3 直到 2020 年才正式發布，但當時已經有各種猜測。

## 對 GPT-2 的觀察

### 規模的增長

```
GPT-1: 1.17 億參數 (2018)
GPT-2: 15 億參數 (2019)
GPT-3: 1750 億參數 (猜測中...)
```

### 能力推測

根據 GPT-2 的表現，業界猜測 GPT-3 可能的能力：

```
猜測 1：更強的生成能力
- 更長、更連貫的文本生成
- 更好的創意寫作

猜測 2：更多的 zero-shot 能力
- 執行未見過的任務
- 更強的泛化能力

猜測 3：更好的知識整合
- 更精確的事實回答
- 更少的幻覺
```
