# 主題總覽：機器學習入門

機器學習是讓電腦從資料中自動學習規律的技術，無需明確程式設計即可完成任務。2019 年初，機器學習已成為人工智慧領域的核心技術，廣泛應用於影像辨識、語音識別、推薦系統等。

## 為什麼要學習機器學習？

隨著資料量爆炸性增長，傳統程式設計已無法處理複雜的現實問題。機器學習透過讓系統從資料中自動發現模式，大幅提升了問題解決的效率與準確性。

## 機器學習類型地圖

### 監督式學習（Supervised Learning）

監督式學習使用已標記的資料進行訓練：

- **分類（Classification）**：預測類別標籤，如垃圾郵件偵測
- **迴歸（Regression）**：預測連續數值，如房價預測

常見演算法包括線性迴歸、邏輯斯迴歸、決策樹、支持向量機（SVM）、神經網路等。

### 非監督式學習（Unsupervised Learning）

非監督式學習使用未標記的資料發現結構：

- **聚類（Clustering）**：將相似資料分組，如客戶分群
- **降維（Dimensionality Reduction）**：減少特徵數量，如 PCA

常見演算法包括 K-means、層級聚類、DBSCAN、PCA、t-SNE 等。

### 增強式學習（Reinforcement Learning）

增強式學習透過環境回饋學習最佳策略，廣泛應用於遊戲 AI 與機器人控制。

## scikit-learn 在機器學習的角色

scikit-learn 是 Python 生態系中最受歡迎的機器學習庫，提供了一致的 API 設計、豐富的演算法實現以及完善的文檔。
