# 本月科技新聞 — 2019年2月

## Google 發布 BERT 程式碼與模型

2019 年 2 月，Google 正式開源 BERT（Bidirectional Encoder Representations from Transformers）的程式碼與預訓練模型。BERT 於 2018 年 10 月發布，在自然語言處理領域創下多項紀錄。2019 年初，社群開始廣泛應用 BERT 進行文字分類、問答系統等任務。

## scikit-learn 0.20 版本發布

scikit-learn 在 2019 年 2 月發布 0.20 版本，新增多項功能包括 ColumnTransformer、Pipeline 的改進以及更多预处理工具。scikit-learn 持續是 Python 機器學習的標準工具。

## TensorFlow 2.0 預覽版發布

Google 在 2019 年 2 月發布 TensorFlow 2.0 的預覽版本。2.0 版本強調 eager execution 預設開啟、Keras 整合以及簡化 API。正式版預計於 2019 年下半年發布。

## PyTorch 1.0 正式版生態系擴展

Facebook 的 PyTorch 1.0 正式版於 2018 年 12 月發布，2019 年 2 月其生態系持續擴展。PyTorch Mobile、TorchServe 等工具陸續推出，使得部署深度學習模型更加便利。

## AWS 發布機器學習新服務

AWS 在 2019 年 2 月宣布多项机器学习相关服务更新，包括 Amazon SageMaker 的新功能與强化學習工具。這些服務降低了機器學習的進入門檻。

## Python 仍是資料科學主流語言

根據 2019 年初的調查，Python 在資料科學與機器學習領域仍是首選語言。scikit-learn、NumPy、Pandas 形成了完整的機器學習工具鏈，吸引了大量開發者與研究者。
