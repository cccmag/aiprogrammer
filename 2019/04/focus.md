## 自然語言處理初探：從文字到語意向量

### 引言

自然語言處理（Natural Language Processing，NLP）是人工智慧領域中最具挑戰性的研究方向之一。與結構化的圖像或數值資料不同，人類語言具有豐富的語義、語法和上下文依賴性，這使得電腦理解人類語言變得極為困難。

本期將帶領讀者從 NLP 的基礎概念開始，探索文字預處理技術、詞嵌入方法，並深入了解 Word2Vec 和 GloVe 等經典演算法。我們將追蹤從 2013 年 Word2Vec 誕生到 2018 年 BERT 橫空出世這段時間內，NLP 領域的關鍵突破與技術演進。

## 濃縮回顧

### NLP 的挑戰

自然語言處理面臨諸多挑戰：

- **歧義性**：同一個詞可能有多種含義
- **上下文依賴**：詞的意義取決於上下文
- **結構多變**：句子結構可以非常靈活
- **世界知識**：理解語言需要廣泛的背景知識

### 詞嵌入的革命

2013 年 Word2Vec 的發表是 NLP 領域的重要里程碑：

```
傳統方法：one-hot 編碼（維度爆炸、語意缺失）
    ↓
Word2Vec：稠密向量（語意壓縮、語法/語意關係）
```

Word2Vec 展示了「意義」可以被表示為向量，而意義之間的關係可以透過向量運算來捕捉：

```python
# 經典範例
king - man + woman ≈ queen
```

### 技術發展時間線

| 年份 | 技術 | 意義 |
|-----|------|------|
| 2003 | 神經網路語言模型 | 首次使用神經網路學習詞向量 |
| 2013 | Word2Vec | 詞嵌入技術實用化 |
| 2014 | GloVe | 全域統計與局部上下文結合 |
| 2016 | fastText | 子詞嵌入處理未登錄詞 |
| 2017 | Transformer | 突破序列建模瓶頸 |
| 2018 | BERT | 預訓練語言模型革新 |

### 本期重點

本期將深入探討：

1. **文字預處理是 NLP 的基礎** — 好的預處理能大幅提升模型效果
2. **詞嵌入是 NLP 深度學習的基石** — 從離散符號到連續向量
3. **Word2Vec 开啟了詞嵌入時代** — 簡單而強大的方法
4. **從靜態向量到上下文表示** — BERT 預訓練模型的興起
