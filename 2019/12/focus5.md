# 電腦視覺的進展

## 前言

2019 年，電腦視覺領域繼續快速發展。從物體檢測到圖像分割，從人臉識別到 Vision Transformer，諸多進展值得回顧。

## 物體檢測的進步

### EfficientDet

2019 年，Google 發布了 EfficientDet，這是 EfficientDet 系列的開創性版本：

```
EfficientDet 的特點：
- 更高的效率和準確率
- 複合縮放策略
- 應用於多種視覺任務
```

### YOLOv3

YOLOv3 在 2019 年持續獲得廣泛使用：

```python
# YOLOv3 的優勢

- 易於部署
```

## 人臉識別的爭議

### 2019 年的發展

人臉識別技術在 2019 年取得了顯著進步：

```
進展：
- 準確率持續提升
- 部署範圍擴大
- 倫理問題引發關注
```

### 隱私爭議

人臉識別引發了廣泛的隱私討論：

```
爭議：
- 執法機構使用引發擔憂
- 歧視性問題凸顯
- 多個城市禁止使用
```

## Vision Transformer 的先驅

### 2019 年的研究

雖然 Vision Transformer（ViT）在 2020 年才正式發布，但相關研究在 2019 年就已經開始：

```
先驅研究：
- Image Transformer
- Self-attention for image recognition
- Early Vision Transformers
```

## 結論

2019 年電腦視覺領域繼續快速發展。物體檢測更加高效，人臉識別引發倫理討論，而 Vision Transformer 的研究正在為 2020 年的突破做準備。
