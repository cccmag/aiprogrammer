# Python 2019 年度回顧：3.8 與生態系

## 前言

2019 年，Python 繼續保持全球最受歡迎程式語言的地位。Python 3.8 的發布和生態系的持續繁榮是這一年的重要主題。

## Python 3.8 的發布

### 10 月正式發布

Python 3.8 於 2019 年 10 月正式發布：

```
Python 3.8 主要新特性：
- 賦值表達式（:=）
- 位置-only 參數（/）
- 改進的 typing 模組
- f-strings 支援 = 說明符
```

### Walrus Operator

賦值表達式是 Python 3.8 最重要的新特性：

```python
# 之前
if len(data) > 10:
    print(f"Data too long: {len(data)}")

# Python 3.8
if (n := len(data)) > 10:
    print(f"Data too long: {n}")
```
