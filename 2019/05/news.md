# 本月新知

## 2019 年 5 月程式與 AI 技術動態

### 程式語言與框架

**Python 生態持續壯大**

Python 3.7 穩定版在本月繼續獲得採用。隨著更多大型專案完成從 Python 2 的遷移，社群正在積極準備未來的功能更新。typing 模組的改進和 dataclass 的使用逐漸普及。

**深度學習框架更新**

TensorFlow 1.13 和 PyTorch 1.1 先後發布。PyTorch 1.1 加強了 JIT 編譯器和量化支援，而 TensorFlow 繼續為 2.0 版本做準備。Keras 已成為 TensorFlow 的官方高階 API。

### AI 與機器學習

**電腦視覺持續突破**

2019 年 5 月，電腦視覺領域見證多項進展。影像分類、物件偵測和語意分割的標準持續提升。ResNet 及其變體仍是最廣泛使用的骨幹網路。

**ImageNet 競賽的回顧**

ImageNet 競賽見證了深度學習的驚人進步。從 2012 年 AlexNet 突破性的 16% 錯誤率，到 2015 年 ResNet 達到的 3.6% 人類水準表現，這段歷史展示了深度卷積神經網路的潛力。

**物體偵測技術成熟**

YOLO（You Only Look Once）系列持續迭代，RCNN 系利（RCNN、Fast RCNN、Faster RCNN）在精確度和速度之間取得更好平衡。這些技術已廣泛應用於自動駕駛、醫學影像等領域。

**生成對抗網路（GAN）**

GAN 技術在 2019 年持續發展，從影像生成到風格遷移都有顯著進展。CycleGAN、StarGAN 等模型展示了卓越的圖像轉換能力。

### GPU 與硬體

**NVIDIA 持續更新**

NVIDIA 在本月繼續推動 CUDA 和 cuDNN 的更新。RTX 系列顯示卡的光線追蹤和 AI 加速能力為深度學習提供更強大的運算支援。

**邊緣運算興起**

隨著 Edge TPU 和 NVIDIA Jetson 等硬體的普及，深度學習模型在邊緣設備上的部署變得更加便利。

### 雲端服務

**Google I/O 2019**

Google 在本月舉辦 I/O 開發者大會，發布了多項 AI 相關更新：
- TensorFlow 2.0 預覽
- AI Platform 更新
- AutoML Vision 強化

**雲端 ML 平台竟爭**

AWS SageMaker、Azure ML、Google Cloud AI Platform 持續競爭，紛紛推出新功能搶占市場。

### 業界動態

- **華為實體清單風波**：影響部分 AI 硬體和軟體供應
- **Google 發布 MediaPipe**：多模態感知框架
- **Facebook AI 開源更多工具**：PyTorch 生態持續擴展
- **百度開源 Apollo 3.5**：自動駕駛平臺更新

### 標準與規範

- **ONNX 持續推廣**：跨框架模型交換格式
- **WebDNN 標準化**：Web 上的深度學習推理
- **HTTP/3 草案完成**：QUIC 協定接近完成
