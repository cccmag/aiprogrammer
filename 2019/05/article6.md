# ImageNet 競賽：深度學習的里程碑

## 前言

ImageNet Large Scale Visual Recognition Challenge (ILSVRC) 是電腦視覺領域最重要的競賽之一。

## 競賽歷史

| 年份 | 冠軍模型 | Top-5 錯誤率 |
|-----|---------|------------|
| 2010 | NEC-MU | 28.2% |
| 2011 | SJVC | 25.7% |
| 2012 | AlexNet | 16.4% |
| 2013 | VGG | 7.3% |
| 2014 | GoogLeNet | 6.7% |
| 2015 | ResNet | 3.6% |

## 為什麼 ImageNet 重要？

1. **大規模資料**：1400萬張圖像，1000個類別
2. **人類水準**：人類的錯誤率約 5.1%
3. **推動創新**：AlexNet、ResNet 等突破性模型誕生

## AlexNet 的突破（2012）

```python
# AlexNet 關鍵創新
- 深度卷積網路（8層）
- ReLU 激活函數
- GPU 訓練（使用 CUDA）
- Dropout 正規化
- Data Augmentation
```

## ResNet 達到人類水準（2015）

何愷明等人提出殘差網路，在 ImageNet 上達到 3.6% 的錯誤率，首次超越人類。
