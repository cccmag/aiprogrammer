# 本月科技新聞 — 2019年1月

## Python 3.7 生態系持續成長

2019 年 1 月，Python 3.7 發布已滿六個月。根據 PyPI 統計，Python 3.7 的採用率持續攀升，已超過 Python 3.6 成为主流版本。社群開始討論 Python 3.8 的未來特性，包括 walrus 運算子與位置參數限定。Python 3.8 預計於 2019 年 10 月發布。

## BERT 被選為 NAACL 2019 最佳論文

Google AI 的 BERT（Bidirectional Encoder Representations from Transformers）在 NAACL 2019（北美計算語言學年會）獲得最佳論文獎。BERT 於 2018 年 10 月發布，其雙向 Transformer 架構在十一項自然語言處理任務中刷新紀錄。2019 年 1 月，社群開始廣泛採用 BERT 進行各種 NLP 任務。

## OpenAI 公布 GPT-2 開發計畫

2019 年 2 月，OpenAI 宣布將發布 GPT-2，這是一個擁有 15 億參數的大型語言模型。基於 GPT（2018 年 6 月）的成功，GPT-2 在生成文字方面展現了卓越能力。OpenAI 表示將分階段發布，以评估模型的社會影響。

## PyTorch 1.0 生態系擴展

PyTorch 1.0 於 2018 年 12 月發布，2019 年 1 月其生態系持續擴展。Facebook 宣布加強對 PyTorch 的支援，包括 TorchServe 服務框架與量化工具。學術界對 PyTorch 的採用率持續上升，挑戰 TensorFlow 的主導地位。

## AWS 發布 SageMaker新功能

AWS 在 2019 年 1 月的 CES 期間宣布 Amazon SageMaker 新功能，包括 Autopilot 自動化機器學習與 Marketplace 模型市集。這些工具降低了機器學習的進入門檻，讓更多開發者能夠快速建置 ML 模型。

## Python 封裝工具持續演進

Python Packaging Authority 在 2019 年 1 月持續改進 pip 工具。PEP 517/518 已成標準，越來越多專案採用 pyproject.toml 設定。Poetry 與 Pipenv 持續獲得採用，但 setup.py 仍是主流。這個生態系的演進代表 Python 專業化的趨勢。
