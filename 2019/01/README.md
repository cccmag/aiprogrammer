# AI 程式人雜誌 — 2019年01月號

## 主題：Python 3.7 生態系

Python 3.7 於 2018 年 6 月發布，經過半年的發展，2019 年 1 月已成為新專案的主流選擇。本期聚焦於 Python 3.7 的核心特性：asyncio 非同步程式設計、型態提示（Type Hints）、資料類別（Dataclasses）以及效能改進。這些特性大幅提升了 Python 的程式碼品質與開發效率。
