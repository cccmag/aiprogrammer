# Node.js 13 發布：ES Module 支援

## 前言

Node.js 13 於 2019 年 10 月發布，持續推進 ES Module 的支援。雖然穩定版要到 Node.js 14 才會默認開啟，但 ES Module 功能已經取得了重要進展。

## ES Module 支援

### 導入方式

```javascript
// 命名導入
import { readFile } from 'fs';

// 默認導入
import express from 'express';

// 名稱導入
import * as fs from 'fs';
```

---

## 結語

Node.js 對 ES Module 的支援正在穩步前進，未來版本將全面支持。
