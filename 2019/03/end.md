# 結語

類神經網路是深度學習的基石，2019 年初已成為人工智慧領域最重要的技術之一。

## 關鍵 takeaways

1. **感知器基礎**：理解感知器的原理與局限性
2. **MLP 架構**：多層網路能夠學習非線性問題
3. **激活函數**：ReLU 是隱藏層的首選
4. **反向傳播**：理解鏈式法則在梯度計算中的應用
5. **實務技巧**：權重初始化、Batch Normalization、Early Stopping

## 未來學習方向

### CNN（卷積神經網路）
- 適合影像處理
- 局部感受野、權重共享

### RNN/LSTM
- 適合序列資料
- 自然語言處理

### Transformer
- 自注意力機制
- BERT、GPT 的基礎

## 框架選擇

2019 年主流框架：
- **TensorFlow/Keras**：生態系完整，生產部署能力強
- **PyTorch**：動態計算圖，研究友好
- **FastAI**：高層 API，快速原型
