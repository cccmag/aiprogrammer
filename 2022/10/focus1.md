# 強化學習基礎

## Agent、環境、獎勵

### MDP 框架

強化學習的核心是馬可夫決策過程（Markov Decision Process, MDP），由五元組 (S, A, P, R, γ) 定義：

- **S**：狀態空間，環境的所有可能狀態
- **A**：動作空間，Agent 可以採取的所有動作
- **P(s' | s, a)**：狀態轉移機率，從狀態 s 執行動作 a 後到達 s' 的機率
- **R(s, a)**：獎勵函數，Agent 在狀態 s 執行動作 a 獲得的即時獎勵
- **γ ∈ [0,1]**：折扣因子，平衡即時與長期獎勵

### Agent 與環境的互動

```
Agent ──→ 動作 a
       ←── 狀態 s'、獎勵 r

環境：接收動作，更新狀態，產生獎勵
```

在 RLHF 的語境中，Agent 是語言模型，環境是「人類使用者 + 任務」，動作是生成的回應，獎勵則來自獎勵模型（或人類直接評分）。

### 策略與值函數

**策略 π(a|s)**：Agent 在狀態 s 下採取動作 a 的機率。策略分為：
- 確定性策略：a = π(s)，一個狀態對應一個動作
- 隨機性策略：a ~ π(·|s)，一個狀態對應動作的機率分佈

語言模型本身就是一個隨機性策略：給定提示（狀態），生成詞元（動作）的機率分佈。

**狀態值函數 V^π(s)**：在策略 π 下，從狀態 s 出發的期望累積折扣獎勵：

```
V^π(s) = E[ Σ γ^t r_t | s_0 = s, π ]
```

**動作值函數 Q^π(s, a)**：在策略 π 下，從狀態 s 執行動作 a 後的期望累積折扣獎勵：

```
Q^π(s, a) = E[ Σ γ^t r_t | s_0 = s, a_0 = a, π ]
```

### 優勢函數

優勢函數 A^π(s, a) = Q^π(s, a) - V^π(s) 衡量在狀態 s 下執行動作 a 相對於平均水準的優勢。正值表示該動作優於平均，負值表示劣於平均。PPO 演算法中，優勢函數是計算策略梯度的關鍵。

### 貝爾曼方程

值函數滿足貝爾曼方程，這是動態規劃和許多 RL 演算法的基礎：

```
V^π(s) = Σ_a π(a|s) [ R(s, a) + γ Σ_{s'} P(s'|s,a) V^π(s') ]
Q^π(s, a) = R(s, a) + γ Σ_{s'} P(s'|s,a) Σ_{a'} π(a'|s') Q^π(s', a')
```

### RL 的三大類方法

1. **基於值的方法**（Value-based）：學習 Q 函數，透過 Q 函數推導策略。代表：Q-Learning、DQN
2. **基於策略的方法**（Policy-based）：直接最佳化策略參數。代表：REINFORCE、PPO
3. **Actor-Critic 方法**：同時學習策略（Actor）和值函數（Critic）。代表：A2C、PPO

RLHF 中的 PPO 屬於 Actor-Critic 方法——語言模型是 Actor，獎勵模型或值函數是 Critic。

### 探索與利用

RL 中永恆的權衡：Agent 應該利用已知的好策略（利用），還是嘗試未知的動作（探索）？在 RLHF 中，這個權衡體現在語言模型的溫度參數和多樣性控制上。

### 與監督學習的區別

| 特性 | 監督學習 | 強化學習 |
|------|---------|---------|
| 資料 | 靜態標籤 | 動態互動 |
| 目標 | 擬合標籤 | 最大化獎勵 |
| 回饋 | 每個樣本 | 延遲、稀疏 |
| 分佈 | 固定 | 策略相關 |
