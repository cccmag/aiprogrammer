# 反向傳播推導

## 預備知識

反向傳播（Backpropagation）的核心是微積分中的鏈式法則。在推導之前，先定義符號：

```
L      = 損失函數
a^(l)  = 第 l 層的啟用值
z^(l)  = 第 l 層的加權和（z = w·a + b）
w^(l)  = 第 l 層的權重矩陣
b^(l)  = 第 l 層的偏置向量
σ      = 啟用函數
n_l    = 第 l 層的神經元數
```

前向傳播：
```
z^(l) = w^(l)·a^(l-1) + b^(l)
a^(l) = σ(z^(l))
```

## 輸出層的梯度

考慮一個 3 層網路（L=3），使用均方誤差損失和 Sigmoid 啟用。

損失函數：
```
L = (1/2) · Σ(a^L_j - y_j)²
```

對輸出層第 j 個神經元的梯度：

```
∂L/∂a^(L)_j = a^(L)_j - y_j
```

由鏈式法則：
```
∂L/∂w^(L)_jk = (∂L/∂a^(L)_j) · (∂a^(L)_j/∂z^(L)_j) · (∂z^(L)_j/∂w^(L)_jk)
```

各項計算：
```
∂a^(L)_j/∂z^(L)_j = σ'(z^(L)_j)
∂z^(L)_j/∂w^(L)_jk = a^(L-1)_k
```

因此：
```
∂L/∂w^(L)_jk = (a^(L)_j - y_j) · σ'(z^(L)_j) · a^(L-1)_k
```

## 輸出層誤差的向量化

定義誤差項 δ^(L)_j：
```
δ^(L)_j = ∂L/∂z^(L)_j = (a^(L)_j - y_j) · σ'(z^(L)_j)
```

向量化：
```
δ^(L) = (a^(L) - y) ⊙ σ'(z^(L))
```

## 誤差反向傳播

目標：用 δ^(l+1) 表示 δ^(l)。

由鏈式法則：
```
δ^(l)_j = ∂L/∂z^(l)_j
        = Σ_k (∂L/∂z^(l+1)_k) · (∂z^(l+1)_k/∂z^(l)_j)
        = Σ_k δ^(l+1)_k · (∂z^(l+1)_k/∂z^(l)_j)
```

由於 z^(l+1)_k = Σ_i w^(l+1)_ki · σ(z^(l)_i) + b^(l+1)_k：

```
∂z^(l+1)_k/∂z^(l)_j = w^(l+1)_kj · σ'(z^(l)_j)
```

因此：
```
δ^(l)_j = Σ_k δ^(l+1)_k · w^(l+1)_kj · σ'(z^(l)_j)
        = (Σ_k δ^(l+1)_k · w^(l+1)_kj) · σ'(z^(l)_j)
```

向量化：
```
δ^(l) = ((w^(l+1))^T · δ^(l+1)) ⊙ σ'(z^(l))
```

## 權重梯度

得到了 δ^(l) 後，權重梯度為：

```
∂L/∂w^(l)_jk = δ^(l)_j · a^(l-1)_k
```

向量化：
```
∂L/∂w^(l) = δ^(l) · (a^(l-1))^T
```

偏置梯度：
```
∂L/∂b^(l)_j = δ^(l)_j
```

向量化：
```
∂L/∂b^(l) = δ^(l)
```

## 完整的演算法

**輸入**：訓練樣本 (x, y)，網路結構

**步驟 1：前向傳播**
```
a^(0) = x
for l = 1 to L:
    z^(l) = w^(l)·a^(l-1) + b^(l)
    a^(l) = σ(z^(l))
```

**步驟 2：輸出層誤差**
```
δ^(L) = (a^(L) - y) ⊙ σ'(z^(L))
```

**步驟 3：反向傳播**
```
for l = L-1 to 1:
    δ^(l) = ((w^(l+1))^T · δ^(l+1)) ⊙ σ'(z^(l))
```

**步驟 4：梯度下降**
```
for l = 1 to L:
    w^(l) ← w^(l) - η · δ^(l) · (a^(l-1))^T
    b^(l) ← b^(l) - η · δ^(l)
```

## 手算範例

以 XOR 問題為例，一個 2-2-1 網路：

```
輸入 x = [0, 1]，目標 y = [1]

前向傳播：
z^(1) = [0.1, -0.2]     a^(1) = [0.52, 0.45]
z^(2) = 0.15            a^(2) = 0.54

反向傳播：
δ^(2) = (0.54-1) · 0.54·0.46 = -0.114
δ^(1)_1 = -0.114 · 0.3 · 0.52·0.48 = -0.0085
δ^(1)_2 = -0.114 · 0.4 · 0.45·0.55 = -0.0113
```

## 總結

反向傳播的優雅在於：
1. 使用鏈式法則系統性地計算梯度
2. 計算效率高（O(n) 相對於 O(n²) 的樸素方法）
3. 與網路結構無關，適用於任何可微模型
4. 自動微分框架將其推廣到任意計算圖
