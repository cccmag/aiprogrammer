## 神經網路與深度學習復興 — 從感知器到多層網路

### 引言

在人工智慧的發展史上，有一段長達數十年的「AI 寒冬」時期，其中神經網路曾被視為沒有前途的研究方向。然而，從 2006 年 Geoffrey Hinton 提出深度信念網路的預訓練方法開始，神經網路經歷了一場輝煌的復興，最終催生了改變世界的深度學習革命。

本期雜誌將帶領讀者穿越這段技術演進的時光隧道，從最基礎的感知器開始，一步步建立對神經網路和深度學習的全面理解。

### 核心概念圖

```
感知器 (1958) ── 線性分類器，無法解決 XOR
    │
    ▼
多層感知器 + 反向傳播 (1986) ── 非線性分類的突破
    │
    ▼
啟用函數進化 (Sigmoid → ReLU → GELU) ── 訓練效率不斷提升
    │
    ▼
正則化技術 (Dropout, BatchNorm) ── 解決過擬合
    │
    ▼
深層網路架構 (CNN, RNN, ResNet, Transformer) ── 各領域的突破
```

## 濃縮回顧

### 感知器的誕生

1958 年，Frank Rosenblatt 在 Cornell 航空實驗室提出了感知器（Perceptron），這是第一個具有學習能力的神經網路模型。感知器的核心是一個線性分類器：

```
y = sign(w·x + b)
```

感知器能學習 AND 和 OR 這樣的線性可分問題，但無法解決 XOR 問題。

### 反向傳播的革命

1986 年，David Rumelhart、Geoffrey Hinton 和 Ronald Williams 發表了反向傳播演算法（Backpropagation），使得多層神經網路的訓練成為可能。反向傳播利用鏈式法則計算損失函數對每層權重的梯度，隨後進行梯度下降更新。

### 深度學習的崛起

2006 年，Hinton 提出深度信念網路和逐層預訓練方法，標誌著深度學習的正式崛起。隨後，2012 年 AlexNet 在 ImageNet 競賽上的勝利，開啓了深度學習在各領域的全面應用。

```
2012 AlexNet (CNN) ── 影像分類的突破
2014 GAN, Seq2Seq ── 生成模型與序列學習
2015 ResNet ── 極深網路的訓練
2017 Transformer ── NLP 的革命
2020 GPT-3 ── 大規模語言模型
```

---

## 結論與展望

神經網路從 1958 年的感知器到今天的 Transformer，經歷了漫長而曲折的發展道路。每個階段的突破都依賴於前人的理論基礎，同時也為下一代技術鋪平了道路。

當前，深度學習正朝著更高效、更可解釋、更通用的方向發展。自監督學習、多模態模型和 AI Agent 是值得關注的前沿方向。
