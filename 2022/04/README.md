## 本月摘要

2022年4月，PyTorch 1.11 正式發布，深度學習框架市佔持續向 PyTorch 傾斜。本月我們深入探討 PyTorch 的核心概念與實戰技巧，從 Tensor 基礎操作到完整訓練管線，幫助讀者掌握這個當今最重要的深度學習框架。
