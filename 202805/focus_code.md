# 程式實作：從零實作強化學習

## 簡介

從零實作強化學習演算法，涵蓋 Q-Learning、Policy Gradient (REINFORCE)、RLHF 模擬。完整程式碼在 `_code/reinforcement_learning.py`。

## 核心功能

- **GridWorld 環境**：自訂迷宮、障礙物、目標
- **Q-Learning**：表格 Q 學習、ε-greedy 探索
- **Policy Gradient**：Softmax 策略、REINFORCE
- **RLHF 模擬**：偏好收集、獎勵模型

## 執行方式

```bash
cd _code
python3 reinforcement_learning.py
```
