# 本期焦點

## 知識管理與 RAG 進階 — 從向量搜尋到知識圖譜

RAG（Retrieval-Augmented Generation）已經成為 LLM 應用的標準架構。但隨著應用場景的複雜化，簡單的向量搜尋已不足以滿足需求。知識圖譜、圖資料庫、結構化資料檢索——新一代 RAG 正在整合多種知識來源。

本期探討 RAG 的進階主題：GraphRAG、多跳檢索、結構化資料查詢、知識圖譜建構、Agentic RAG、以及 RAG 系統的評估與監控。
