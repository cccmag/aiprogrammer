# 本月新知

## 2008 年 5 月程式與技術動態

### NoSQL 運動

**BigTable 論文影響深遠**

2006 年 Google 發表 BigTable 論文，介紹了列式儲存的概念。這種新型資料庫設計影響了 HBase、Cassandra 等開源專案的發展。

**Amazon Dynamo 論文發表**

2007 年 Amazon 發表 Dynamo 論文，介紹了最終一致性的分散式鍵值儲存。LinkedIn、Netflix 等公司開始採用類似的設計。

**文件資料庫興起**

CouchDB、MongoDB 等文件資料庫開始受到關注。JSON 文件儲存提供了靈活的資料模型。

### 程式語言與框架

**Python 2.6 持續開發**

Python 2.6 增加了多項新特性，包括文字格式化、效能改進等。Pylons、Web.py 等框架持續活躍。

**jQuery 1.2 發布**

jQuery 1.2 帶來了重要的效能改進和外掛系統。jQuery 迅速成為最受歡迎的 JavaScript 框架。

**Ruby on Rails 2.1 發布**

Rails 2.1 帶來了時區支援、垃圾回收改進等新特性。

### 雲端運算

**Google App Engine 開放**

Google App Engine 讓開發者能夠在 Google 基礎設施上運行 Python Web 應用。雲端平台開始普及。

**Amazon S3 服務成長**

Amazon S3 成為雲端儲存的標準，每日處理數十億次請求。

**雲端資料庫服務**

SimpleDB、RDS 等資料庫服務開始提供，簡化了資料庫管理。

### AI 與機器學習

**深度學習研究進展**

Geoffrey Hinton 等研究者持續推進深度學習研究。神經網路的深度架構開始受到關注。

**雲端機器學習**

Amazon Machine Learning 等服務開始出現，讓更多開發者能夠使用 ML 技術。

**推薦系統成熟**

Netflix、Amazon 等公司的推薦系統持續優化，成為電子商務的核心技術。

### 瀏覽器與網頁技術

**Chrome 開發中**

Google 積極開發 Chrome 瀏覽器，採用 V8 JavaScript 引擎。將於 2008 年 9 月正式發布。

**Web 2.0 應用持續增長**

社交網路、微博客、協作工具等 Web 2.0 應用持續增長。

**HTML 5 草案制定**

W3C 持續制定 HTML 5 標準，引入 Video、Canvas 等新元素。

### 業界動態

- **Facebook 推出 Platform API**：催生了大量第三方應用
- **Twitter 快速成長**：微博服務改變資訊傳播方式
- **GitHub 使用者增加**：程式碼托管平台日益受歡迎

### 開源專案

- **MongoDB**：文件資料庫開始獲得關注
- **CouchDB**：Apache 專案，CouchDB 0.8 發布
- **Cassandra**：Facebook 貢獻給 Apache
