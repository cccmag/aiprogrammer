# 語音辨識技術

## 前言

語音辨識讓電腦能夠理解人類的語音。2008 年時，基於隱藏馬可夫模型的語音辨識系統已經相當成熟，但深度學習即將帶來革命性的改變。

## 語音辨識的基本原理

```python
# 語音辨識系統的基本架構

asr_pipeline = {
```

## 聲學特徵

### MFCC（梅爾頻率倒譜系數）

```python
# 最常用的語音特徵

    "1. FFT": "將時域訊號轉換為頻域",
    "2. 梅爾濾波器組": "模擬人耳對頻率的感知",
    "3. 對數運算": "取對數壓縮動態範圍",
    "4. DCT": "離散餘弦轉換得到倒譜係數",
    "結果": "13-40 維特徵向量（通常 13 或 26 維）"
```
