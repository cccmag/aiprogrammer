# 本月新知

## 2008 年 2 月程式與技術動態

### 瀏覽器與網頁技術

**Firefox 3 Beta 持續改進**

Mozilla 在 2008 年持續開發 Firefox 3，代號「Gran Paradiso」。新版本帶來了全新的取樣引擎（TraceMonkey）、改進的書籤系統（Places）、以及更快的頁面渲染速度。

**Internet Explorer 8 Beta 1 發布**

微軟在 2008 年 3 月將發布 IE8，但 2008 年初已有測試版本流出。IE8 引入了相容性模式、網頁切片（Web Slices）、Activities 等新功能。

**Web 標準之爭**

瀏覽器市場的競爭讓各大廠商更加重視網頁標準的支援。CSS 2.1、HTML 5 草案、SVG、Canvas 等技術的標準化進程成為業界焦點。

### 作業系統

**Windows Vista Service Pack 1**

Vista SP1 在 2008 年 2 月正式發布，修復了大量問題，提升了效能和穩定性。SP1 還簡化了部署流程，更容易企業採用。

**Mac OS X Leopard 一周年**

Mac OS X 10.5 Leopard 自 2007 年 10 月發布以來，持續獲得更新。Time Machine 備份、Spaces 虛擬桌面、Boot Camp 等功能受到用戶歡迎。

**Linux 核心 2.6.24**

2008 年 1 月發布的 Linux 核心 2.6.24 引入了重要的仮想化支援和電源管理改進。

### 程式語言與框架

**Ruby on Rails 2.0**

Rails 2.0 在 2007 年 12 月發布，帶來了 RESTful 路由、資源導向架構、以及更好的效能。Rails 社群持續快速成長。

**Python 2.5.1 穩定版**

Python 2.5 系列持續是主流，社群開始討論 Python 3000（Python 3.0）的發展方向。

**JavaScript 框架普及**

Dojo、Prototype、jQuery、ExtJS 等 JavaScript 框架開始普及，簡化了 AJAX 開發和 DOM 操作。

### AI 與機器學習

**Deep Learning 初期研究**

2008 年時，深度學習仍是學術界的研究課題。Geoffrey Hinton 和他的學生們持續推進類神經網路的理論研究。

**Watson 在 Jeopardy! 競賽前的研究**

IBM 的 Watson 系統正在為日後與人類的 Jeopardy! 對決做準備。這基於多年的問答系統和自然語言處理研究。

**電腦視覺競賽**

ImageNet 計畫正在準備中，將在 2009 年啟動第一屆競賽。這成為日後電腦視覺研究的重要基準。

### 業界動態

- **Google 收購 DoubleClick**：線上廣告市場整合加劇
- **亞馬遜 AWS 營收成長**：雲端運算商业模式得到驗證
- **Facebook 用戶破億**：社交網路影響力持續擴大
- **智慧型手機市場持續成長**：iPhone 和 BlackBerry 熱賣

### 標準與規範

- **HTML 5 草案制定中**：W3C 和 WHATWG 持續合作
- **ECMAScript 4 討論激烈**：JavaScript 未來版本方向未定
- **CSS 2.1 接近完成**：成為重要的參照標準

### 值得關注的開源專案

- **Chromium**：Google Chrome 的開放原始碼專案
- **WebKit**：蘋果推動的瀏覽器引擎
- **V8**：Google 的 JavaScript 引擎
- **jQuery**：簡潔的 JavaScript 函式庫
