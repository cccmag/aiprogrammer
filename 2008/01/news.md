# 本月新知

## 2008 年 1 月程式與技術動態

### 作業系統與平台

**Android SDK 預覽版發布**

2007 年 11 月，Google 與開放手機聯盟（Open Handset Alliance）宣布 Android 計畫。2008 年 1 月，Android SDK 預覽版正式開放開發者下載。Android 是第一個基於 Linux 的完整手機作業系統，為行動開發帶來嶄新的可能性。

**開放手機聯盟成立**

開放手機聯盟由 Google 領導，包含 34 家硬體、軟體、電信業者。聯盟目標是建立開放的行動電話標準，讓所有開發者都能為 Android 平台創造豐富的應用程式。

**Vista 上市一週年**

Windows Vista 於 2007 年 1 月上市，至 2008 年 1 月滿一週年。Vista 的使用者介面改進和安全性提升受到關注，但相容性問題和效能需求也受到批評。

### 程式語言與框架

**Python 2.6 開發中**

Python 社群持續發展 2.6 版本，引入了更多現代化特性。雖然 Python 3.0 也在規劃中，但 2.x 系列仍將是主流。

**Java SE 6 普及**

Java SE 6 自 2006 年底發布後，在企業環境中的採用率持續提升。Java 在伺服器端和 Android 開發中的地位日益重要。

**Ruby on Rails 持續熱門**

Ruby on Rails 的「Convention over Configuration」理念影響了众多網頁框架的設計。2008 年初，Rails 社群持續成長。

### AI 與機器學習

**Netflix 競賽推動推薦系統研究**

Netflix 舉辦的推薦系統競賽（Netflix Prize）進入關鍵階段，參賽團隊，持續最佳化演算法以提升電影推薦準確率。這推動了協同過濾技術的顯著進步。

**GPU 運算用於機器學習**

研究人員開始使用 GPU 加速機器學習計算。NVIDIA 的 CUDA 平台讓平行運算變得更易於使用，為日後的深度學習奠定基礎。

**雲端運算概念普及**

Amazon Web Services（AWS）在 2008 年初持續擴展，EC2、S3、SimpleDB 等服務日益成熟。「雲端運算」成為熱門關鍵字。

### 瀏覽器與網頁技術

**Firefox 3 Beta 發布**

Mozilla 持續改進 Firefox 3，引入了新的取樣引擎和效能優化。瀏覽器市場競爭激烈。

**Web 2.0 應用持續爆發**

Ajax、RESTful API、社交網路整合等技术继续成熟。2008 年初，Facebook、Twitter 等平台的 API 開放催生了大量第三方應用。

### 業界動態

- **亞馬遜 Kindle 热销**：電子閱讀器市場開始受到關注
- **蘋果 iPhone 進入企業市場**：蘋果開始重視企業應用
- **微軟收購 Yahoo 失敗**：驚動科技業的重大收購案
- **Linux 核心 2.6.24 發布**：納入更多驅動程式和虛擬化支援

### 標準與規範

- **HTML 5 草案持續制定**：W3C 開始制定 HTML 5 標準
- **ECMAScript 4 討論熱烈**：JavaScript 未來版本的方向
- **RESTful API 成主流**：SOAP 被逐漸邊緣化

### 值得關注的開源專案

- **Android**：Google 的開放手機平台
- **Hadoop**：大規模資料處理框架
- **MongoDB**：文件導向資料庫
- **Redis**：記憶體 key-value 資料庫
