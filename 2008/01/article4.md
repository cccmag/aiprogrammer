2008 年是行動運算的轉捩點。智慧型手機、輕省筆電（Netbook）、 MID（Mobile Internet Device）等新型態的運算裝置開始普及。本文探討這些趨勢。

## 輕省筆電（Netbook）現象

### Eee PC 的成功

華碩的 Eee PC 在 2007 年 10 月發布，開創了 Netbook 類別：

```
Eee PC 規格（2007）：
├── 處理器：Intel Celeron / Atom
├── 記憶體：512MB - 1GB
├── 儲存：2GB - 8GB SSD
├── 螢幕：7-10 吋
├── 價格：$199 - $399
└── OS：Linux / Windows XP
```

### 市場反應

Netbook 的成功原因：

1. **價格親民**：傳統筆電的一半價格
2. **輕巧便攜**：易於攜帶
3. **基本功能**：上網、文書處理足夠
4. **Linux 選項**：降低授權成本

## 行動網路的進化

### 3G 網路普及

2008 年，3G 網路在全球快速普及：

- 美國：Verizon、Ericsson 3G 網路覆蓋擴大
- 歐洲：HSPA 商業服務
- 亞洲：WCDMA 持續擴展

### WiMAX 的出現

WiMAX（802.16e）作為新興的寬頻技術：

- 理論速度：70 Mbps
- 覆蓋範圍：數公里
- 2008 年：Sprint 宣佈 WiMAX 商業服務

### Wi-Fi 與行動網路的結合

```
現代行動網路策略：

在家/辦公室 ──→ Wi-Fi（免費、高速）
外出 ──────→ 3G/HSPA（持續連線）
偏遠地區 ────→ Edge/GPRS（備援）
```

## 觸控介面的革新

### 多點觸控技術

iPhone 的多點觸控技術影響深遠：

| 技術 | 供應商 | 應用 |
|------|--------|------|
| 電容式觸控 | Apple, LGE | 智慧型手機 |
| 電阻式觸控 | 傳統廠商 | 功能手機、GPS |
| 觸控筆 | Wacom | 繪圖板 |

### 觸控 UI 設計原則

```
觸控介面設計原則：

1. 手指友好的目標大小（至少 44px）
2. 清晰的視覺回饋
3. 手勢支援（滑動、捏合）
4. 方向感測（自動翻轉）
5. 減少輸入錯誤的設計
```

## 未來展望

### 融合的趨勢

未來幾年的重要趨勢：

1. **手機與電腦的融合**：智慧型手機效能接近筆電
2. **雲端服務**：資料和運算移到雲端
3. **永遠連線**：3G/4G 成為標準
4. **觸控普及**：各種裝置都支援觸控
