# 本月新知

## 2008 年 6 月程式與技術動態

### 前端開發

**jQuery 1.2 持續進化**

jQuery 1.2 帶來了重要的效能改進和外掛系統。jQuery 迅速成為最受歡迎的 JavaScript 框架。

**Prototype 和 Scriptaculous**

Prototype 是另一個流行的 JavaScript 框架，Scriptaculous 提供視覺效果庫。

**Dojo Toolkit**

Dojo 提供完整的 UI 元件和離線支援。

### 瀏覽器

**Chrome 開發衝刺**

Google Chrome 積極開發中，採用 V8 JavaScript 引擎。將於 2008 年 9 月正式發布。

**Firefox 3 下載日**

Mozilla 舉辦 Firefox 3 下載日，推動瀏覽器升級。

**IE8 發布測試**

Internet Explorer 8 開始提供測試版。

### 程式語言

**ECMAScript 4 討論**

JavaScript 未來版本的方向持續討論中。

**ActionScript 3**

Flash 和 Flex 採用 ActionScript 3，提供更強大的RIA 開發能力。

### Ajax 與互動

**Web 2.0 API 丰富**

各平台開放 API，催生大量 Ajax 應用。

**JSON 作為資料交換格式**

JSON 取代 XML 成為 Ajax 應用的首選資料格式。

### 開發工具

**Firebug 广泛使用**

Firebug 成為前端開發者的必備工具。

**Chrome DevTools 開發中**

Chrome 內建開發者工具開始開發。

### 業界動態

- **Google 收購 DoubleClick**：線上廣告市場整合
- **YouTube 影片平台**：使用者快速成長
- **Twitter API 開放**：微博平台開始生態建設
