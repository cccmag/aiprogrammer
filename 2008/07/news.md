# 本月新知

## 2008 年 7 月程式與技術動態

### SaaS 發展

**Salesforce 持續增長**

Salesforce 作為 SaaS CRM 的先驅，2008 年持續擴大市場份額。其 Force.com 平台吸引更多開發者。

**Google Apps 普及**

Google Apps 企業版提供郵件、文件、行事曆等雲端服務，越來越多企業採用。

**SaaS 多元化**

從 CRM 到 HR、財務，各種軟體服務都開始雲端化。

### 雲端運算

**Amazon AWS 服務增加**

S3、EC2、SimpleDB、SQS 等服務持續改進，奠定雲端運算標準。

**Windows Azure 開發中**

Microsoft 積極開發 Azure 平台，進入雲端市場。

**雲端安全關注**

雲端安全和合規成為企業採用雲端服務的重要考量。

### 程式語言

**Python 2.6 開發中**

Python 持續演進，適用於雲端應用開發。

**Java 雲端框架**

Spring、Play 等框架開始支援雲端部署。

### AI 與機器學習

**雲端 ML 服務**

Amazon ML、Google Prediction API 等雲端 ML 服務開始出現。

**推薦系統成熟**

電子商務平台的推薦系統持續優化。

### 業界動態

- **Yahoo! 收購 Maven**：強化程式碼托管
- **Sun 加入 OpenOffice.org**：開源辦公室軟體
- **Facebook 使用 Hadoop**：大資料處理
