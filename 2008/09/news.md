# 本月新知

## 2008 年 9 月程式與技術動態

### 版本控制

**GitHub 成長驚人**

GitHub 於 2008 年 2 月上線，9 月已有數萬個 repository。社交程式設計的概念開始流行。

**Git 1.6 發布**

Git 1.6 帶來了重要的改進，包括更好的分支管理和效能優化。

**分散式版本控制普及**

Google、Microsoft、Twitter 等公司開始採用 Git 或分散式版本控制系統。

### 瀏覽器

**Google Chrome 發布**

2008 年 9 月 2 日，Google 正式發布 Chrome 瀏覽器，採用 V8 JavaScript 引擎。

**瀏覽器市場競爭加劇**

Chrome 的加入使瀏覽器市場競爭更加激烈。

### 程式語言

**Python 2.6 發布**

Python 2.6 帶來了更多現代化特性。

**Rails 2.2**

Rails 2.2 增加了執行緒安全和 i18n 支援。

### AI 與機器學習

**深度學習研究進展**

Geoffrey Hinton 等研究者持續推進深度學習研究。

**雲端 ML 服務**

Amazon、Google 提供更多 ML API。

### 業界動態

- **Google 發布 Chrome**：瀏覽器市場大地震
- **Facebook 開放 Platform API**：生態系統持續擴大
- **Twitter API 廣泛使用**：第三方用戶端百花齊放
