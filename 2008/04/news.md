# 本月新知

## 2008 年 4 月程式與技術動態

### 巨量資料與分散式運算

**Hadoop 0.18 開發中**

Apache Hadoop 是處理大規模資料集的開源框架。2008 年，Hadoop 生態系統持續壯大，Pig、Hive、HBase 等工具相繼出現，為巨量資料處理提供了完整的解決方案。

**Google MapReduce 論文影響深遠**

Google 發表的 MapReduce 論文成為分散式運算的經典文献。越來越多的公司和組織開始採用 MapReduce 架構處理大規模資料。

**雲端運算服務普及**

Amazon Web Services 持續擴展，EC2、S3、SimpleDB 等服務被廣泛採用。Yahoo、Facebook 等公司也在建設自己的巨量資料處理基礎設施。

### 程式語言與框架

**Python 2.6 持續發展**

Python 社群持續改進 2.6 版本，同時 Python 3.0 的開發也在進行中。 Django Web 框架持續獲得關注。

**Ruby on Rails 2.1 發布**

Rails 2.1 帶來了時區支援、垃圾回收改進等新特性。Ruby 語言在 Web 開發領域的影響力持續擴大。

**JavaScript AJAX 框架火熱**

jQuery、Prototype、Dojo 等 JavaScript 框架讓 Ajax 開發變得更加簡單。前端開發現代化加速。

### 資料庫技術

**NoSQL 運動興起**

BigTable、Dynamo、CouchDB 等文件資料庫開始受到關注。傳統關聯式資料庫在處理巨量資料時面臨挑戰，NoSQL 成為新選擇。

**MongoDB 早期發展**

文件導向資料庫 MongoDB 開始獲得開發者社群關注。其灵活的 JSON 式文件儲存吸引了許多應用。

**Cassandra 由 Facebook 開源**

Facebook 將 Cassandra 專案貢獻給 Apache 基金會。這是一個分散式、非關聯式的資料庫系統。

### AI 與機器學習

**深度學習概念萌芽**

Geoffrey Hinton 等研究者的深度學習研究開始受到關注。Boltzmann Machine 和類神經網路的深度架構成為熱門研究方向。

**Netflix 競賽持續**

Netflix Prize 競賽進入關鍵階段，參賽團隊持續最佳化推薦演算法。這推動了協同過濾技術的發展。

**雲端機器學習服務**

Amazon、Google 開始提供機器學習雲端服務，讓更多開發者能夠使用 ML 技術。

### 瀏覽器與網頁技術

**Chrome 瀏覽器即將發布**

Google 宣布開發 Chrome 瀏覽器，採用 V8 JavaScript 引擎。2008 年 9 月將正式發布。

**Web 2.0 應用持續爆發**

社交網路、微博客、雲端儲存等 Web 2.0 應用持續增長。RESTful API 成為標準。

**HTML 5 草案制定中**

W3C 持續制定 HTML 5 標準，Canvas、Web Storage、WebSocket 等新 API 浮現。

### 業界動態

- **Facebook 成長迅速**：使用者突破 1 億大關
- **Twitter 用戶增加**：微博服務開始流行
- **Google 收購 DoubleClick**：線上廣告市場整合
- **微軟收購 aQuantive**：加強線上廣告業務

### 開源專案動態

- **Android SDK 1.0 即將發布**：預計 2008 年第三季度
- **GitHub 成長驚人**：程式碼托管平台使用者增加
- **Linux 核心 2.6.25 發布**：更多驅動程式支援
