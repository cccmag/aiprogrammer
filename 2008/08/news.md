# 本月新知

## 2008 年 8 月程式與技術動態

### 效能優化

**YSlow 發布**

Yahoo 發布 YSlow 效能分析工具，基於 Yahoo! 的效能法則，幫助開發者分析和最佳化網站效能。

**前端效能標準**

Steve Souders 的《High Performance Web Sites》一書出版，系統化了前端效能優化的最佳實踐。

**HTTP 加速**

Varnish、Web加速器等工具開始普及。

### 網路技術

**HTTP 持續改進**

HTTP/1.1 持續優化，Pipeline 等特性開始被採用。

**CDN 普及**

Akamai、Cloudflare 等 CDN 服務持續擴展。

### 瀏覽器

**Chrome 接近發布**

Google Chrome 積極開發中，V8 引擎效能出色。將於 2008 年 9 月正式發布。

**瀏覽器競爭**

Firefox 3 取得成功，IE 面臨競爭壓力。

### 程式語言

**V8 JavaScript 引擎**

Google 發布 V8 JavaScript 引擎，採用即時編譯技術，大幅提升 JavaScript 執行速度。

**效能分析工具**

Firebug、Chrome DevTools 等工具持續改進。

### 雲端服務

**雲端 CDN**

Amazon CloudFront 等 CDN 服務開始提供。

**負載均衡服務**

雲端負載均衡服務開始普及。

### 業界動態

- **Twitter 快速成長**：微博平台需要高效能架構
- **Facebook 達 1 億使用者**：社交網路需要高效能基礎設施
