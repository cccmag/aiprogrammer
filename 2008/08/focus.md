## 濃縮回顧

### 頁面載入時間分解

```
總載入時間 = DNS + TCP + Request + Response + Render
           = 50ms + 100ms + 100ms + 200ms + 150ms
           = 600ms
```

### 效能優化金字塔

```
         ┌─────────────┐
         │  內容優化    │
         ├─────────────┤
         │  伺服器優化  │
         ├─────────────┤
         │  網路優化    │
         ├─────────────┤
         │  瀏覽器優化  │
         └─────────────┘
```

### Yahoo 效能法則（精選）

1. 減少 HTTP 請求
2. 使用 CDN
3. 添加 Expires 頭
4. Gzip 壓縮
5. 將 CSS 放在頂部
6. 將 JS 放在底部
