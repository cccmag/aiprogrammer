# 本月新知

## 2008 年 3 月程式與技術動態

### Python 生態

**Django 0.96 發布**

2008 年 3 月，Django 0.96.1 正式發布。Django 1.0 的穩定版將在 9 月發布，這是一個重要的里程碑。社群對 Django 的關注度持續提高。

**Python 2.5.2 安全更新**

Python 2.5 系列持續獲得安全性更新。Python 2.6 正在開發中，將帶來更多現代化特性。

**Plone 3.0 發布**

Plone 是基於 Zope 的企業級內容管理系統，3.0 版本帶來了更友善的使用者介面和更好的效能。

### Web 開發

**Ruby on Rails 2.0 熱潮**

Rails 的 RESTful 路由和資源導向設計持續影響 Python 框架的發展。Django 和 TurboGears 都在參考 Rails 的設計理念。

**jQuery 1.2.6**

jQuery 持續進化，成為最受歡迎的 JavaScript 框架之一。Python 網頁開發者開始整合 jQuery 到他們的專案中。

**Comet 技術普及**

基於長輪詢的 Comet 技術讓 Web 即時應用變得可行。Django 社群開始討論如何更好地支援即時功能。

### 資料庫

**PostgreSQL 8.3**

2008 年 2 月發布的 PostgreSQL 8.3 帶來了豐富的新功能，包括對全文搜尋的原生支援和 HSTORE 鍵值儲存。

**MySQL 5.1 開發中**

MySQL 5.1 的開發持續進行，將帶來更多企業級功能。

**SQLite 3.5**

SQLite 持續改進，成為嵌入式和測試環境的首選資料庫。

### AI 與機器學習

**scikit-learn 初期發展**

scikit-learn（前身為 scikits.learn）正在快速发展，成為 Python 機器學習的標準工具。

**NumPy 和 SciPy 成熟**

這兩個基礎庫已經相當成熟，支撐著整個 Python 科學計算生態。

**Hadoop 和 MapReduce 熱潮**

雲端運算和大資料處理的概念持續升溫，Python 開始整合這些技術。

### 業界動態

- **Google App Engine 預覽版**：Python 支援是亮點之一
- **GitHub 成立**：2008 年 4 月，Ruby 平台的 GitHub 成立
- **Facebook PHP 成功**：PHP 作為 Web 開發語言仍然主流
- **微軟 ASP.NET MVC**：.NET 平台也有了 MVC 框架

### 標準與規範

- **WSGI 持續演進**：PEP 3333 提案討論中
- **Python 3.0 準備**：Python 3000 即將來臨
- **HTML 5 草案**：持續制定中

### 值得關注的開源專案

- **Django**：Python Web 框架旗艦
- **Pylons**：靈活的 WSGI 框架
- **TurboGears**：多框架整合
- **Zope**：老牌物件資料庫 CMS
