# 智慧手機市佔率變化

## 2010 年智慧手機市場

### 第一季統計

```
全球智慧手機 OS 市佔率（2010 Q1）：
───────────────────────────
Symbian：       44.4%（Nokia 主導）
BlackBerry：    19.9%（RIM）
iPhone OS：     17.8%（Apple）
Android：       9.6%（快速成長）
Windows Mobile： 6.0%（微軟）
其他：          2.3%
```

### 趨勢分析

```
市場趨勢（2010 年）：
───────────────────────────
贏家：         Android、iOS
輸家：         Symbian、Windows Mobile
黑馬：         Android（年底達 30%）
```

## 主要平台表現

### Symbian 的困境

```
Symbian 市佔變化：
───────────────────────────
2009 Q4：      47.0%
2010 Q1：      44.4%（下滑）
原因：         iPhone 和 Android 競爭
Nokia 回應：   等待 Symbian^3
```

### iPhone 的穩健

```
iPhone OS 市佔變化：
───────────────────────────
2009 Q4：      16.2%
2010 Q1：      17.8%（上升）
原因：         iPhone 3GS 成功、iPad 發布
區域：         美國市場領先
```

### Android 的爆發

```
Android 市佔變化：
───────────────────────────
2009 Q4：      5.0%
2010 Q1：      9.6%（幾乎翻倍）
成長率：       +92%（年增）
原因：         多廠商支持、開放平台
```

## 區域差異

### 美國市場

```
美國智慧手機市佔（2010 Q1）：
───────────────────────────
iPhone：        28%
BlackBerry：    35%

```

```
比較：

美國 BlackBerry 強勢，全球一般

Android 全球快速滲透
```

## 開發者視角

### 營收比較

```
平台營收能力（2010 年）：
───────────────────────────
App Store：     營收最高，平均售價 $2.50
Android Market：成長快但營收低，平均 $3.50
BlackBerry：    App World 營收有限
Windows Mobile：市佔低、營收更低
```

### 建議策略

```
開發平台選擇建議（2010 年）：
───────────────────────────
iOS：          優先考慮，營收高
Android：      重要第二平台，市佔成長快
Symbian：      特定市場可考慮
BlackBerry：    企業市場可考慮
Windows Mobile：可觀望
```

---

## 結論

2010 年 Q1 的市佔資料顯示 Android 的崛起已不可阻擋。從 5% 到 30%+ 的成長只是時間問題。Symbian 和 Windows Mobile 將持續下滑，iPhone 將保持高價值市場的領先地位。

對於開發者來說，多平台策略越來越重要，但 iOS 和 Android 將是首選。
