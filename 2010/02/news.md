# 本月新知

## 2010 年 2 月程式技術動態

### Android 生態系

**Android 2.1 發布**

Google 在 2010 年 1 月發布 Android 2.1，2 月成為主流版本：

```
Android 2.1 (Eclair) 主要特性：
───────────────────────────────
藍牙 A2DP：    更好的藍牙音訊支援
多點觸控：     支援多點觸控螢幕
Wallpaper：   即時桌布 API
VPN：         改進的 VPN 支援
效能優化：     啟動速度、記憶體管理
```

**Nexus One 上市**

Google 在 1 月發布 Nexus One，2 月持續熱銷：

```
Nexus One 規格：
──────────────────
螢幕：      3.7" AMOLED 800x480
處理器：    Qualcomm QSD8250 1GHz
記憶體：    512MB RAM / 512MB ROM
相機：      500萬畫素自動對焦
系統：      Android 2.1（可升級 2.2）
價格：      $529（無約）
```
