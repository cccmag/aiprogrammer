# 本月新知

## 2010 年 1 月程式技術動態

### 平板電腦熱潮

**CES 2010 平板電腦大爆發**

2010 年 1 月的 CES 消費電子展成為平板電腦的舞台。多家廠商展示了他們的平板裝置：

- **Apple iPad**（預計 4 月發售）- 9.7 吋多點觸控螢幕，A4 處理器
- **Dell Streak** - 5 吋迷你平板，兼具手機功能
- **Pegatron** - 多款白牌平板裝置
- **Archos** - Android 平板系列

```
平板電腦規格對比：
─────────────────────
iPad:     9.7" IPS, 1024x768, iOS
Dell:     5"   LCD,  800x480, Android
Archos:   7"   LCD,  800x480, Android
```

**iPad 發表前的市場預測**

分析師普遍看好平板市場：

- 預測 2010 年平板出貨量將達 1,000 萬台
- 企業市場將是主要成長區塊
- 教育市場潛力巨大

### 瀏覽器與 Web 技術

**Chrome OS 正式發布**

Google 在 2009 年 12 月發布 Chrome OS 原始碼，2010 年 1 月持續引發關注：

- 純 Web 應用架構
- 快速開機設計
- 雲端運算優先
- 開源 Chromium OS 計畫

**Internet Explorer 9 開發中**

微軟預覽 IE9，將支援：

- HTML5 標準
- CSS3 發展中的功能
- 硬體加速渲染
- 更好的效能

### Adobe Flash 與 HTML5 之爭

**Flash Player 10.1 發布**

Adobe 發布 Flash Player 10.1 for Mobile：

- 支援 Android、WebOS、Symbian
- 硬體加速播放
- 低功耗設計
- 瞄準行動裝置市場

```
Flash vs HTML5 爭議：
────────────────────
Flash:  豐富的生態系、相容性佳、支援視訊
HTML5:  開放標準、原生支援、跨平台
```

### 開發工具進展

**Xcode 3.6 發布**

Apple 發布 Xcode 3.6，包含：

- iPad 模擬器支援

Eclipse Foundation 發布 Helios (3.6) 開發版：

- Git 支援整合

- 多語言支援強化

### 程式語言動態

**Python 3.2 alpha 發布**

Python 3.2 alpha 版本帶來：

- asyncio 模組藍圖
- 使用者目錄套件支援
- 改進的 HTML 解析器

Scala 2.8 持續開發中，強調：

Git 1.7 帶來多項改進：

- git submodules 改進
