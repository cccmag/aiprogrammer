# 智慧手機市佔率變化

## 2010 年智慧手機市場

2010 年是智慧手機市場爆發的一年，iPhone 和 Android 的競爭，改變了整個手機產業。

```
智慧手機市場（2010年第一季）：
──────────────────────────────
全球市佔：
 iPhone OS (iOS)：   16%
 Android：          5%（快速成長中）
 BlackBerry：       20%
 Symbian (Nokia)：  47%
 Windows Mobile：   6%
```

## 主要平台演變

### 市場份額變化

```
智慧手機 OS 市佔率變化（2009-2010）：
────────────────────────────────────
Symbian：    2009: 52% → 2010: 44%（下滑）
BlackBerry： 2009: 20% → 2010: 19%（穩定）
iOS：        2009: 15% → 2010: 18%（上升）
Android：    2009: 4%  → 2010: 10%（爆發成長）
Windows：    2009: 9%  → 2010: 6%（下滑）
```

### 成長率比較

```
各平台年成長率（2009-2010）：
──────────────────────────────
Android：    +150%（爆發成長）
iOS：        +60%
Symbian：    -10%（持續下滑）
BlackBerry： +10%
Windows：    -30%
```

## Nokia 的困境

### Symbian 的衰落

```
Nokia 的挑戰：
──────────────────
市場份額：   持續下滑
品牌形象：   被視為過時
iPhone：     被視為更好的選擇
Android：    性價比更高

```

### Nokia 的回應

```
Nokia 的策略調整：
──────────────────

2010:  Symbian^3 發布
```

## BlackBerry 的挑戰

### BlackBerry 的問題

## 開發者生態系

### App 開發平台比較

```
各平台開發吸引力（2010年）：
───────────────────────────
iOS：
  付費意願：   高（平均 $2.50/應用）
  用戶品質：   高
  開發工具：   Xcode（Mac 專屬）
  審核：       需要通過 App Store 審核

Android：
  市佔成長：   快速
  付費意願：   低（偏好免費+廣告）
  開發工具：   Eclipse（跨平台）
  審核：       快速，審核較鬆

Windows Phone：
  市佔：       起步階段
  付費意願：   不明
  開發工具：   Visual Studio
  機會：       應用較少，曝光容易
```

---

## 結論

2010 年是智慧手機市場的轉捩點。Symbian 的衰落、Nokia 的困境開始顯現，iPhone 持續成長，而 Android 以爆發性的速度崛起。

最終的贏家尚未確定，但市場趨勢已經明確：開放的 Android 將獲得最大市佔，而 iOS 將在高端市場保持領先。Nokia 和 BlackBerry 將面臨艱難的轉型。
