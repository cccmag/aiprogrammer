# 回顧與結語

## 2015 年 3 月號結語

### 本月主題回顧

三月是資料庫技術的豐收月份。本期雜誌全面介紹了關聯式資料庫的基礎知識和主流資料庫系統的特性。

#### SQL 的普遍性

SQL 是資料庫的通用語言。無論是 PostgreSQL、MySQL 還是 SQLite，基本的 SQL 語法都是相通的。學會 SQL，就能輕鬆在各主流資料庫之間遷移。

#### 三大開源資料庫的定位

PostgreSQL 以功能完整著稱，適合需要進階 SQL 功能的企業應用；MySQL 以效能取勝，是 Web 應用的首選；SQLite 以輕量取勝，統治了行動裝置和嵌入式系統。

#### NoSQL 帶來的思考

NoSQL 的崛起提醒我們，沒有萬能的資料庫。根據應用需求選擇合適的工具，才是正確的做法。

### 未來展望

**混合式儲存**：未來的應用可能同時使用 SQL 進行交易處理、NoSQL 進行快速查詢。

**雲端優先**：越來越多的應用直接採用雲端資料庫服務，減少維運負擔。

**智慧化管理**：AI 將輔助資料庫管理，自動進行效能調優和異常偵測。
