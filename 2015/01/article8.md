# GitHub 獲得 AMD 策略投資

## 前言

2015 年，GitHub 獲得了 AMD 的策略投資，這標誌著開源運動在硬體領域的影響力不斷擴大。

## 投資背景

```
GitHub 融資歷史：
─────────────────
2012: A 輪 1 億美元
2014: B 輪 3.5 億美元
2015: AMD 策略投資
```

## 開源運動的影響

- GitHub 突破 500 萬開發者
- 大量企業開源專案
- 開放程式碼成為主流

## 對台灣的影響

台灣的開源社群也在 2015 年快速成長：
- COSCUP 參與人數創新高
- 更多本土開源專案出現
