# 本月新知

## 2015 年 5 月程式與技術動態

### Python 生態

**Python 3.5 功能凍結**

2015 年 5 月，Python 3.5 完成功能凍結（Feature Freeze），進入錯誤修正階段。預計於 9 月正式發布。這個版本帶來多個重要的新特性：

- **PEP 492**：`async` 和 `await` 語法，大幅簡化非同步程式設計
- **PEP 484**：Type Hints，為靜態類型檢查鋪路
- **PEP 471**：`os.scandir()`，更高效的目錄遍歷
- **矩陣乘法運算子 `@`：使用 `A @ B` 代替 `np.dot(A, B)`

**pip 7.0 發布**

pip 7.0 支援 Python 3.5，並改善了依賴解析演算法。

**Anaconda 3.4 發布**

Continuum Analytics 發布了 Anaconda 3.4，這是一個包含 Python 和科學計算套件的發行版。

### 網頁框架

**Django 1.9 開發中**

Django 1.9 正在積極開發中，將包含多項改進，包括更好的密碼驗證 API。

**Flask 0.10 發布**

Flask 微框架發布 0.10 版本，包含多項改進和錯誤修正。

### JavaScript 生態

**npm 3.0 測試版**

Node.js 的套件管理器 npm 3.0 進入測試階段，採用扁平的 `node_modules` 結構，解決了諸多依賴地獄問題。

**Babel 6.0 發布**

JavaScript 編譯器 Babel 6.0 正式發布，採用更模組化的架構。

### 開發工具

**Git 2.4 發布**

Git 2.4 帶來了 `git --force-with-lease` 等安全性改進。

**Visual Studio Code 發布**

Microsoft 發布了 Visual Studio Code 跨平台程式碼編輯器，支援 Python 擴展。

### 雲端與部署

**Docker 1.6 發布**

Docker 1.6 增強了 Dockerfile 和日誌驅動程式的功能。

**AWS Lambda 支援 Python**

AWS Lambda 增加對 Python 的支援，開發者可以使用 Python 編寫無伺服器函數。

- **GitHub 完成 10 億美元融資**：估值飆升至 20 億美元
- **Stack Overflow 用戶突破 500 萬**：全球最大的程式問答社區
- **Microsoft 收購忘憂樹**：Coding 編輯器（WPF 的前身）傳聞
- **Swift 語言開源傳言**：社群熱烈討論 Apple Swift 的開源可能性

- **HTTP/2 協議推進**：預計 2015 年底正式批准
- **ECMAScript 2015 定稿**：ES6 語法標準即將發布
- **HTML5 正式推薦**：W3C 宣佈 HTML5 作為正式標準
