# 本月新知

## 2015 年 6 月程式與技術動態

### C 語言生態

**C11 標準持續採用**

C11 標準（ISO/IEC 9899:2011）於 2011 年發布後，2015 年越來越多的編譯器開始完整支援。C11 引入的新特性包括：
- 多執行緒支援（`_Thread_local`, `<threads.h>`）
- 泛型宏（`_Generic`）
- 匿名結構和聯合
- `_Alignas` 和 `_Alignof`
- 邊界檢查介面（可選）

**GCC 5.0 發布**

GCC 5.0 帶來了多项改進，包括更好的 C++11/14 支援和最佳化。

**Clang 3.6 發布**

Clang 持續改進對 C11 和 C++11/14 標準的支援。

### Unix 系統

**Linux Kernel 4.0 發布**

Linux 4.0 帶來了重要的更新，包括 live patching 支援和更好的容器隔離。

**FreeBSD 10.2 開發中**

FreeBSD 持續改进，作為伺服器作業系統的選擇之一。

### 開發工具

**Git 2.4 發布**

增加了 `git --force-with-lease` 等安全性改進。

**LLDB 持續改進**

LLVM 的除錯器 LLDB 持續獲得改善，成為 GDB 外的另一個選擇。

### 雲端與部署

**Docker 1.6 持續普及**

容器化技術在 2015 年繼續獲得關注。

**AWS Lambda 支援**

AWS 的無伺服器運算服務為系統程式設計提供了新的部署模式。

### 業界動態

- **Apple 推出 Swift 2.0**：Swift 語言持續演進
- **Google 開源 TensorFlow**：深度學習框架改變 AI 版圖
- **微軟成立 .NET 基金會**：.NET 生態更加開放
- **IBM 收購 Red Hat**：影響伺服器領域格局

### 標準化進展

- **C17 開始討論**：C11 的小幅改進版
- **C++17 進展**：新特性持續提出
- **HTTP/2 標準化完成**：網路通訊新標準
