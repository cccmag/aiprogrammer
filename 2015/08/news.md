# 2015 年 08 月號：本月新知

## Linux Kernel 4.2 發布

2015 年 8 月，Linux Kernel 4.2 正式發布，這是 2015 年的重要版本更新。

### 主要新特性

- **新增 FGK 司機**：全新的「Freedreno」GPU 驅動程式
- **Btrfs 改進**：增強的快照和壓縮功能
- **記憶體管理優化**：更好的記憶體回收機制
- **新增架構支援**：更多 ARM 變體支援

詳細資訊請參閱 [Linux Kernel 4.2 Release Notes](https://www.google.com/search?q=Linux+kernel+4.2+release+notes)

## systemd 爭議持續

systemd 作為 Linux 初始化系統的爭議在 2015 年持續發酵。

### 支持者觀點

- 標準化的服務管理介面
- 自動化相依處理
- 更好的日誌管理（journald）

### 反對者觀點

- 系統復雜度增加
- 偏離 Unix 設計哲學
- 單一故障點風險

[搜尋 systemd controversy 2015](https://www.google.com/search?q=systemd+controversy+2015)

## 容器技術演進

### Docker 1.8 發布

Docker 1.8 帶來多項改進：

- **Docker Toolbox**：一站式安裝體驗
- **Content Trust**：映像檔簽章驗證
- **更快的 Registry 操作**

### rkt 容器引擎

CoreOS 發布 rkt（發音為「rocket」），作為 Docker 的替代方案，強調安全性和標準相容性。
