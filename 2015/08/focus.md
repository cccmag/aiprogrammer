## 為什麼要學 Linux 系統管理？

- **伺服器**：超過 90% 的網頁伺服器運行 Linux
- **雲端**：AWS、Azure、GCP 的主要作業系統
- **開發**：大多數開發工具原生支援 Linux
- **物聯網**：Raspberry Pi 等裝置使用 Linux
