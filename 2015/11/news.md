# 本月新知

## 2015 年 11 月嵌入式系統與物聯網動態

### 硬體發展

**Arduino 與 GitHub 整合**

Arduino 在本月宣布與 GitHub 深度整合，開發者可以直接在 Arduino IDE 中瀏覽、克隆和分享社群專案。這項整合簡化了硬體專案的協作流程，讓更多開發者能夠參與開源硬體專案。

同時 Arduino.cc 發布了 Arduino Create 平台，提供雲端編譯和遠端部署功能。這是 Arduino 向雲端開發工具邁進的重要一步。

**Raspberry Pi 推出 Compute Module 3**

Raspberry Pi 基金會發布了 Compute Module 3（CM3），這是一款專為工業應用設計的模組化運算解決方案。CM3 採用與 Raspberry Pi 3 相同的 BCM2837 處理器，但採用 DDR2 SODIMM 封裝，方便整合到嵌入式系統中。

**Intel Edison 停產**

Intel 宣布停產 Edison 模組，這款曾經被寄予厚望的物聯網運算平台在推出不到兩年後就走向終點。Intel 表示將專注於 Curie 和其他物聯網解決方案。

### 物聯網標準

**OCF 統一物聯網標準**

Open Connectivity Foundation（OCF）在本月發布了 IoTivity 1.0，這是一個開源的物聯網互操作性框架。IoTivity 實現了 OCF 規範，旨在解決智慧家庭裝置之間的互通性問題。

主要參與者包括 Intel、Samsung、LG、Electrolux 等。這是物聯網標準統一的重要一步。

**Thread 網路協定正式發布**

Google 旗下的 Nest 宣布 Thread 網路協定正式可用。Thread 是一種基於 IEEE 802.15.4 的低功耗網狀網路協定，專為智慧家庭裝置設計。與 ZigBee 和 Z-Wave 相比，Thread 具有更好的 IP 整合和更容易的設定流程。

### 開發工具

**mbed OS 5 發布**

ARM 發布了 mbed OS 5，這是專為 ARM Cortex-M 處理器設計的物聯網作業系統。新版本帶來了：

- 完整的 RTOS 功能
- 更好的電源管理
- 支援 ARMv8-M 架構
- 更簡單的 API

**PlatformIO 3.0**

開源的物聯網開發平台 PlatformIO 發布了 3.0 版本，新增了支援 800 多種開發板的能力。PlatformIO 支援 Arduino、mbed、ESP8266 等多種框架，大幅簡化了跨平台的嵌入式開發。

### 產業應用

**智慧城市在亞洲崛起**

多個亞洲城市在 11 月宣布了智慧城市計畫：

- **新加坡**：宣布智慧國家計畫 2.0，涵蓋交通、醫療、能源
- **東京**：2020 奧運智慧城市計畫
- **首爾**：智慧交通系統升級

**工業物聯網成長**

IBM、Bosch、Siemens 等大廠持續投資工業物聯網（IIoT）。根據報告，2015 年工業物聯網市場規模達到約 110 億美元，預計 2020 年將達到 250 億美元。

### 開源專案

**NodeMCU 1.0 發布**

NodeMCU 是一個基於 ESP8266 的開源物聯網開發平台，本月發布了 1.0 版本。NodeMCU 使用 Lua 腳本語言，大幅簡化了 ESP8266 的開發流程。

**Home Assistant 發布**

開源的智慧家庭平台 Home Assistant 持續獲得關注，成為 Raspberry Pi 上最熱門的智慧家庭軟體之一。

### 重要事件

**中國物聯網博覽會**

2015 年中國國際物聯網博覽會在上海舉行，展示了最新的物聯網技術和應用。中國作為全球最大的物聯網市場之一，其發展方向受到全球關注。
