# 2015 年 07 月號：本月新知

## Git 2.4 發布

2015 年 5 月，Git 2.4 正式發布，這是截至當時最具影響力的版本更新之一。主要新特性包括：

- **HTTPS 傳輸優化**：智慧型 HTTP 傳輸協議，大幅提升克隆和拉取速度
- **Git worktree 支援**：可以同時檢出同一個倉庫的多個分支
- **互動式 rebase 改進**：`git rebase -i` 現在支援更好的衝突解決流程
- **`git push --force-with-lease`**：更安全的強制推送選項，避免覆蓋他人的工作

詳細資訊請參閱 [Git 2.4 Release Notes](https://github.com/git/git/blob/master/Documentation/RelNotes/2.4.0.txt)
