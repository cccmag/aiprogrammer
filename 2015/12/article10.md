# 開源硬體回顧與展望

## 2015 年發展

開源硬體運動在 2015 年持續發展：

- **Arduino**：生態完善
- **Raspberry Pi**：計算模組 3 發布
- ** BeagleBone**：Black 繼續支援
- **BBC micro:bit**：教育市場

## Arduino 生態

### 重要更新

- Arduino Create 雲端平台
- Arduino 1.6.x IDE
- 更廣泛的 Shield 相容性

### 應用領域

| 領域 | 應用 |
|------|------|
| 教育 | 程式設計入門 |
| 嗜好 | 機器人、IoT |
| 原型 | 快速開發 |
| 生產 | 小批量製造 |

## Raspberry Pi

### Compute Module 3

- **處理器**：BCM2837
- **記憶體**：1GB LPDDR2
- **用途**：工業應用

### 軟體生態

- **Raspbian**：官方系統
- **NOOBS**：簡易安裝
- **Pi Store**：應用商店

## BBC micro:bit

### 發布

BBC micro:bit 在 2015 年發布：

- 專為英國兒童教育設計
- ARM mbed 平台
- 藍牙、加速計、LED 矩陣

## 開源硬體協會

### OHA 共識

開源硬體協會（Open Hardware Association）定義了開源硬體標準：

- **硬體文件**：設計檔可取得
- **元件**：可取得的元件
- **衍生**：允許衍生設計

## 未來展望

### 2016 年趨勢

1. **更多 SBC**：低成本單板電腦
2. **IoT 整合**：WiFi/BLE 內建
3. **Edge AI**：邊緣運算能力
4. **教育普及**：更多學校採用

## 小結

開源硬體降低了創新門檻，讓更多人參與硬體開發。
