# 本月新知

## 2015 年 12 月：年度回顧

### 2015 年十大技術事件

#### 1. Swift 開放源代碼

Apple 在 12 月宣布 Swift 正式開源，這是 2015 年最重磅的新聞之一。Swift 現在可以在 Linux 上運行，並且允許開發者社群參與語言的發展。Swift 的開源被視為 Apple 對開發者社群的重要姿態，也為 Swift 在伺服器端開發的應用打開了大門。

#### 2. React 0.14 發布

Facebook 在 10 月發布了 React 0.14，這是 React 史上最重要的更新之一。React 0.14 將 DOM 相關功能分離到 ReactDOM 套件中，引入了無狀態函數元件，大幅簡化了元件的開發。

#### 3. Docker 1.9 網路革新

Docker 1.9 在 11 月發布，引入了全新的網路功能。Docker Swarm 支援多主機網路，解決了容器之間跨主機通訊的問題。同時，Docker 正式推出了 Docker Store 和 Docker Cloud 服務。

#### 4. GitHub 達到 10 百万用戶

GitHub 在 2015 年達到了 10 百万用戶的里程碑，這是開源軟體發展的重要象徵。GitHub 也發布了 Enterprise 2.0 版本，引入了更多企業級功能。

#### 5. Angular 2 持續開發

Google 持續推進 Angular 2 的開發，多個 Alpha 版本發布。Angular 2 採用了全新的架構設計，支援 TypeScript，引入了行動優先的開發理念。

#### 6. Node.js 4.0 LTS

Node.js 4.0 在 2015 年成為 LTS（長期支援）版本，這是 Node.js 基金會成立後的第一個 LTS 版本。Node.js 4.0 整合了 io.js 的功能，提供了更穩定的執行環境。

#### 7. 微服務成為主流

2015 年，微服務架構從早期採用者的實驗走向主流。Docker 和容器技術的成熟使得微服務的部署更加簡單。多家大型科技公司分享了他們的微服務實踐經驗。

#### 8. 物聯網標準進展

多個物聯網標準在 2015 年取得進展：OCF 發布 IoTivity、Thread 網路協定正式可用。這為未來物聯網裝置的互通性奠定了基礎。

#### 9. 機器學習火熱

2015 年是機器學習爆發的一年。Google 開源了 TensorFlow，這是 Google 內部使用的機器學習框架。Microsoft 推出了 Azure Machine Learning。深度學習的應用場景不斷擴展。

#### 10. 雲端運算持續增長

雲端運算市場持續高速增長。AWS 推出了更多服務，Azure 追趕勢頭明顯，Google Cloud Platform 在某些領域取得了領先。雲端運算已經成為企業 IT 的標準配置。
