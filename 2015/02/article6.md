# JavaScript 在伺服端的崛起

## 前言

Node.js 的出現改變了 JavaScript 的版圖，讓 JS 成為真正的全端語言。

## 統一開發體驗

```
傳統開發：              Node.js 開發：
───────────            ────────────
前端：JavaScript        統一：JavaScript
後端：Java/C#/PHP       後端：Node.js
資料庫：SQL             資料庫：JS（MongoDB）

優點：
- 程式碼可共享
- 學習曲線降低
- 生態系龐大
```

## 事件驅動的優勢

Node.js 的非阻塞 I/O 適合 I/O 密集型應用，這使得它特別適合：

- 即時應用（聊天、遊戲）
- API 服務
- 微服務
- IoT 後端
