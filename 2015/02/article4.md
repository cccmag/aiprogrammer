# MEAN 技術棧完全指南

## 前言

MEAN（MongoDB, Express, Angular, Node.js）是全端 JavaScript 開發的熱門組合。

## 技術棧組成

```
MEAN 架構：
───────────

前端 (Angular)
    ↓ HTTP/REST
後端 (Express + Node.js)
    ↓
資料庫 (MongoDB)
```

## MongoDB

文件導向資料庫，儲存 JSON 格式資料。

## Express

Node.js Web 框架，處理 HTTP 請求。

## Angular

前端框架，雙向資料綁定。

## Node.js

JavaScript 執行環境，伺服器端。
