# Node.js 在企業的採用

## 前言

2015 年，越來越多大型企業採用 Node.js 作為後端開發平台。

## 知名案例

### Netflix

```
Netflix Node.js 採用：
──────────────────────
- 用於 UI 層渲染
- 加快首頁載入速度 50%
- 程式碼共享（前後端）
```

### PayPal

```
PayPal Node.js 採用：
────────────────────
- 從 Java 遷移到 Node.js
- 減少 33% 的程式碼
- 加快開發速度
```

### Walmart

```
Walmart Node.js 採用：
─────────────────────
- 行動端 API
- 高併發處理
- 黑色星期五購物季支撐
```
