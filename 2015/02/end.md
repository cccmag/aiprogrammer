# 回顧與結語

## 2015 年 2 月號結語

### 本月主題回顧

二月是 Node.js 生態系蓬勃發展的月份。本期雜誌聚焦了伺服端 JavaScript 開發的各個面向，從 Node.js 核心到 Express 框架，從資料庫整合到部署維運。

#### Node.js 的優勢

Node.js 的事件驅動和非阻塞 I/O 讓它非常適合 I/O 密集型的 Web 應用。統一 JavaScript 語言降低了開發者的學習曲線，讓前後端代码可以共享。

#### Express 的簡潔哲學

Express 的設計哲學是「最小化抽象」。它提供了足夠的工具，但不限制你的選擇。這種彈性讓 Express 成為 Node.js 開發的首選框架。

#### 即時通訊的普及

Socket.IO 讓即時功能的實現變得簡單。聊天室、協作工具、即時通知等應用場景越來越普遍。

### 未來展望

**async/await**：ES2017 的 async/await 將徹底改變 Node.js 的非同步程式設計。

**微服務**：Node.js 的輕量特性非常適合微服務架構。

**無伺服器**：AWS Lambda 等 FaaS 平台讓 Node.js 的部署更加便捷。
