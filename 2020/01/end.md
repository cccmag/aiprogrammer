# 結語

Python 生態系的豐富選擇是雙面刃。2020 年初，我們有 pip、poetry、pipenv、conda 等多種工具可以選擇。了解每種工具的優缺點，才能在正確的場景使用正確的工具。

## 關鍵 takeaways

1. **venv 適合簡單場景**：Python 3.3+ 內建，無需額外安裝
2. **virtualenv 適合進階需求**：支援自訂 Python 版本與更靈活的設定
3. **conda 適合科學計算**：跨語言管理與預編譯套件是最大優勢
4. **Poetry/Pipenv 代表未來**：統一的設定檔與鎖定機制更加現代

## 環境管理的未來

Python Packaging 在 2020 年持續演進。PEP 517/518 已成標準，pyproject.toml 正在取代 setup.py。建議新專案評估使用 Poetry 或 Pipenv，它們提供了更好的開發者體驗。

## 下一步

有了穩定的環境管理基礎，下一步就是選擇合適的開發工具。Jupyter Notebook/Lab 是資料科學家的最愛，VS Code + Pylance 提供了優異的 Python 開發體驗。下一期我們將深入探討 Jupyter 生態系。
