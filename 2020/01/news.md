# 本月科技新聞 — 2020年1月

## Python 3.8.1 安全性更新發布

Python 3.8.1 於 2019 年 12 月發布，修復了多項安全性問題與 bug。2020 年 1 月社群持續推動 Python 3.9 的開發，預計於 10 月發布。Python 3.7 仍是大多數伺服器環境的主流版本，建議新專案可開始評估 Python 3.8 的相容性。

## TensorFlow 2.1 正式發布

Google 於 2020 年 1 月發布 TensorFlow 2.1，這是 TF 2.x 系列的重要更新。2.1 版本將 Keras 作為官方高階 API，並強化了 tf.function 與自動圖執行。對 PyTorch 用戶而言，2.1 的 eager execution 預設開啟大幅降低了學習門檻。

## JupyterLab 1.0 穩定版發布

JupyterLab 在 2019 年 12 月達到 1.0 里程碑，2020 年 1 月社群開始推動企業採用。JupyterLab 的分頁介面、檔案總管與延伸模組系統使其成為資料科學家的首選 IDE。相較於傳統 Notebook，1.0 版本提供了更穩定的開發體驗。

## PyTorch 1.4 正式發布

PyTorch 1.4 於 2019 年 12 月發布，JavaScript 綁定（PyTorch JS）進入公測階段。行動裝置支援（PyTorch Mobile）也持續改進。這些更新讓 PyTorch 生態系從訓練擴展到邊緣部署。2020 年初 PyTorch 在學術界的採用率已超越 TensorFlow。

## Python Packaging 2020 發展方向

Python Packaging Authority (PyPA) 在 2020 年初持續推動 pip 改進。PEP 517/518 已被廣泛採用，pyproject.toml 成為新的標準封裝設定檔。Poetry、Flit 等現代工具開始挑戰傳統的 setup.py 方式。
