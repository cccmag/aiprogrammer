# Few-shot 的限制與挑戰

## 模型越大不等於越好

### 規模法則的極限

2020 年的研究開始揭示規模法則的複雜性：

- **J 曲線**：模型能力並非線性增長
- **湧現能力**：某些能力在規模超過臨界點後突然出現
- **但**：並非所有任務都受益於更大規模

### Benchmark 的選擇至關重要

| 任務類型 | 規模擴大效果 |
|---------|------------|
| 文字補全 | 持續提升 |
| 問答 | 持續提升 |
| 數學推理 | 有限提升 |
| 常識推理 | 瓶頸 |

### 特定任務的失敗案例

GPT-3 在某些簡單任務上表現不佳：

```
輸入："Jump over the dog and run"
任務：識別主詞
GPT-3 有時會錯誤識別
```

---

## 偏見問題

### 訓練資料的偏見

GPT-3 從網路資料中學習，不可避免地繼承了社會偏見：

- 性別偏見：「工程師」被聯想為男性
- 種族偏見：某些群體被負面刻板化
- 文化偏見：西方中心的觀點

### Prompt injection

攻擊者可以透過精心設計的 Prompt 操控模型輸出有害內容。

### 防範措施

- 內容過濾
- 輸出審查
- 人類回饋強化學習（RLHF）

---

## 事實一致性

### 幻覺問題

大型語言模型經常產生「幻覺」——看似流暢但事實上錯誤的陳述。

```
輸入：告訴我陳鍾誠的生日
GPT-3 可能產生錯誤或不存在的日期
```

### 無法訪問即時資訊

GPT-3 的知識截止至訓練時間，無法提供最新資訊。

### 解決方向

- 結合檢索增強生成（RAG）
- 引用來源追蹤
- 不確定時表示不知道

---

## 計算資源

### 訓練成本

訓練 GPT-3 估計需要：
- 460 萬美元（以 0.06$/GPU-hour 計算）
- 數千 GPU 並行訓練數週

### 推理成本

部署大型模型的挑戰：

- 高記憶體需求
- 高延遲
- 高電力和冷卻成本

### 環境影響

大型模型的碳足跡引發關注。

---

## 安全與倫理

### 深度偽造

GPT-3 可能被用於生成誤導性內容：

- 假新聞
- 假冒身份
- 垃圾郵件

### 武器化風險

- 自動化網路攻擊
- 軍事應用
- 宣傳機器

### 監管需求

各國政府開始討論 AI 法規。

---

## 總結

Few-shot learning 是重大突破，但我們必須正視其局限：

1. **技術局限**：幻覺、偏見、資源消耗
2. **安全問題**：深度偽造、武器化
3. **倫理挑戰**：責任歸屬、公平性
