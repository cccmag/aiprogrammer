# 本月新知

## 2020 年 7 月程式與 AI 技術動態

### 程式語言與框架

**Python 3.9 發布：新增 Dict Union 運算子與類型提示改進**

Python 3.9 於 2020 年 7 月正式發布。多項新特性備受關注：
- `dict |= dict` 運算子：合併字典的新語法
- 類型提示泛型：`list[int]`、`dict[str, int]` 等語法
- 字串方法改進：`str.removeprefix()`、`str.removesuffix()`
- 加快了 list.sort() 的速度

**TypeScript 4.0 RC 發布**

微軟於 7 月發布 TypeScript 4.0 的候選版本。重點改進包括：
- 可變參數元組類型
- 標籤元組元素
- 類屬性宣告的短路分配運算子

**Deno 1.2 發布**

Deno 團隊發布 1.2 版本，支援：
- 穩定的 HTTP/2 伺服器 API
- 支援 WebAssembly System Interface (WASI)
- 改進了 npm 模組相容性

### AI 與機器學習

**GPT-3 論文發表：語言模型的新里程碑**

2020 年 7 月，OpenAI 發表了 GPT-3 論文「Language Models are Few-Shot Learners」。GPT-3 擁有 1750 億參數，是史上最大語言模型之一。GPT-3 展示了驚人的少樣本學習能力，僅需少量範例即可完成任務，無需梯度更新。

**PyTorch 1.6 發布**

Meta（原 Facebook）發布 PyTorch 1.6，包含：
- 16 位浮點精度訓練的改進
- 分散式訓練增強
- 新的量化 API

**TensorFlow 2.3 發布**

Google 發布 TensorFlow 2.3，重點改進：
- Keras Functional API 改進
- 新增 `tf.data` 的立即載入功能
- TensorFlow Hub 模型整合增強

**Hugging Face Transformers 3.0**

Hugging Face 發布 Transformers 3.0 版本，支援：
- 40 多種語言的模型
- 新的 token 分詞器
- 改進的 Trainer API

### 深度學習框架動態

**JAX 0.1.55 發布**

Google 的 JAX 框架持續更新，增加了：
- 更好的 TPU 支援
- 新的最佳化器
- 改進的自動微分效能

**ONNX Runtime 1.4 發布**

Microsoft 發布 ONNX Runtime 1.4，加入：
- 新的運算子支援
- 效能最佳化
- CUDA 11 支援

### 開發工具與雲端服務

**GitHub Codespaces 公開測試**

GitHub 宣佈 Codespaces 公開測試，開發者可在瀏覽器中獲得完整的 VS Code 體驗。

**Google Colab 支援 GPU 加速**

Google Colab 推出付費版，提供更強大的 GPU 選項，包括 V100 和 TPU。

### 業界動態

- **OpenAI 開放 GPT-3 API 申請**：2020 年 7 月開放首批 API 訪問
- **亞馬遜收購 Zoox**：自動駕駛領域再添併購
- **NVIDIA 推出 A100 GPU**：資料中心 AI 運算能力大幅提升
- **蘋果發布 Swift 5.3**：Swift 語言，持續演進

### 標準與規範

- **W3C 發布 WebAssembly 規範更新**：推進 Web 效能標準
- **ISO 發布 AI 倫理指南草案**：AI 發展的規範框架
