# NLP 領域的專家系統與深度學習

## 前言

在深度學習席捲 NLP 之前，專家系統長期是主流方法。本文回顧專家系統與深度學習在 NLP 中的較量與融合。

---

## 一、專家系統的時代

### 1.1 專家系統的原理

```
專家系統 = 知識庫 + 推理引擎 + 專家規則
```

### 1.2 經典 NLP 專家系統

| 系統 | 領域 | 方法 |
|------|------|------|
| ELIZA | 對話 | 模式匹配 |
| SHRDLU | 積木世界 | 語法 + 語義 |
| SYSTRAN | 翻譯 | 規則 + 詞典 |
| CYC | 知識圖譜 | 邏輯推理 |

### 1.3 限制

- **規則難以獲取**：需要領域專家耗費大量時間
- **覆蓋範圍有限**：無法處理所有情況
- **難以擴展**：新領域需要從頭建立規則

---

## 二、深度學習的崛起

### 2.1 轉折點：Word2Vec 與神經網路

2013 年，Mikolov 等人提出 Word2Vec，開啟了 NLP 的深度學習時代。

### 2.2 關鍵突破時間線

| 年份 | 突破 | 影響 |
|------|------|------|
| 2013 | Word2Vec | 詞表示學習 |
| 2014 | Seq2Seq | 序列到序列 |
| 2015 | Attention | 長距離依賴 |
| 2017 | Transformer | 並行訓練 |
| 2018 | BERT/ELMo | 上下文表示 |
| 2019 | GPT-2 | 大型預訓練 |
| 2020 | GPT-3 | Few-shot Learning |

### 2.3 深度學習的優勢

1. **端到端學習**：無需手動特徵工程
2. **自動學習表示**：從原始資料學習
3. **可擴展性**：可以利用大規模資料

---

## 三、兩種方法的比較

### 3.1 知識表示

| 方面 | 專家系統 | 深度學習 |
|------|---------|---------|
| 知識形式 | 符號規則 | 向量表示 |
| 獲取方式 | 手動定義 | 自動學習 |
| 可解釋性 | 高 | 低 |

### 3.2 推理能力

| 方面 | 專家系統 | 深度學習 |
|------|---------|---------|
| 邏輯推理 | 強 | 弱 |
| 模式識別 | 弱 | 強 |
| 泛化能力 | 有限 | 強 |

### 3.3 資料需求

| 方面 | 專家系統 | 深度學習 |
|------|---------|---------|
| 訓練資料 | 少量 | 大量 |
| 專家知識 | 必要 | 可選 |
| 計算資源 | 低 | 高 |

## 四、神經符號 AI 的融合

### 4.1 為何融合？

| 專家系統 | 深度學習 |
|---------|---------|
| 可解釋 | 端到端 |
| 邏輯推理 | 模式識別 |
| 少量資料 | 大規模資料 |

兩者互補，融合可以結合雙方優勢。

### 4.2 融合方法

#### 神經網路 + 知識圖譜

```python
# 將知識圖譜嵌入到神經網路中
class KnowledgeEnhancedNLP(nn.Module):
    def __init__(self, vocabulary, knowledge_graph):
        self.embedding = nn.Embedding(len(vocabulary), 300)
        self.kg_embeddings = load_kg_embeddings(knowledge_graph)
        
    def forward(self, text):
        embeddings = self.embedding(text)
        # 结合知识图谱嵌入
        enhanced = fuse_with_kg(embeddings, self.kg_embeddings)
        return self.classifier(enhanced)
```

#### 規則增強的深度學習

```python
# 在損失函數中加入邏輯約束
def logical_loss(y_pred, y_true, constraints):
    ce_loss = cross_entropy(y_pred, y_true)
    
    # 添加邏輯約束
    logical_penalty = 0
    for rule in constraints:
        logical_penalty += rule_violation(y_pred, rule)
    
    return ce_loss + lambda * logical_penalty
```

## 五、2020 年的發展

### 5.1 預訓練語言模型的主導

2020 年，BERT、GPT 系列成為 NLP 的主流方法，專家系統似乎被邊緣化。

### 5.2 但問題仍在

- **可解釋性**：黑盒模型的問題
- **事實一致性**：幻覺問題
- **邏輯推理**：複雜推理仍困難

### 5.3 復甦的跡象

研究者開始重新關注：
- **知識圖譜增強的語言模型**
- **神經符號推理**
- **可解釋的注意力機制**
