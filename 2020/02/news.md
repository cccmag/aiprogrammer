# 本月科技新聞 — 2020年2月

## JupyterLab 1.0 正式達到里程碑

JupyterLab 在 2019 年 12 月達到 1.0 穩定版後，2020 年 2 月社群開始積極推動企業採用。JupyterLab 的分頁介面、整合終端機、延伸模組系統與豐富的主題支援，使其成為資料科學家首選的開發環境。多數雲端 Jupyter 服務（如 Google Colab、Azure Notebooks）已開始支援 JupyterLab 介面。

## Python 3.9 第一個開發版發布

Python 3.9 第一個 development 版本於 2020 年 2 月發布。3.9 將引入字典合并運算子（`|`）、型態提示改進與效能優化。預計於 2020 年 10 月發布穩定版。

## PyTorch 1.5 開發中

PyTorch 團隊在 2020 年 2 月持續開發 1.5 版本，將強化對行動裝置部署的支援（PyTorch Mobile）。torchvision、torchtext 生態系持續壯大，預計 4 月發布穩定版。

## VS Code Python 擴充突破千萬安裝

VS Code 的 Python 擴充於 2020 年 2 月達到 1,000 萬次安裝里程碑。微軟持續強化 Jupyter Notebook 整合、程式碼導航與偵錯功能。Python 開發者現在可以在 VS Code 中獲得接近 IDE 的完整體驗。

## R 語言 4.0 預覽版發布

R 4.0.0 的 first release candidate 於 2020 年 2 月發布。新版本將带来諸多改進，包括字串處理與字母工廠函式的現代化。R 與 Python 在資料科學領域的競爭持續升溫。
