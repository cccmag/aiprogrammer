# 主題總覽：Jupyter 與互動式計算

Jupyter 是開放原始碼的互動式計算環境的名稱，代表 Julia、Python、R 三種語言。的名稱是三種語言的字母組合：Ju（Julia）、Py（Python）、R（R）。然而，Jupyter 的影響力早已超越這三種語言，成為整個資料科學與機器學習領域的標準工具。

## Jupyter 的核心價值

### 互動式探索
Jupyter Notebook 讓開發者可以逐個單元格（cell）執行程式碼，即時看到輸出結果。這種互動式的方式非常適合資料探索與實驗。

### 圖文並茂
每個單元格可以是程式碼、Markdown 文字、數學公式或圖表。這種格式讓 Notebook 不只是程式碼，更像是技術文章或教學筆記。

### 可重現研究
將程式碼、輸出與說明文件整合在同一個檔案中，確保研究過程完全可重現。這對學術研究與資料分析至關重要。

## 生態系組成

### Jupyter Notebook
經典的單頁面介面，適合簡單的資料分析任務。Notebook 6.0 在 2019 年發布，带來了改善的使用者體驗。

### JupyterLab
現代化的整合開發環境，支援分頁、多面板拖拽、整合終端機與延伸模組。JupyterLab 1.0 在 2019 年 12 月達到穩定版。

### Voilà
將 Jupyter Notebook 轉換為獨立靜態網頁或互動式儀表板的工具。不需要顯示程式碼，適合分享結果給非技術人員。

### JupyterHub
多使用者 Jupyter 伺服器，支援團隊協作與雲端部署。廣泛用於教學環境與企業資料平台。

## 與傳統 IDE 的比較

| 特性 | Jupyter | VS Code / PyCharm |
|------|---------|-------------------|
| 互動式執行 | 是 | 部分（IPython 終端） |
| 圖表整合 | 優秀 | 需要外掛 |
| 適合資料探索 | 極佳 | 一般 |
| 適合大型專案 | 一般 | 極佳 |
| 敘事性文件 | 優秀 | 需要外掛 |
