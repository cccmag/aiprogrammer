# AlphaFold2：50 年問題的突破

## 前言

2020 年 11 月，DeepMind 的 AlphaFold2 在 CASP14 競賽中達到了實驗室精度，解決了蛋白質折疊問題。這是 2020 年最重要的 AI 突破。

## 蛋白質折疊問題

```
什麼是蛋白質折疊？
────────────────────────────────

胺基酸序列 → 3D 結構 → 蛋白質功能

問題困擾了生物學家 50 年

解決方案的重要性：
- 藥物開發
- 疾病研究
- 生物學突破
```

## CASP14 結果

```
CASP14（2020 年 11 月）：
────────────────────────────────

```
