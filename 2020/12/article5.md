# DevOps 與 MLOps

## 前言

MLOps 將 DevOps 原則應用於機器學習，2020 年成為 AI 領域的熱門話題。

## MLOps 成熟度模型

```
MLOps 成熟度級別：
────────────────────────────────

Level 0: 純手工
- 手動訓練模型
- 無自動化

Level 1: 自動化訓練
- 自動化的 ML 管線
- 持續訓練

Level 2: 自動化部署
- CI/CD 整合
- 自動化部署

Level 3: 自動監控
- 模型監控
- 自動重訓練
```

## 工具生態

```
MLOps 工具（2020 年）：
────────────────────────────────

- Kubeflow
- MLflow
- Airflow
- Prefect
- DVC (Data Version Control)
```
