# GPT-3：大型語言模型元年

## 前言

2020 年 6 月，OpenAI 發布了 GPT-3，這是有史以來最大的語言模型之一，標誌著大型語言模型時代的來臨。

## GPT-3 的規模

```
GPT-3 規格：
────────────────────────────────

參數數量：1750 億
訓練 tokens：約 3000 億
模型數量：8 個（從 125M 到 175B）

訓練成本：估計 ~1200 萬美元

對比：
- GPT-1 (2018): 1.17 億
- GPT-2 (2019): 15 億
- GPT-3 (2020): 1750 億  ◄─ 100x 增長
```

## GPT-3 的能力

```python
# GPT-3 的 Few-Shot 能力

# 翻譯
prompt = """Translate English to French:

English: "hello"
French: "bonjour"

English: "goodbye"
French: "au revoir"

English: "thank you"
French:"""

# GPT-3 輸出: "merci"
```
