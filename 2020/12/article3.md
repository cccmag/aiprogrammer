# 雲端運算的發展

## 前言

2020 年疫情加速了雲端運算的採用。企業數位轉型需求大增，雲端市場規模突破 3000 億美元。

## 雲端市場

```
雲端服務市場（2020 年）：
────────────────────────────────

- 市場規模：3000+ 億美元
- 年增長率：~20%
- 主要廠商：AWS、Azure、GCP

趨勢：
- 多雲策略普及
- 無伺服器採用增加
- 邊緣運算興起
```

## 主要進展

```
雲端新服務（2020 年）：
────────────────────────────────

AWS:
- Lambda 容器支援
- SageMaker 增強
- 新區域開通

Azure:
- Azure Arc
- Synapse Analytics
- Kubernetes 服務

GCP:
- Cloud Run 正式版
- BigQuery 增強
- AI Platform 更新
```
