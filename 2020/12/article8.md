# 自動駕駛的進展

## 前言

2020 年，自動駕駛技術繼續發展。Tesla FSD 持續改進，Waymo 在特定區域提供服務，整個行業向完全自動駕駛邁進。

## Tesla Autopilot

```
Tesla FSD（2020 年）：
────────────────────────────────

- FSD Beta 開始推送
- 城市道路駕駛能力
- 視覺化為主的方案
- 持續 OTA 更新

硬體：
- HW3.0 FSD 晶片
- 8 個攝影機
- 360° 覆蓋
```
