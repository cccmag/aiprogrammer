# 量子計算與 AI

## 前言

2020 年，量子計算持續發展。Google、IBM 在量子錯誤更正方面取得進展，量子機器學習成為新興領域。

## 量子計算進展

```
量子計算 2020：
────────────────────────────────

Google:
- 量子錯誤更正突破
- 53 量子位元處理器

IBM:
- 量子網路計畫
- 持續擴展量子位元數

量子硬體進步：
- 更多量子位元
- 更低錯誤率
- 更長相干時間
```
