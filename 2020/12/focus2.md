# AlphaFold2 與蛋白質折疊：生物學的突破

## 前言

蛋白質折疊問題困擾了生物學家 50 年。DeepMind 的 AlphaFold2 在 2020 年 11 月的 CASP14 競賽中達到了實驗室精度，這是 2020 年最重要的 AI 突破。

## 蛋白質折疊問題

```
什麼是蛋白質折疊？
────────────────────────────────

胺基酸序列 → 3D 結構 → 蛋白質功能

問題：
- 胺基酸鏈如何摺疊成複雜的 3D 結構？
- 這個結構如何決定蛋白質的功能？
- 能否從序列預測結構？

重要性：
- 理解疾病機制
- 藥物設計
- 合成生物學
```

## CASP14 競賽

```
CASP14 結果（2020 年 11 月）：
────────────────────────────────

AlphaFold2 GDT 分數：92.4/100
- 專業結構生物學家平均：~90
- 其他團隊最高：~60-70
- 傳統方法：~40-50

AlphaFold2 是第一個達到實驗室精度的系統！
```

## AlphaFold2 技術

```python
"""
AlphaFold2 核心創新：

1. Evoformer 架構
   - 多序列對齊（MSA）注意力
   - 配對關係建模

2. 直接結構預測
   - 端到端學習
   - 直接輸出 3D 座標

3. 幾何約束
   - 物理可行性
   - 鍵角、鍵長約束
"""
```

### 注意力機制的應用

```
AlphaFold2 中的注意力：
────────────────────────────────

1. MSA 自我注意力
   - 學習胺基酸之間的進化關係
   - 捕捉長程交互

2. 成對注意力
   - 學習殘基對之間的距離
   - 預測接觸圖

3. 結構模組
   - 不動點迭代
   - 3D 旋轉和平移
```

## 對科學的影響

```
藥物開發加速：
────────────────────────────────

傳統方法：
- X 射線晶體學：數月到數年
- 核磁共振：需要大量樣本
- 冷凍電顯：設備昂貴

AlphaFold2：
- 數小時到數天
- 只需要序列
- 成本大幅降低

影響：
- 快速識別藥物靶點
- 設計更精準的藥物
- 個性化醫療
```
