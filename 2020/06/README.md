# AI 程式人雜誌 — 2020年6月號

## 主題：GPT-3 誕生與其影響

本期聚焦於 2020 年 6 月發布的 GPT-3，這是 AI 領域的重大里程碑。GPT-3 以 1750 億參數的規模展示了大型語言模型的湧現能力，特別是 Few-shot Learning 讓人印象深刻。本期深入探討 GPT-3 的技術架構、能力與限制、以及對 AI 領域的深遠影響。
