# 回顧與結語

## 2020 年 10 月號結語

### 本月主題回顧

十月是 Python 部署與自動化的月份。本期雜誌深入探討了從 pip 到 Kubernetes 的完整 Python 部署技術棧。

#### Python 部署的演進

Python 部署從簡單的 pip 安裝，發展到複雜的容器編排和 Kubernetes 叢集管理：

- **pip 與 PyPI**：Python 生態系的基礎設施，27 萬+ 套件
- **虛擬環境**：解決依賴衝突的利器
- **Docker 容器化**：實現「寫一次，處處運行」
- **CI/CD 自動化**：GitHub Actions 讓每次提交都經過自動化流程
- **Kubernetes**：大規模部署的標準方案

#### 本月技術亮點

**pip 20.2** 的新解析器解決了長期以來的依賴地獄問題。**Docker Desktop for Apple Silicon** 的預覽版開創了新的可能。**GitHub CLI 1.0** 讓開發者可以在終端機中完成大多數操作。

### AI 領域的發展

**GPT-3** 的發布（6月）持續影響著 AI 應用的發展方向。到了 10 月，開發者社群已經開始探索大型語言模型的各種應用場景，從文案生成到程式碼輔助。

**AlphaFold2** 的突破雖然在 11 月的 CASP14 競賽中才正式公布，但我們已經看到了蛋白質結構預測的曙光。

### 未來展望

**Serverless 的普及**：Lambda、Cloud Run 等無伺服器平台讓 Python 部署更加簡單，未來將有更多無伺服器 Python 應用。

**MLOps 的興起**：機器學習模型的部署和維運正在形成一套獨特的最佳實踐，從模型訓練到生產部署的流程將更加標準化。

**容器化的標準化**：Docker 將繼續是部署的基礎，Kubernetes 將成為大規模應用的標準。服務網格（Service Mesh）和 GitOps 等概念也將更加普及。

**自動化的深化**：從 CI/CD 到 CDE（持續部署），自動化將涵蓋軟體交付的每個環節。基礎設施即程式碼（IaC）的概念將更加普遍。
