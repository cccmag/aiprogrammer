# ResNet 為何有效？

## 前言

ResNet（殘差網路）是深度學習史上最重要的突破之一。本文深入分析為何殘差連接如此有效。

---

## 一、深度網路訓練的困難

### 梯度消失/爆炸

當網路很深時，反向傳播的梯度在鏈式法則下趨近於 0 或無窮大：

```
∂L/∂x_k = ∂L/∂x_n × ∏_{i=k}^{n-1} ∂x_{i+1}/∂x_i
```

### 退化問題

實驗觀察：更深的網路反而有更高的訓練誤差。

---

## 二、殘差學習

### 核心思想

不要讓網路直接學習底層映射，而是學習殘差：

```
原本：學習 H(x)
現在：學習 F(x) = H(x) - x，輸出 H(x) = F(x) + x
```

### 為何更簡單？

學習 identity mapping（即 H(x) ≈ x）比學習非平凡映射更容易：
- 梯度可以直接流過捷徑連接
- 淺層網路可以視為深層網路的子集

---

## 三、梯度流動分析

### 前向傳播

```
x_{l+1} = x_l + F(x_l)

x_{l+2} = x_l + F(x_l) + F(x_{l+1})
x_{l+3} = x_l + F(x_l) + F(x_{l+1}) + F(x_{l+2})
...
x_L = x_l + Σ_{i=l}^{L-1} F(x_i)
```

### 反向傳播

```
∂L/∂x_l = ∂L/∂x_L × ∂x_L/∂x_l
        = ∂L/∂x_L × (1 + ∂/∂x_l Σ F(x_i))
```

關鍵點：**1 + ... 項**確保梯度至少為 1，可以無衰減地傳播。

---

## 四、為何有效：多個視角

### 1. 梯度流動

殘差連接提供了一条「梯度高速公路」，避免梯度消失。

### 2. 網路整合

多個淺層網路的集成。ResNet 可以視為許多路徑的集合，每條路徑對應不同的深度。

### 3. 迭代細化

每個殘差塊可以看作對前一層輸出的細化。

### 4. 減少了要學習的函數複雜度

學習「微小調整」比學習「完整映射」更簡單。

---

## 五、實驗證據

### 殘差連接的重要性

移除捷徑連接，網路效能顯著下降。

### 路徑分析

使用不同深度的路徑對最終預測的貢獻不同，但所有路徑都對結果有貢獻。

### 表徵重複使用

淺層特徵直接傳遞到深層，促進了表示的重複使用。

---

## 六、變體與改進

### Pre-activation ResNet

將 BN-ReLU 放在卷積之前：

```
原本：Conv -> BN -> ReLU
現在：BN -> ReLU -> Conv
```

減少了殘差分支的表示能力差距。

### SE-ResNet

加入 Squeeze-and-Excitation 模組：

```python
# Squeeze: 全域池化
# Excitation: 兩個全連接層
# Scale: 通道重加權
```

### DenseNet

密集連接：每層連接到所有後續層：

```python
x = concat([x, F1(x), F2(x, x1), F3(x, x1, x2), ...])
```

---

## 七、理論解釋

### 訊息路徑

ResNet 提供了兩條路徑：
1. **捷徑路徑**：幾乎不變的信息流
2. **殘差路徑**：學習的調整

### 梯度與表示

理論上：
- ResNet 可以表示任意深度的網路
- 訓練複雜度與深度無關（因為梯度流動）
- 淺層網路是深層網路的特例

---

## 八、實際應用

### 何時使用 ResNet？

1. 網路深度 > 20 層
2. 訓練數據充足
3. 需要良好的效能

### 替代方案

| 方法 | 原理 |
|------|------|
| DenseNet | 密集連接 |
| Highway Network | 學習的門控 |
| Neural ODE | 連續深度的微分方程 |

---

## 結語

ResNet 的成功源於其簡單而有效的設計：殘差連接解決了深度網路訓練的梯度問題，同時保持了網路學習複雜映射的能力。
