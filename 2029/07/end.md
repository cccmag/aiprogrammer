# 結語

## 從防禦到免疫

AI 安全不再只是防禦已知攻擊，而是要建立能夠適應新威脅的免疫系統。當 AI 成為基礎設施，安全就是生死問題。

### 關鍵學習

1. **安全是架構問題** — 安全不是部署前打補丁，而是要從架構層面設計。隔離、最小權限、輸入驗證是基礎。

2. **供應鏈風險被低估** — 預訓練模型、開源套件、第三方 API 都是攻擊面。供應鏈安全需要工具和流程。

3. **紅隊測試不可或缺** — 自動化紅隊測試應該成為 CI/CD 的一部分。定期測試才能發現新漏洞。
