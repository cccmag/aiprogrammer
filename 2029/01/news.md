# 2029 年 1 月 — AI 代理工作流新聞

## LangChain 收購 Airflow 核心團隊

LangChain 在 2029 年初收購了 Apache Airflow 的核心開發團隊，意圖打造 AI 原生版的工作流排程器。此舉被視為 Agent 編排將與傳統工作流引擎深度融合的信號。

參考：[LangChain Agent Workflows](https://www.google.com/search?q=LangChain+Airflow+acquisition+2029)

## Temporal 發布 AI 工作流 SDK

Temporal 在 2028 年底推出 AI 工作流專用 SDK，支援長時間運行的 Agent 任務、人機互動暫停點與自動重試，成為生產級 Agent 部署的首選。

參考：[Temporal AI SDK](https://www.google.com/search?q=Temporal+AI+workflow+SDK+2029)

## Agent 編排標準提案發布

OpenAI、Google、Anthropic 聯合提出了 Agent Communication Protocol (ACP) 標準草案，定義跨平台 Agent 之間的工作流通訊協定。

參考：[Agent Communication Protocol](https://www.google.com/search?q=Agent+Communication+Protocol+2029)

## CrewAI 達到 10 萬星標

多 Agent 編排框架 CrewAI 在 GitHub 達到 100,000 星標，成為最受歡迎的 Agent 工作流程架。

參考：[CrewAI GitHub Stars](https://www.google.com/search?q=CrewAI+stars+2029)

## 人機協作工作流研究突破

MIT 與微軟聯合發表了「漸進式自主」研究成果，量化 Human-in-the-Loop 中的人類介入成本與 Agent 自主程度的最佳平衡點。

## 工作流可觀測性工具成熟

基於 OpenTelemetry 的 Agent 工作流追蹤工具（如 LangSmith、Weights & Biases Prompts）全面支援 DAG 級別的分散式追蹤。

參考：[OpenTelemetry Agent Tracing](https://www.google.com/search?q=OpenTelemetry+agent+workflow+tracing+2029)

## AI 工作流市場規模突破 200 億美元

市場研究機構預測，AI Agent 工作流平台市場在 2029 年將突破 200 億美元，年增長率超過 60%。
