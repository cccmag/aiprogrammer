# 結語

## 視覺、語音、文字的統一

多模態 Agent 代表了 AI 從純文字到全感官互動的重大躍進。本期探討了從架構設計、視覺語言整合、語音互動到評估與記憶管理的完整技術棧。

### 關鍵學習

1. **模態融合是核心挑戰** — 如何有效整合視覺、語音與文字資訊，並讓 Agent 在不同模態之間進行推理與決策，是多模態系統設計的核心問題。

2. **即時互動改變一切** — 語音互動的即時性要求 Agent 在 200ms 內完成感知、推理與回應，這對架構設計與部署提出了全新的挑戰。

3. **評估需要新方法** — 多模態 Agent 的評估不能沿用純文字的基準，需要涵蓋模態理解、跨模態推理與端到端任務完成等新維度。
