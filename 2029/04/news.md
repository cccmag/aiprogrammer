# 2029 年 4 月 — 多模態 Agent 新聞

## GPT-6 Vision 全面開放

OpenAI 的 GPT-6 Vision 模型在 2029 年初全面開放 API，支援即時影像理解、影片分析與多輪視覺對話，成為多模態 Agent 的核心引擎。

參考：[GPT-6 Vision](https://www.google.com/search?q=GPT-6+Vision+OpenAI+2029)

## Google Gemini 3 支援即時語音互動

Google 發布 Gemini 3，支援低延遲的即時語音輸入與輸出，延遲降至 200ms 以下，使語音 Agent 達到自然人對話水準。

參考：[Gemini 3 Voice](https://www.google.com/search?q=Gemini+3+real+time+voice+2029)

## 螢幕理解 Agent 成為新興應用類別

多家新創推出螢幕理解 Agent，能夠像人類一樣「看」螢幕、操作 GUI 應用程式，在測試自動化與 RPA 領域引起廣泛關注。

參考：[Screen Understanding Agent](https://www.google.com/search?q=screen+understanding+agent+2029)

## 多模態 RAG 論文突破百篇

根據 arXiv 統計，2028 年多模態 RAG 相關論文超過 100 篇，主要研究方向包括跨模態檢索、圖文融合生成與影片級別的 RAG 管線。

## Anthropic 發布多模態安全評估框架

Anthropic 發布了多模態 Agent 的安全評估框架，涵蓋視覺注入、語音欺騙與跨模態越獄等威脅模型。

參考：[Anthropic Multimodal Safety](https://www.google.com/search?q=Anthropic+multimodal+agent+safety+2029)

## 開源多模態 Agent 框架成熟

CrewAI 與 AutoGen 在 2029 年加入了完整的多模態支援，包括影像輸入、語音處理與工具使用的統一抽象層。

參考：[CrewAI Multimodal](https://www.google.com/search?q=CrewAI+multimodal+agent+2029)

## 多模態 Agent 市場預測達 150 億美元

市場分析預測，多模態 Agent 平台與服務市場在 2030 年將達到 150 億美元，年複合成長率超過 80%。

參考：[Multimodal Agent Market](https://www.google.com/search?q=multimodal+agent+market+size+2029+2030)
